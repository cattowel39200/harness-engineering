# RC 기둥 검토

KDS 41 12 00 Ch.6 기준에 따라 철근콘크리트 기둥의 축력강도, P-M 상호작용, 세장비, 철근비, 띠철근을 검토하여 구조적 적합성을 확인한다.

---

## 규칙

1. **기준 준수**: KDS 41 12 00 제6장 (압축부재)을 적용한다
2. **모든 검토항목 필수**: 최소단면, 축력, P-M 상호작용, 세장비, 철근비, 띠철근 전부 검토한다
3. **판정 명시**: 각 항목에 대해 O.K. / N.G. 판정을 반드시 표기한다
4. **단위 통일**: kN, mm, MPa 단위계를 사용한다
5. **결과표 작성**: 최종 결과를 항목별 판정표로 정리한다

---

## 1. 최소단면 검토

```
일반 건축물: 300 × 300 mm 이상
저층 건축물(3층 이하): 250 × 250 mm 이상

판정: b ≥ b_min, h ≥ h_min → O.K.
```

- 축력을 받는 부재는 최소 단면을 확보해야 한다

---

## 2. 축력강도 검토

띠철근 기둥의 최대 설계축강도:

```
φPn_max = 0.80 × φ × [0.85 × fck × (Ag - Ast) + fy × Ast]
φ = 0.65 (띠철근 기둥)

판정: Pu ≤ φPn_max → O.K.
```

- Pu: 계수축력 (kN)
- Ag: 기둥 총단면적 (mm²)
- Ast: 주근 총단면적 (mm²)
- fck: 콘크리트 설계기준압축강도 (MPa)
- fy: 철근 항복강도 (MPa)

---

## 3. P-M 상호작용 검토

축력과 휨모멘트를 동시에 받는 기둥은 PM 다이어그램으로 검토한다.

```
PM 다이어그램 작성 주요점:
  (1) 순수압축점: (φPn_max, 0)
  (2) 균형파괴점: (φPb, φMb) — εs = εy 일 때
  (3) 순수휨점: (0, φMn)
  (4) 순수인장점: (-φTn, 0)

판정: (Pu, Mu) 점이 PM 곡선 내부 → O.K.
```

- 양방향 모멘트(Mux, Muy)가 있는 경우 Bresler 상반하중법 적용:
  ```
  1/φPn = 1/φPnx + 1/φPny - 1/φPn0
  ```

---

## 4. 세장비 검토

```
세장비 = k × lu / r

단주 판정: k × lu / r < 22 → 단주 (세장효과 무시)
장주 판정: k × lu / r ≥ 22 → 장주 (모멘트 확대 필요)
```

- k: 유효길이계수 (횡구속 0.5~1.0, 비횡구속 1.0~2.0)
- lu: 비지지길이 (mm)
- r: 회전반경 (mm), 직사각형 단면: r = 0.3h

### 장주 모멘트 확대

```
Mc = δ × Mu
δ = Cm / (1 - Pu/φPc) ≥ 1.0
Pc = π² × EI / (k × lu)²
```

---

## 5. 철근비 검토

```
ρ = Ast / Ag

허용 범위: 1% ≤ ρ ≤ 8% (KDS 기준)
실무 권장: 1% ≤ ρ ≤ 4%

판정: 1% ≤ ρ ≤ 8% → O.K.
```

- 최소 4개 주근 (띠철근 기둥)
- 최소 6개 주근 (나선철근 기둥)

---

## 6. 띠철근 간격 검토

```
s_max = min(16 × db, 48 × dt, 최소단면치수)

판정: s ≤ s_max → O.K.
```

- db: 주근 직경 (mm)
- dt: 띠철근 직경 (mm)
- 최소단면치수: 기둥 단변 (mm)
- 내진설계 시 별도의 횡보강 규정을 따른다
