# 팀 구성

정부사업 수행에 적합한 팀을 구성하고 역할분담, 채용 계획, 자문위원 구성을 체계적으로 준비한다.

---

## 규칙

1. **역할 기반 구성**: 사람에 맞추지 말고 필요한 역할을 먼저 정의한다
2. **핵심 인력 우선**: 대표, CTO 등 핵심 인력의 역량이 심사의 핵심이다
3. **정부사업 인건비 기준 준수**: 참여율, 급여 수준, 4대보험 등 규정을 따른다
4. **점진적 채용**: 사업 단계에 맞춰 필요 시점에 인력을 확보한다
5. **외부 자원 활용**: 자문위원, 멘토, 협력기관으로 팀 역량을 보완한다

---

## 초기 팀 구성 (5~7명)

### 핵심 역할 정의

```
대표이사/CEO (1명)
  - 사업 총괄, 비전/전략 수립
  - 대외 활동 (IR, 파트너십, 정부사업)
  - 심사 가점: 관련 분야 경력, 창업 경험

CTO/기술총괄 (1명)
  - 기술 아키텍처, 핵심 기술 개발
  - 기술 로드맵 관리, 팀 리드
  - 심사 가점: 기술 역량, 특허, 논문

기획/PM (1명)
  - 제품 기획, 프로젝트 관리
  - 시장 조사, 고객 분석
  - 사업계획서 작성 주도

개발자 (1~2명)
  - 백엔드/프론트엔드/풀스택
  - MVP/시제품 개발
  - 기술 문서 작성

디자이너 (0.5~1명)
  - UI/UX 설계
  - 발표자료, 마케팅 자료
  - 프로토타입 디자인 (Figma)
```

## RACI 역할분담 매트릭스

```
업무                CEO  CTO   PM   개발  디자인
사업 전략 수립       A    C    R    I    I
기술 아키텍처        I    A    C    R    I
제품 기획/요구분석   C    C    A    R    R
UI/UX 설계          I    C    C    I    A
핵심 기능 개발       I    A    I    R    I
테스트/QA           I    A    R    R    I
사업계획서 작성      A    C    R    C    C
발표/IR             A    C    R    I    C
고객 인터뷰         C    I    A    I    I
사업비 관리         A    I    R    I    I

R=Responsible(실행) A=Accountable(책임) C=Consulted(자문) I=Informed(통보)
```

## 정부사업 인건비 기준

```
필수: 4대보험 가입, 근로계약서 체결, 급여 이체 증빙
참여율: 대표 100%, 핵심 50~100%, 보조 30~50%
산정: 월급여 x 참여율 x 개월수 (연봉 상한 6,000~8,000만원)
비목: 기본급/상여금 가능, 퇴직금/복리후생 불가, 현금지급 불가
```

## 채용 계획

### 단계별 채용 전략

```
초기(0~3개월): 핵심 인력(CTO, 시니어) - 지인추천, 스톡옵션
중기(4~8개월): 개발 보강(주니어, QA) - 원티드, 로켓펀치
후기(9~12개월): 사업화(마케팅, 세일즈) - 잡코리아, 링크드인
```

## 멘토/자문위원 구성

| 유형 | 역할 | 확보 방법 | 비용 |
|------|------|-----------|------|
| 기술 자문 | 기술 검증, 아키텍처 리뷰 | 대학교수, 연구원 | 월 50~100만원 |
| 사업 자문 | BM 검증, 시장 전략 | 선배 창업자 | 회당 20~50만원 |
| 법률/회계 | 계약, IP, 세무, 정산 | 법무법인, 세무사 | 건별 상이 |

**정부사업 활용**: 자문료는 전문가활용비 비목에 편성. 자문회의록, 지급 증빙 필요.

---

## 체크리스트

- [ ] 필요한 역할(직무)을 모두 정의했는가
- [ ] 핵심 인력(대표, CTO)의 역량이 심사 기준에 부합하는가
- [ ] RACI 매트릭스로 역할 분담이 명확한가
- [ ] 인건비 대상 인력이 4대보험, 참여율 기준을 충족하는가
- [ ] 채용 시기와 채널이 사업 일정에 맞게 계획되었는가
- [ ] 자문위원 구성이 기술/사업/법률 영역을 커버하는가
- [ ] 조직도가 사업계획서에 포함할 수준으로 정리되었는가
- [ ] 인건비 상한선과 참여율 제한을 확인했는가
