# 보강토옹벽 내적안정 검토

보강토옹벽 내부의 보강재에 대해 각 층별로 인발(Pullout), 파단(Rupture), 연결부 강도를 검토한다.
외적안정(msew-external) 검토가 선행되어야 한다.

---

## 규칙

1. **결정론적 계산**: 동일 설계조건 → 동일 결과. 임의 가정 금지.
2. **전체 산식 출력**: 모든 계산은 6단계 형식을 반드시 따른다. 중간 과정을 생략하거나 압축하지 않는다.
3. **기준 조항 명시**: 모든 공식에 KDS 조항번호를 인용한다.
4. **단위 표기**: 모든 수치에 단위를 표기한다.
5. **안전율 판정**: 기준값과 비교하여 PASS/FAIL을 명시한다.
6. **가정값 표시**: 기준에 없는 값은 [가정] 표시 + 근거를 기재한다.
7. **층별 검토 필수**: 모든 보강재 층에 대해 개별 검토한다 (층수 = H / Sv).

### 6단계 산식 출력 형식 (모든 계산에 필수 적용)

```
① 기준 근거: KDS XX XX XX 조항번호 — 조항 제목
② 공식: 기준에 명시된 공식 원문
③ 변수 정의: 공식의 모든 변수 (의미 + 단위)
④ 값 대입: 각 변수에 실제 값을 넣은 식
⑤ 계산 과정: 단계별 산술 (중간값 전부 기재, 생략 금지)
⑥ 결과 판정: 계산값 vs 허용값 → PASS/FAIL
```

---

## 계산 순서 (7 Step, 순서 변경 금지)

```
Step 1: 설계조건 확인
Step 2: 장기설계강도 (Tal) 산정
Step 3: 각 층별 수평토압 산정
Step 4: 보강재 파단 (Rupture) 검토
Step 5: 보강재 인발 (Pullout) 검토
Step 6: 연결부 강도 검토
Step 7: 결과 종합표
```

---

## Step 1: 설계조건 확인

```
[필수 입력 — 하나라도 빠지면 진행 금지]

보강토체:
  벽체 높이 H         : _____ m
  보강재 길이 L        : _____ m
  보강재 수직간격 Sv   : _____ m
  보강재 수평간격 Sh   : _____ m (= 1.0 m 기본)
  보강재 층수 N        : H / Sv = _____ 층

보강재 물성:
  보강재 형식          : 지오그리드 / 지오텍스타일 / 스틸스트립
  극한인장강도 Tult    : _____ kN/m
  크리프감소계수 RF_CR : _____ (지오그리드: 1.6~2.5)
  시공손상계수 RF_ID   : _____ (지오그리드: 1.1~1.5)
  내구성계수 RF_D      : _____ (지오그리드: 1.1~1.3)

보강토 물성:
  단위중량 γ_r         : _____ kN/㎥
  내부마찰각 φ_r       : _____ °

인발 파라미터:
  마찰계수 F*          : _____ (지오그리드: tanφ_r ~ 0.8×tanφ_r)
  스케일보정계수 α      : _____ (시트형: 0.6~1.0, 그리드: 0.8)
  상재계수 C            : 2 (양면 마찰)

상재하중 q             : _____ kN/㎡ (있는 경우)
```

---

## Step 2: 장기설계강도 (Tal) 산정

```
① 기준 근거: KDS 11 80 10 — 보강재 장기설계강도

② 공식:
   Tal = Tult / (RF_CR × RF_ID × RF_D)

③ 변수 정의:
   Tal    = 장기설계강도 (kN/m)
   Tult   = 극한인장강도 (kN/m)
   RF_CR  = 크리프감소계수 (무차원)
   RF_ID  = 시공손상감소계수 (무차원)
   RF_D   = 내구성감소계수 (무차원)

④ 값 대입:
   Tal = [Tult] / ([RF_CR] × [RF_ID] × [RF_D])

⑤ 계산 과정:
   감소계수 곱 = [RF_CR] × [RF_ID] × [RF_D]
              = [RF_CR] × [RF_ID] = [중간값]
              = [중간값] × [RF_D] = [총감소계수]
   Tal = [Tult] / [총감소계수] = [Tal값] kN/m

⑥ 결과:
   Tal = [Tal값] kN/m
```

---

## Step 3: 각 층별 수평토압 산정

각 보강재 층(i = 1, 2, ..., N)에 대해 해당 깊이에서의 수평토압을 산정한다.

```
① 기준 근거: KDS 11 80 10 — 내적안정 토압

② 제 i 층 보강재 깊이 (벽체 상단에서):
   z_i = H − (i − 0.5) × Sv

③ 연직응력:
   σ_v,i = γ_r × z_i + q

④ 수평토압 (주동토압):
   Ka_r = tan²(45° − φ_r/2) (보강토 내부)
   σ_h,i = Ka_r × σ_v,i

⑤ 보강재 1개가 부담하는 인장력:
   T_i = σ_h,i × Sv × Sh

⑥ 각 층별 T_i 산정표:
   (N개 층 전부 기재)
```

---

## Step 4: 보강재 파단 (Rupture) 검토

각 층별로 보강재 인장력이 장기설계강도 이내인지 확인.

```
① 기준 근거: KDS 11 80 10 — 보강재 파단

② 공식:
   T_i ≤ Tal / FS_r

   FS_r = 1.5 (상시), 1.1 (지진시)

③ 각 층별 검토:
   제 i 층: T_i = [T_i값] kN/m
           Tal / FS_r = [Tal] / 1.5 = [허용값] kN/m
           T_i ≤ 허용값 → PASS ✅ 또는 FAIL ❌

④ 최대 인장력 층 확인:
   T_max 발생 층: 제 ____ 층 (z = ____ m)
   T_max = [값] kN/m
```

---

## Step 5: 보강재 인발 (Pullout) 검토

각 층별로 보강재의 유효 부착길이에 대한 인발저항력을 검토한다.

```
① 기준 근거: KDS 11 80 10 — 보강재 인발

② 공식:
   Pr = F* × α × σ_v,i × Le × C

③ 변수 정의:
   Pr     = 인발저항력 (kN/m)
   F*     = 인발마찰계수 (무차원)
   α      = 스케일보정계수 (무차원)
   σ_v,i  = 제 i 층의 연직응력 (kN/㎡)
   Le     = 유효 부착길이 (m) = L − La
   La     = 주동파괴면 뒤쪽 길이 (m)
   C      = 상재계수 = 2 (양면 마찰)

④ 주동파괴면 위치:
   주동파괴면 경사각 = 45° + φ_r/2
   제 i 층에서 주동파괴면까지 수평거리:
   La_i = (H − z_i) × tan(45° − φ_r/2)

⑤ 유효 부착길이:
   Le_i = L − La_i
   Le_i ≥ 1.0 m (최소 부착길이)

⑥ 인발저항력:
   Pr_i = F* × α × σ_v,i × Le_i × C

⑦ 안전율:
   FS_po = Pr_i / T_i ≥ 1.5 (상시)

⑧ 각 층별 검토:
   제 i 층: Le_i = [값] m, Pr_i = [값] kN/m
           FS_po = [Pr_i] / [T_i] = [값] ≥ 1.5
           → PASS ✅ 또는 FAIL ❌
```

---

## Step 6: 연결부 강도 검토

```
① 기준 근거: KDS 11 80 10 — 보강재-전면판 연결부

② 검토 항목:
   (a) 연결부 장기강도 T_ac:
       T_ac = Tult_conn / (RF_CR × RF_D)
       Tult_conn = 연결부 극한강도 (시험값)

   (b) 연결부 안전율:
       T_i ≤ T_ac / FS_conn
       FS_conn = 1.5

③ 연결 형식별:
   콘크리트 패널: 핀/볼트 연결 → 전단강도
   블록식: 마찰/상호맞물림 → 마찰+수동저항
   와이어메쉬: 직접 연결

④ 각 층별 검토:
   제 i 층: T_i = [값] ≤ T_ac / 1.5 = [허용값]
           → PASS ✅ 또는 FAIL ❌
```

## 체크리스트

- [ ] 모든 설계조건이 입력되었는가
- [ ] 장기설계강도 Tal = Tult / (RF_CR × RF_ID × RF_D) 가 정확한가
- [ ] 각 감소계수(RF_CR, RF_ID, RF_D)의 근거가 명시되었는가
- [ ] 보강재 층수 N = H / Sv 가 정확한가
- [ ] **모든 층에 대해 개별 검토가 수행되었는가**
- [ ] 각 층의 깊이 z_i 가 정확한가
- [ ] 파단 검토: T_i ≤ Tal / FS_r (각 층)
- [ ] 인발저항력 Pr = F* × α × σv × Le × C (각 층)
- [ ] 유효 부착길이 Le = L − La ≥ 1.0 m (각 층)
- [ ] 주동파괴면 위치가 정확한가
- [ ] 연결부 강도가 확인되었는가
- [ ] **모든 계산에 6단계 형식 적용되었는가**
- [ ] **중간 과정 생략이 없는가**
- [ ] **모든 수치에 단위가 표기되었는가**
