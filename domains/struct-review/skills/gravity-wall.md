# 중력식/반중력식 옹벽 검토

무근콘크리트 중력식(또는 반중력식) 옹벽의 외적 안정성(전도, 활동, 지지력)과 벽체 압축응력을 KDS 11 80 05에 따라 검토한다.
무근콘크리트이므로 **철근 검토는 수행하지 않으며**, 벽체 압축응력만 확인한다.

---

## 규칙

1. **결정론적 계산**: 동일 설계조건 → 동일 결과. 임의 가정 금지.
2. **전체 산식 출력**: 모든 계산은 6단계 형식을 반드시 따른다. 중간 과정을 생략하거나 압축하지 않는다.
3. **기준 조항 명시**: 모든 공식에 KDS 조항번호를 인용한다.
4. **단위 표기**: 모든 수치에 단위를 표기한다.
5. **안전율 판정**: 기준값과 비교하여 PASS/FAIL을 명시한다.
6. **가정값 표시**: 기준에 없는 값은 [가정] 표시 + 근거를 기재한다.
7. **무근콘크리트 원칙**: 철근이 없으므로 휨/전단 검토 대신 벽체 압축응력만 검토한다.

### 6단계 산식 출력 형식 (모든 계산에 필수 적용)

```
① 기준 근거: KDS XX XX XX 조항번호 — 조항 제목
② 공식: 기준에 명시된 공식 원문
③ 변수 정의: 공식의 모든 변수 (의미 + 단위)
④ 값 대입: 각 변수에 실제 값을 넣은 식
⑤ 계산 과정: 단계별 산술 (중간값 전부 기재, 생략 금지)
⑥ 결과 판정: 계산값 vs 허용값 → PASS/FAIL
```

---

## 계산 순서 (7 Step, 순서 변경 금지)

```
Step 1: 설계조건 확인
Step 2: 토압 산정 (Rankine)
Step 3: 자중 산정 (사다리꼴 단면)
Step 4: 전도 안전율 검토
Step 5: 활동 안전율 검토
Step 6: 지지력 검토
Step 7: 벽체 압축응력 검토
Step 8: 결과 종합표
```

---

## Step 1: 설계조건 확인

아래 값이 모두 정의되어야 한다. 미정의 시 계산 진행 불가.

```
[필수 입력 — 하나라도 빠지면 진행 금지]

옹벽 형식          : 중력식 / 반중력식
옹벽 높이 H        : _____ m
벽체 상단 폭 B1    : _____ m
벽체 하단 폭 B2    : _____ m
전면 경사           : 수직 / 1:n (경사비)
배면 경사           : 수직 / 1:n (경사비)

콘크리트 강도 fck   : _____ MPa (무근콘크리트)

뒤채움 단위중량 γs  : _____ kN/㎥
뒤채움 내부마찰각 φ : _____ °
뒤채움 점착력 c     : _____ kPa

지반 허용지지력 qa  : _____ kN/㎡
기초 마찰계수 μ     : _____

[고정 상수 — 변경 금지]
콘크리트 단위중량 γc : 24.0 kN/㎥ (KDS 41 10 15)
```

## Step 2: 토압 산정 (Rankine)

### 2-1. 주동토압계수

```
① 기준 근거: KDS 11 80 05 3.3.1 — Rankine 주동토압

② 공식:
   Ka = tan²(45° − φ/2)

③ 변수 정의:
   Ka = Rankine 주동토압계수 (무차원)
   φ  = 뒤채움 흙의 내부마찰각 (°)

④ 값 대입:
   Ka = tan²(45° − [φ값]/2)

⑤ 계산 과정:
   45° − φ/2 = [결과]°
   tan([결과]°) = [탄젠트값]
   Ka = [탄젠트값]² = [Ka값]

⑥ 결과:
   Ka = [Ka값]
```

### 2-2. 주동토압 합력

```
① 기준 근거: KDS 11 80 05 3.3.2 — 주동토압 합력

② 공식:
   Pa = (1/2) × γs × H² × Ka

③ 변수 정의:
   Pa = 주동토압 합력 (kN/m)
   γs = 뒤채움 단위중량 (kN/㎥)
   H  = 옹벽 전체 높이 (m)
   Ka = 주동토압계수 (무차원)

④ 값 대입:
   Pa = (1/2) × [γs] × [H]² × [Ka]

⑤ 계산 과정:
   Pa = 0.5 × [γs] × [H²] × [Ka] = [Pa값] kN/m

⑥ 결과:
   Pa = [Pa값] kN/m
   수평성분 Pah = Pa (수평 뒤채움)
   작용점 y_a = H / 3 = [y_a값] m (저면에서)
```

## Step 3: 자중 산정 (사다리꼴 단면)

모멘트 기준점: 저면 앞꿈치 끝 (전도 검토 기준점)

```
① 기준 근거: KDS 11 80 05 4.2 — 자중에 의한 안정

② 중력식 옹벽 단면: 사다리꼴 (상단 B1, 하단 B2, 높이 H)

[벽체 자중]
  면적 A = (B1 + B2) / 2 × H
        = ([B1] + [B2]) / 2 × [H]
        = [A값] ㎡

  W = A × γc × 1.0(단위폭)
    = [A값] × 24.0
    = [W값] kN/m

[도심 위치] (앞꿈치 끝에서)
  사다리꼴 도심 x_cg = (B2 + 2×B1) / (3×(B1 + B2)) × B2
  ※ 전면/배면 경사에 따라 도심 산정 (개별 계산 필수)

[합계]
  ΣV = W = [W값] kN/m
  ΣM_R = W × x_cg = [W] × [x_cg] = [ΣM_R값] kN·m/m
```

## Step 4: 전도 안전율 검토

```
① 기준 근거: KDS 11 80 05 4.3.2 — 전도에 대한 안정

② 공식:
   FS_ot = ΣM_R / ΣM_O
   기준: FS_ot ≥ 1.5 (상시), FS_ot ≥ 1.2 (지진시)

③ 변수 정의:
   FS_ot = 전도 안전율 (무차원)
   ΣM_R  = 저항 모멘트 (kN·m/m)
   ΣM_O  = 전도 모멘트 (kN·m/m) = Pah × y_a

④ 계산:
   ΣM_O = [Pah] × [y_a] = [ΣM_O값] kN·m/m
   FS_ot = [ΣM_R] / [ΣM_O] = [FS_ot값]

⑥ 판정:
   FS_ot = [FS_ot값] ≥ 1.5 (상시)
   → PASS ✅ 또는 FAIL ❌
```

## Step 5: 활동 안전율 검토

```
① 기준 근거: KDS 11 80 05 4.3.3 — 활동에 대한 안정

② 공식:
   FS_sl = (μ × ΣV) / ΣH
   기준: FS_sl ≥ 1.5 (상시), FS_sl ≥ 1.2 (지진시)

③ 변수 정의:
   μ  = 기초저면 마찰계수
   ΣV = 연직력 합 (kN/m)
   ΣH = 수평력 합 = Pah (kN/m)

④ 계산:
   저항력 = μ × ΣV = [μ] × [ΣV] = [저항력] kN/m
   FS_sl = [저항력] / [Pah] = [FS_sl값]

⑥ 판정:
   FS_sl = [FS_sl값] ≥ 1.5 (상시)
   → PASS ✅ 또는 FAIL ❌
```

## Step 6: 지지력 검토

```
① 기준 근거: KDS 11 80 05 4.3.4 — 지지력에 대한 안정

② 편심거리:
   e = B2/2 − (ΣM_R − ΣM_O) / ΣV

③ 편심 확인:
   e ≤ B2/6 → 전단면 압축

④ 최대 지반반력:
   σ_max = (ΣV/B2) × (1 + 6e/B2)

⑤ 계산 과정:
   e = [B2]/2 − ([ΣM_R] − [ΣM_O]) / [ΣV] = [e값] m
   B2/6 = [B2/6값] m
   σ_max = ([ΣV]/[B2]) × (1 + 6×[e]/[B2]) = [σ_max값] kN/㎡

⑥ 판정:
   σ_max = [σ_max값] kN/㎡ ≤ qa = [qa] kN/㎡
   → PASS ✅ 또는 FAIL ❌
```

## Step 7: 벽체 압축응력 검토

무근콘크리트이므로 철근 검토 없이 벽체 최소 단면의 압축응력만 확인한다.

```
① 기준 근거: KDS 14 20 72 — 무근콘크리트 구조

② 공식:
   σ_c = ΣV / A_min ≤ 0.6 × fck

③ 변수 정의:
   σ_c   = 벽체 압축응력 (MPa)
   ΣV    = 연직력 합 (kN/m)
   A_min = 벽체 최소 단면적 = B1 × 1000 mm × 1.0 m (상단 최소폭)
   fck   = 콘크리트 설계기준강도 (MPa)
   0.6   = 무근콘크리트 허용응력비

④ 값 대입:
   A_min = [B1] × 1000 = [A_min값] mm × 1000 mm
   σ_c = [ΣV] × 1000 / ([B1]×1000 × 1000) = [σ_c값] MPa

⑤ 허용 압축응력:
   f_ca = 0.6 × [fck] = [f_ca값] MPa

⑥ 판정:
   σ_c = [σ_c값] MPa ≤ f_ca = [f_ca값] MPa
   → PASS ✅ 또는 FAIL ❌
```

## 산출물

1. **설계조건 확인서** (Step 1)
2. **토압 산정서** — Ka, Pa, 작용점 (Step 2)
3. **자중 산정서** — 사다리꼴 단면 무게, 도심, 모멘트 (Step 3)
4. **안정성 검토서** — 전도/활동/지지력 (Step 4~6)
5. **벽체 압축응력 검토서** — 무근콘크리트 압축 확인 (Step 7)
6. **결과 종합표** (Step 8)

## 체크리스트

- [ ] 모든 설계조건이 입력되었는가 (미입력 항목 없음)
- [ ] 옹벽 형식이 중력식/반중력식으로 확인되었는가
- [ ] 사다리꼴 단면의 면적과 도심이 정확히 산정되었는가
- [ ] 토압계수 Ka가 Rankine 공식으로 산정되었는가
- [ ] 전도 안전율 ≥ 1.5 (상시), ≥ 1.2 (지진시)
- [ ] 활동 안전율 ≥ 1.5 (상시), ≥ 1.2 (지진시)
- [ ] 최대 지반반력 ≤ 허용지지력
- [ ] 편심 e ≤ B2/6 (전단면 압축 확인)
- [ ] 벽체 압축응력 ≤ 0.6 × fck (무근콘크리트)
- [ ] **철근 검토를 수행하지 않았는가** (무근콘크리트 원칙)
- [ ] **모든 계산에 6단계 형식 적용되었는가**
- [ ] **중간 과정 생략이 없는가**
- [ ] **모든 수치에 단위가 표기되었는가**
