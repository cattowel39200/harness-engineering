# 보고서 표지·개요 작성

구조계산서의 표지와 제1장(개요), 제2장(설계조건)을 작성한다.

---

## 규칙

1. 표지에는 프로젝트명, 구조물 유형, 설계기준, 작성일을 반드시 포함한다.
2. 개요에는 규모(층수/높이/스팬), 검토 목적, 적용 기준 목록을 명시한다.
3. 설계조건에는 재료(fck, fy), 강도감소계수(φ), 피복두께, 구조물 제원을 표로 정리한다.
4. 모든 값에 단위와 기준 근거를 명시한다.

---

프로젝트명 : ____________________
구조물유형 : ____________________
설계기준   : KDS __ __ __

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
      Harness Engineering
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 제1장. 개요

```
1.1 프로젝트 개요
  프로젝트명  : [입력]
  구조물 유형 : [입력]
  규      모 : [입력]
  검토 목적  : [신축/증축/용도변경/안전진단]

1.2 적용 설계기준
  ┌────────────────────────────────┐
  │ (해당 기준 목록)                │
  └────────────────────────────────┘
```

## 제2장. 설계조건

```
2.1 재료
  ┌──────────────┬──────┬──────┬──────────┐
  │ 항목          │ 값   │ 단위 │ 기준      │
  ├──────────────┼──────┼──────┼──────────┤
  │ fck          │      │ MPa  │ KDS 41 12│
  │ fy           │      │ MPa  │ KS D 3504│
  │ Ec           │      │ MPa  │ 산정식    │
  │ Es           │200000│ MPa  │ KDS 41 12│
  └──────────────┴──────┴──────┴──────────┘

2.2 강도감소계수
  φ_휨=0.85, φ_전단=0.75, φ_축력=0.65

2.3 피복두께
  슬래브 20mm, 보 40mm, 기둥 40mm, 기초 75mm

2.4 구조물 제원 (부재별 치수 표)
```
