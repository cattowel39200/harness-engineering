# 지원사업 DB

정부 창업지원사업의 주요 프로그램을 체계적으로 정리하여 매칭 및 신청 전략 수립의 기초 데이터를 제공한다.

---

## 규칙

1. **매년 공고 확인 필수**: 사업별 세부 요건은 매년 변경되므로 반드시 해당 연도 공고문을 기준으로 한다
2. **K-Startup 기준**: 중기부 사업은 K-Startup(k-startup.go.kr) 공고를 1차 출처로 사용한다
3. **최신 연도 기준**: 아래 DB는 일반적 기준이며, 실제 지원규모/자격/일정은 연도별 공고에 따라 달라진다
4. **전담기관 직접 확인**: 세부 심사기준, 제출서류, 가점항목은 전담기관 공고를 반드시 확인한다

## 연간 주요 일정

| 월 | 주요 공고/일정 |
|----|---------------|
| 1월 | 중소기업 R&D 과제 공고, 예비창업패키지 공고 시작 |
| 2월 | 예비창업/초기창업/도약패키지 접수, 소상공인 창업 공고 |
| 3월 | 초기/도약 접수 마감, IITP R&D 공고, 지자체 사업 공고 시작 |
| 4월 | 서면/발표평가 진행, 중소기업 R&D 접수 마감 |
| 5월 | 선정 결과 발표 시작, 협약 체결 |
| 6~8월 | 사업 수행, 중간점검, 하반기 추가 공고 |
| 9~10월 | 성과 보고, 하반기 지자체 사업 공고 |
| 11월 | 차년도 청년창업사관학교 모집, 성과평가 |
| 12월 | TIPS 상시 접수, 차년도 사업 예고 |

## 체크리스트

- [ ] 올해 공고문을 K-Startup에서 직접 확인했는가
- [ ] 지원 자격요건(업력, 나이, 기술분야)을 충족하는가
- [ ] 사업별 심사기준 배점을 확인하고 기록했는가
- [ ] 주요 일정(접수~선정)을 캘린더에 등록했는가
- [ ] 전담기관 홈페이지에서 세부 공고를 확인했는가
- [ ] 전년도 선정 결과(경쟁률, 선정 수)를 파악했는가
- [ ] 중복지원 제한 사항을 확인했는가
- [ ] 가점 항목(여성, 장애인, 지역 등)을 확인했는가
