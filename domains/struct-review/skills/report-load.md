# 보고서 하중산정편 작성

구조계산서 제3장(하중 산정)을 작성한다. 고정하중, 활하중, 풍하중, 지진하중, 하중조합의 **전체 산식**을 수록한다.

---

## 규칙

1. 각 하중 항목별로 **6단계 산식**(기준→공식→변수→대입→과정→결과)을 전부 기재한다.
2. "정리하면", "위와 동일" 등 생략 표현을 사용하지 않는다.
3. 하중 산정 결과를 표로 정리한다.
4. 하중조합 목록과 지배조합을 명시한다.

---

## 작성 내용

### 3.1 고정하중 (D)

```
① 기준: KDS 41 10 15

┌────────────────┬──────────────────┬─────────┐
│ 구분            │ 산식              │ 하중     │
├────────────────┼──────────────────┼─────────┤
│ RC 자중        │ γc × t =          │    kN/㎡│
│ 바닥 마감       │ (항목별 나열)      │    kN/㎡│
│ ...            │                  │         │
├────────────────┼──────────────────┼─────────┤
│ 합계           │                  │    kN/㎡│
└────────────────┴──────────────────┴─────────┘
```

### 3.2 활하중 (L)
- 용도별 KDS 41 10 15 Table 4.2 인용

### 3.3 풍하중 (W)
- 풍속→풍속압→풍하중 6단계 산식

### 3.4 지진하중 (E)
- 지역계수→SDS/SD1→Cs→V 6단계 산식

### 3.5 하중조합
```
┌──────┬──────────────────┬──────┐
│ 조합  │ 하중조합          │ 비고 │
├──────┼──────────────────┼──────┤
│ LC1  │ 1.2D + 1.6L      │      │
│ LC2  │ 1.2D + 1.0L+1.0E │      │
│ ...  │                  │      │
│ 지배  │                  │      │
└──────┴──────────────────┴──────┘
```

## 옹벽/홍수방지벽의 경우

토압, 수압, 양압력을 추가로 산정:
- 토압계수 Ka (Rankine/Coulomb)
- 토압 합력 Pa
- 수압 Ph (홍수시)
- 양압력 U
