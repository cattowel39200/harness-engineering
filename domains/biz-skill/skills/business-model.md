# 비즈니스 모델

시장분석 결과를 바탕으로 Business Model Canvas를 완성하고, 수익모델과 가격전략을 구체화하여 3개년 매출을 추정한다.

---

## 규칙

1. **BMC 9블록 완성**: 모든 블록을 빈칸 없이 채운다. 빈 블록은 사업모델 미완성으로 간주된다
2. **수익모델 구체화**: "어떻게 돈을 버는가"를 명확히 하고, 단가와 과금 단위를 구체적으로 기술한다
3. **단가 근거 제시**: 가격은 반드시 경쟁사 가격, 원가, 또는 고객 지불의향 중 하나 이상의 근거를 포함한다
4. **현실적 추정**: 매출 추정은 보수적/기본/낙관 시나리오로 제시한다
5. **BEP 명시**: 손익분기점 도달 시점을 명확히 제시하여 사업 지속성을 보여준다

## Business Model Canvas 9블록 작성 가이드

```
┌──────────────┬──────────────┬──────────────┬──────────────┬──────────────┐
│              │              │              │              │              │
│  핵심 파트너  │  핵심 활동    │              │  고객 관계    │              │
│  (KP)        │  (KA)        │  가치 제안    │  (CR)        │  고객 세그먼트│
│              │              │  (VP)        │              │  (CS)        │
│              ├──────────────┤              ├──────────────┤              │
│              │              │              │              │              │
│              │  핵심 자원    │              │  채널         │              │
│              │  (KR)        │              │  (CH)        │              │
│              │              │              │              │              │
├──────────────┴──────────────┴──────┬───────┴──────────────┴──────────────┤
│  비용 구조 (C$)                    │  수익원 (R$)                        │
│                                    │                                     │
└────────────────────────────────────┴─────────────────────────────────────┘
```

### 블록별 작성 포인트

| 블록 | 핵심 질문 | 작성 시 주의점 |
|------|-----------|---------------|
| CS 고객 세그먼트 | 누구를 위한 가치인가? | B2B/B2G/B2C 명확 구분 |
| VP 가치 제안 | 어떤 문제를 해결하는가? | 기능이 아닌 고객 가치로 기술 |
| CH 채널 | 고객에게 어떻게 도달하는가? | 인지→평가→구매→전달→사후 단계별 |
| CR 고객 관계 | 어떤 유형의 관계를 구축하는가? | 자동/셀프/전담/커뮤니티 |
| R$ 수익원 | 고객이 무엇에 대가를 지불하는가? | 과금 모델과 단가를 구체적으로 |
| KR 핵심 자원 | VP 전달에 필수적인 자원은? | 인적/지적/물적/재무적 자원 |
| KA 핵심 활동 | 비즈니스 운영의 핵심 활동은? | 개발/생산/마케팅/운영 중 핵심 |
| KP 핵심 파트너 | 외부에서 조달할 활동/자원은? | 전략적 제휴, 공급자, 위탁처 |
| C$ 비용 구조 | 주요 비용 항목과 구조는? | 고정비/변동비 구분 |

| 유형 | 설명 | 적합한 경우 | 예시 |

```
자사 가격:               원
차이 근거:
```

### 가치 기반 가격

```
고객이 현재 지출하는 비용:          원/월
자사 솔루션으로 절감되는 비용:      원/월
자사 가격 (절감액의 30-50%):        원/월
고객 ROI:                          %
```

### 원가 + 마진

```
단위 원가 (서버, 인건비, 라이선스 등):     원
목표 마진율:        %
판매 가격:          원
```

## BEP (손익분기점) 간이 계산

```
월 고정비 = 인건비 + 임차료 + 서버비 + 기타 고정비
           =        +        +        +
           =              원/월

단위당 공헌이익 = 단가 - 단위 변동비
               =      -
               =        원

BEP 고객 수 = 월 고정비 / 단위당 공헌이익
            =          /
            =        명(개사)

BEP 도달 예상 시점: 사업 개시 후     개월 (근거:                )
```

## 체크리스트

- [ ] BMC 9블록을 모두 빈칸 없이 작성했는가
- [ ] 가치 제안이 기능이 아닌 고객 가치로 기술되어 있는가
- [ ] 수익모델의 과금 단위와 단가가 구체적인가
- [ ] 가격에 경쟁사/원가/가치 중 하나 이상의 근거가 있는가
- [ ] BEP를 계산하고 도달 시점이 현실적인가
- [ ] 3개년 매출 추정이 시나리오별로 제시되었는가
- [ ] 매출 추정의 가정(고객 수, 전환율 등)이 명시되어 있는가
- [ ] 비용 구조에서 고정비와 변동비가 구분되어 있는가
- [ ] 심사위원에게 "돈을 벌 수 있는 사업"임을 설득할 수 있는가
- [ ] 정부지원금 없이도 지속 가능한 모델인가
