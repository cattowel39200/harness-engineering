# 사업비 편성

정부지원사업의 비목 기준에 맞춰 사업비를 편성하고, 심사 감점 요인을 사전에 제거한다.

---

## 규칙

1. **정부 비목 기준 엄격 준수**: 사업별 공고문에 명시된 비목 체계를 반드시 따른다
2. **현물 포함**: 정부출연금 외에 기업 자체 부담(현금+현물)을 반드시 편성한다
3. **민간부담금 비율**: 사업별로 정해진 민간부담금 비율을 충족해야 한다
4. **산출근거 필수**: 모든 비목에 단가 x 수량 x 기간의 산출근거를 명시한다
5. **시장가격 기준**: 장비, 재료 등은 시장 조사 기반 합리적 단가를 적용한다

---

## 비목 체계

### 대분류 구조

```
총 사업비
├─ 정부출연금 (정부 지원금)
│   ├─ 직접비
│   │   ├─ 인건비
│   │   ├─ 재료비 (연구재료, 시약, 소모품)
│   │   ├─ 연구장비/시설 (구입, 임차)
│   │   ├─ 연구활동비 (여비, 회의, 수용비)
│   │   ├─ 위탁연구비 (외부 연구 위탁)
│   │   └─ 기타 (지식재산권, 기술정보, 인쇄)
│   └─ 간접비 (직접비의 10~15%)
│
└─ 민간부담금 (기업 자부담)
    ├─ 현금 (직접 지출)
    └─ 현물 (기존 장비, 인력, 시설 활용)
```

---

## 비목별 편성 기준

| 비목 | 편성 기준 | 비율 가이드 | 주의사항 |
|------|-----------|-------------|----------|
| 인건비 | 공정위 기준단가 x 참여율 x 기간 | 총액의 40~60% | 참여율 합계 100% 초과 불가 |
| 재료비 | 시장단가 x 수량 | 총액의 10~20% | 견적서 첨부 권장 |
| 연구장비 | 시장가 기준, 기존 장비 확인 | 총액의 10~20% | 고가장비는 공동활용 검토 |
| 연구활동비 | 출장 일비/숙박/교통 실비 | 총액의 5~10% | 해외출장은 사전승인 |
| 위탁연구비 | 위탁 범위 명확 설정 | 총액의 20% 이내 | 핵심기술 위탁 지양 |
| 간접비 | 직접비의 10~15% | 사업별 상이 | 공고문 상한 확인 |

---

## 인건비 계산법

### 기본 공식

```
월 인건비 = 연봉 / 12 x 참여율(%)
총 인건비 = 월 인건비 x 참여 기간(개월)
```

### 직급별 기준단가 (중소기업 기준, 참고용)

| 직급 | 연봉 기준 | 월 단가(100%) | 비고 |
|------|-----------|---------------|------|
| 책임연구원(박사급) | 6,000~8,000만원 | 500~667만원 | 경력 10년+ |
| 선임연구원(석사급) | 4,500~6,000만원 | 375~500만원 | 경력 5~10년 |
| 연구원(학사급) | 3,000~4,500만원 | 250~375만원 | 경력 3~5년 |
| 연구보조원 | 2,400~3,000만원 | 200~250만원 | 경력 3년 미만 |

### 참여율 편성 원칙

```
- 과제책임자(PM): 20~50% (타 과제 참여 감안)
- 참여연구원: 30~100%
- 한 연구원의 모든 과제 참여율 합계 <= 130%
- 연봉은 4대보험 사업주 부담분 포함 가능
```

## 현물 편성

### 현물 인정 항목

| 항목 | 산정 방법 | 예시 |
|------|-----------|------|
| 기존 인력 | 연봉 x 참여율 x 기간 | 대표이사 인건비 현물 |
| 기존 장비 | 감가상각비 x 활용률 x 기간 | 서버, 개발장비 |
| 기존 시설 | 임차료 상당액 x 활용률 | 사무실, 연구실 |
| 기보유 재료 | 취득가 기준 | 시약, 부품 재고 |

### 감가상각 계산

```
연간 감가상각비 = 취득가 / 내용연수
월 감가상각비 = 연간 감가상각비 / 12
현물 인정액 = 월 감가상각비 x 활용률(%) x 기간(월)

예시: 서버 1,200만원 / 내용연수 5년 = 연 240만원
      월 20만원 x 활용률 50% x 12개월 = 120만원
```

## 민간부담금 비율

| 사업유형 | 정부출연금 | 민간부담금 | 현금 비율 |
|----------|-----------|-----------|-----------|
| 예비/초기 창업패키지 | 100% | 0% (일부 매칭) | - |
| 창업도약패키지 | 70~80% | 20~30% | 현금 10%+ |
| 중소기업 R&D (창업) | 75% | 25% | 현금 10%+ |
| 중소기업 R&D (일반) | 65~75% | 25~35% | 현금 10%+ |
| IITP R&D | 50~75% | 25~50% | 현금 비율 상이 |
| TIPS | 정부 R&D 5억 | 엔젤 1억+ | 투자금 |

**주의**: 비율은 사업별/연도별로 달라지므로 반드시 해당 연도 공고문을 확인한다.

## 흔한 편성 오류 및 감점 사항

| 오류 유형 | 내용 | 대응 방안 |
|-----------|------|-----------|
| 인건비 과다 | 총액의 70% 이상 인건비 편성 | 인건비 50~60%로 조정 |
| 산출근거 부실 | "일식 OO만원" 으로만 기재 | 단가 x 수량 x 횟수 명시 |
| 핵심기술 위탁 | 핵심 기술개발을 외부 위탁 | 자체 수행으로 변경 |
| 비목 간 불균형 | 특정 비목에 편중 | 사업 내용과 비목 정합성 확인 |
| 현물 과대 산정 | 감가상각 미반영, 과다 활용률 | 합리적 감가상각 적용 |
| 간접비 상한 초과 | 공고 기준 초과 편성 | 공고문 상한 재확인 |
| 인건비 단가 불일치 | 공정위 기준단가와 상이 | 기준단가표 참고 |

## 체크리스트

- [ ] 공고문의 비목 체계와 편성 기준을 확인했는가
- [ ] 인건비가 총액의 40~60% 범위 내인가
- [ ] 모든 비목에 단가 x 수량 x 기간의 산출근거가 있는가
- [ ] 인건비 단가가 공정위 기준 또는 실제 연봉 기준에 부합하는가
- [ ] 참여연구원의 참여율 합계가 130%를 초과하지 않는가
- [ ] 민간부담금(현금+현물) 비율이 공고 기준을 충족하는가
- [ ] 현물은 감가상각을 적용하여 합리적으로 산정했는가
- [ ] 간접비가 공고문 상한 이내인가
- [ ] 핵심기술이 위탁연구로 편성되지 않았는가
- [ ] 사업 내용(기술개발, 사업화)과 비목 편성이 정합적인가
