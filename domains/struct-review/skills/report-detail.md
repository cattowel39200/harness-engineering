# 보고서 배근상세편 작성

구조계산서 제10장(배근 상세 검토)을 작성한다.

---

## 규칙

1. 각 검토 항목별 6단계 산식을 기재한다.
2. 부재별로(슬래브, 보, 기둥) 각각 기재한다.

---

## 작성 내용

### 10.1 최소철근비

```
① 기준: KDS 41 12 00 8.6.1
② 공식: ρ_min = max(0.25√fck/fy, 1.4/fy)
③~⑥: 부재별 산정 + 판정
```

### 10.2 최대철근비

```
② 공식: ρ_max = 0.85β1(fck/fy)(600/(600+fy))×0.75
```

### 10.3 철근 간격

```
슬래브: min(3h, 450mm)
보:     min(2.5h, 350mm)
```

### 10.4 정착길이

```
② 공식: ld = (fy×db)/(4×1.4×√fck) × 보정계수
```

### 10.5 겹침이음길이

```
Class A: 1.0ld
Class B: 1.3ld
```

### 10.6 기둥 띠철근 간격

```
간격 ≤ min(16db, 48dt, 최소단면)
```
