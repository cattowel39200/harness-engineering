# Dead Load

구조물에 작용하는 고정하중(Dead Load)을 산정한다. RC 자중, 마감하중, 칸막이벽 하중, 설비하중을 항목별로 산출하여 총 고정하중을 결정한다.

---

## 규칙

- KDS 41 12 00 (콘크리트구조 설계기준) 및 KDS 41 10 15 (건축물 하중)를 따른다
- 자중은 부재 단면 치수와 단위중량으로 직접 산정한다
- 마감하중은 실제 마감 사양이 확정되지 않은 경우 표준값을 적용한다
- 모든 하중값에 단위 (kN/m2, kN/m3)를 명기한다

## RC 구조물 자중

### 단위중량 (KDS 41 10 15)

| 재료 | 단위중량 | 비고 |
|------|---------|------|
| 철근콘크리트 | 24 kN/m3 | 표준 |
| 무근콘크리트 | 23 kN/m3 | |
| 경량콘크리트 | 18~20 kN/m3 | 종류에 따라 |
| 철골 | 77 kN/m3 | |
| 시멘트 모르타르 | 21 kN/m3 | |

### 슬래브 자중 산정

| 슬래브 두께 (mm) | 자중 (kN/m2) |
|-----------------|-------------|
| 150 | 0.15 x 24 = 3.60 |
| 180 | 0.18 x 24 = 4.32 |
| 200 | 0.20 x 24 = 4.80 |
| 210 | 0.21 x 24 = 5.04 |
| 250 | 0.25 x 24 = 6.00 |

- 산정식: w_slab = t(m) x 24 (kN/m3)

## 체크리스트

- [ ] RC 자중 단위중량 24 kN/m3 적용 확인
- [ ] 슬래브 두께별 자중 산정
- [ ] 바닥 마감하중 적용 (1.0~1.5 kN/m2)
- [ ] 천장 마감하중 적용 (0.5 kN/m2)
- [ ] 칸막이벽 하중 적용 (경량 1.0 / 조적 2.0 kN/m2)
- [ ] 설비하중 적용 (0.5~1.0 kN/m2)
- [ ] 기준층과 지붕층 구분 산정
- [ ] 모든 항목에 단위 (kN/m2) 표기
