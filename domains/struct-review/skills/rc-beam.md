# RC 보 검토

KDS 41 12 00 기준에 따라 철근콘크리트 보의 휨강도, 전단강도, 처짐, 균열폭을 검토하여 구조적 적합성을 확인한다.

---

## 규칙

1. **기준 준수**: KDS 41 12 00 (콘크리트구조 설계기준)을 적용한다
2. **모든 검토항목 필수**: 최소높이, 휨, 전단, 전단철근, 처짐, 균열폭 전부 검토한다
3. **판정 명시**: 각 항목에 대해 O.K. / N.G. 판정을 반드시 표기한다
4. **단위 통일**: kN, mm, MPa 단위계를 사용한다
5. **결과표 작성**: 최종 결과를 항목별 판정표로 정리한다

---

## 1. 최소높이 검토

처짐 계산을 하지 않는 경우의 보 최소높이:

| 지지조건 | 최소높이 |
|----------|----------|
| 단순지지 | L / 16 |
| 연속 | L / 18.5 |
| 캔틸레버 | L / 8 |

```
h_min = L / (해당 계수)
판정: h ≥ h_min → O.K.
```

---

## 2. 휨강도 검토

직사각형 보 단면에 대한 휨강도를 산정한다.

```
a = As × fy / (0.85 × fck × b)
Mn = As × fy × (d - a/2)
φMn = 0.85 × Mn

판정: Mu ≤ φMn → O.K.
```

- Mu: 계수휨모멘트 (kN·m)
- φ: 강도감소계수 = 0.85 (휨)
- As: 인장철근 단면적 (mm²)
- fy: 철근 항복강도 (MPa)
- fck: 콘크리트 설계기준압축강도 (MPa)
- b: 보 폭 (mm)
- d: 유효깊이 (mm)

---

## 3. 전단강도 검토

콘크리트 전단강도와 전단철근 기여분을 합산한다.

```
Vc = (1/6) × √fck × bw × d
Vs = Av × fy × d / s
φVn = 0.75 × (Vc + Vs)

판정: Vu ≤ φVn → O.K.
```

- Vu: 계수전단력 (kN)
- φ: 강도감소계수 = 0.75 (전단)
- bw: 복부 폭 (mm)
- Av: 전단철근 단면적 (mm²)
- s: 전단철근 간격 (mm)

---

## 4. 최소 전단철근 검토

```
Av_min = max(0.062 × √fck, 0.35) × bw × s / fy

판정: Av ≥ Av_min → O.K.
```

- 전단철근이 필요한 구간에서 최소량을 확인한다
- Vu > 0.5 × φVc 인 경우 전단철근 배치 필수

---

## 5. 처짐 검토

### 즉시처짐

```
Δi = 5 × w × L⁴ / (384 × E × I)
```

- w: 사용하중 (kN/mm)
- L: 경간 (mm)
- E: 콘크리트 탄성계수 (MPa), Ec = 8500 × ³√fck
- I: 유효단면2차모멘트 (mm⁴), 균열 여부에 따라 Ig 또는 Ie 적용

### 장기처짐

```
λΔ = ξ / (1 + 50ρ')
Δ_long = λΔ × Δi (지속하중분)
Δ_total = Δi(활하중) + λΔ × Δi(지속하중)

판정: Δ_total ≤ L/240 (또는 L/360) → O.K.
```

- ξ: 시간경과계수 (5년 이상 = 2.0)
- ρ': 압축철근비

---

## 6. 균열폭 검토 (Gergely-Lutz)

```
w = 1.1 × β × fs × ³√(dc × A)
```

- w: 균열폭 (mm)
- β: 인장면에서 중립축까지 거리 / 철근 도심에서 중립축까지 거리
- fs: 사용하중 시 철근응력 (MPa)
- dc: 최외단 인장철근 도심에서 인장면까지 거리 (mm)
- A: 인장철근 1개당 유효 인장면적 = 2 × dc × b / n (mm²)

```
허용 균열폭:
  - 옥내 건조환경: 0.4 mm
  - 옥외 습윤환경: 0.3 mm

판정: w ≤ w_allow → O.K.
```
