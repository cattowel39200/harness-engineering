# 전석 쌓기 구조검토

전석(자연석/잡석) 쌓기의 토압, 자중(사다리꼴), 전도, 활동, 지지력을 검토한다.
석축 쌓기(masonry-wall)와 유사하나 전석 특유의 **공극률, 채움재, 배수** 특성을 추가로 반영한다.

---

## 규칙

1. **결정론적 계산**: 동일 설계조건 → 동일 결과. 임의 가정 금지.
2. **전체 산식 출력**: 모든 계산은 6단계 형식을 반드시 따른다. 중간 과정을 생략하거나 압축하지 않는다.
3. **기준 조항 명시**: 모든 공식에 KDS 조항번호를 인용한다.
4. **단위 표기**: 모든 수치에 단위를 표기한다.
5. **안전율 판정**: 기준값과 비교하여 PASS/FAIL을 명시한다.
6. **가정값 표시**: 기준에 없는 값은 [가정] 표시 + 근거를 기재한다.
7. **전석 특성 반영**: 공극률에 의한 유효 단위중량, 배수 조건을 고려한다.

### 6단계 산식 출력 형식 (모든 계산에 필수 적용)

```
① 기준 근거: KDS XX XX XX 조항번호 — 조항 제목
② 공식: 기준에 명시된 공식 원문
③ 변수 정의: 공식의 모든 변수 (의미 + 단위)
④ 값 대입: 각 변수에 실제 값을 넣은 식
⑤ 계산 과정: 단계별 산술 (중간값 전부 기재, 생략 금지)
⑥ 결과 판정: 계산값 vs 허용값 → PASS/FAIL
```

---

## 계산 순서 (9 Step, 순서 변경 금지)

```
Step 1: 설계조건 확인
Step 2: 전석 유효 단위중량 산정
Step 3: 토압 산정 (경사면 고려)
Step 4: 자중 산정 (사다리꼴 단면)
Step 5: 전도 안전율 검토
Step 6: 활동 안전율 검토
Step 7: 지지력 검토
Step 8: 배수 및 뒷채움 검토
Step 9: 결과 종합표
```

---

## Step 1: 설계조건 확인

```
[필수 입력 — 하나라도 빠지면 진행 금지]

전석 쌓기 형식      : 전석 메쌓기 / 전석 찰쌓기
전석 높이 H         : _____ m
상단 폭 B1          : _____ m
하단 폭 B2          : _____ m
전면 경사           : 1 : n (예: 1:0.3, 1:0.5)
배면 경사           : 수직 / 1:n

석재 종류           : 화강석 / 자연석 / 잡석
석재 고유 단위중량 γ_stone : _____ kN/㎥ (화강석 26, 자연석 25)
공극률 n_v          : _____ % (전석 메쌓기: 30~40%, 찰쌓기: 15~25%)
채움재             : 콘크리트 / 모르타르 / 없음(메쌓기)

뒤채움 단위중량 γs  : _____ kN/㎥
뒤채움 내부마찰각 φ : _____ °
배면 경사각 β       : _____ °

지반 허용지지력 qa  : _____ kN/㎡
기초 마찰계수 μ     : _____

기초 콘크리트:
  폭 B_f            : _____ m
  높이 H_f          : _____ m
  콘크리트 강도 fck  : _____ MPa

[고정 상수 — 변경 금지]
콘크리트 단위중량 γc : 24.0 kN/㎥
물 단위중량 γw      : 9.81 kN/㎥
```

---

## Step 2: 전석 유효 단위중량 산정

```
① 기준 근거: 경험식 — 전석 유효 단위중량

② 공식:
   메쌓기 (채움재 없음):
     γ_eff = γ_stone × (1 − n_v/100)

   찰쌓기 (모르타르/콘크리트 채움):
     γ_eff = γ_stone × (1 − n_v/100) + γ_fill × (n_v/100)

③ 변수 정의:
   γ_eff   = 전석 유효 단위중량 (kN/㎥)
   γ_stone = 석재 고유 단위중량 (kN/㎥)
   n_v     = 공극률 (%)
   γ_fill  = 채움재 단위중량 (kN/㎥) — 콘크리트: 23, 모르타르: 21

④ 값 대입:
   γ_eff = [γ_stone] × (1 − [n_v]/100) + [γ_fill] × ([n_v]/100)
         = [값] kN/㎥

⑥ 결과:
   γ_eff = [γ_eff값] kN/㎥
```

---

## Step 3: 토압 산정 (경사면 고려)

석축 쌓기(masonry-wall)의 Step 2와 동일한 방법 적용.

```
배면 경사 β 가 있는 경우:
  Ka = cosβ × [cosβ − √(cos²β − cos²φ)] / [cosβ + √(cos²β − cos²φ)]

배면이 수평(β=0)인 경우:
  Ka = tan²(45° − φ/2)

Pa = (1/2) × γs × H² × Ka
Pah = Pa × cosβ
Pav = Pa × sinβ
y_a = H / 3
```

---

## Step 4: 자중 산정 (사다리꼴 단면)

```
① 석축 단면: 사다리꼴 (상단 B1, 하단 B2, 높이 H)

[전석 자중] — 유효 단위중량 적용
  면적 A = (B1 + B2) / 2 × H
  W_r = A × γ_eff × 1.0(단위폭)

[기초 콘크리트 자중]
  W_f = B_f × H_f × γc

[뒤채움 토사 자중] (배면 경사 시)
  W_s = 배면 경사에 의한 토사 면적 × γs

[합계]
  ΣV = W_r + W_f + W_s + Pav
  ΣM_R = 각 성분 × 모멘트팔 합계
```

---

## Step 5: 전도 안전율 검토

```
FS_ot = ΣM_R / ΣM_O ≥ 1.5 (상시), ≥ 1.2 (지진시)
ΣM_O = Pah × y_a

판정: PASS ✅ 또는 FAIL ❌
```

---

## Step 6: 활동 안전율 검토

```
FS_sl = (μ × ΣV) / ΣH ≥ 1.5 (상시), ≥ 1.2 (지진시)
ΣH = Pah

판정: PASS ✅ 또는 FAIL ❌
```

---

## Step 7: 지지력 검토

```
편심: e = B2/2 − (ΣM_R − ΣM_O) / ΣV
e ≤ B2/6 확인

σ_max = (ΣV/B2) × (1 + 6e/B2) ≤ qa

판정: PASS ✅ 또는 FAIL ❌
```

---

## Step 8: 배수 및 뒷채움 검토

전석 쌓기 특유의 추가 검토 항목.

### 8-1. 배수 검토

```
① 검토 항목:
   (a) 유공관 설치 여부: 저면부 φ100~150 mm
   (b) 유공관 간격: 2.0~3.0 m 이내
   (c) 배수층 두께: 전석 배면 최소 300 mm 사질토/쇄석

② 수압 고려:
   배수 불량 시 수압 추가: p_w = γw × h_w
   → 전도/활동 안전율 재검토 필요

③ 판정:
   배수시설 적합 → PASS ✅
   배수 불량 → 수압 추가 후 재검토 필요
```

### 8-2. 뒷채움재 검토

```
① 뒷채움재 조건:
   (a) 투수성 재료 사용 (사질토, 쇄석)
   (b) 다짐도: 90% 이상 (KS F 2312)
   (c) 최소 두께: 300 mm 이상

② 전석 간 채움:
   메쌓기: 잔자갈 채움 (공극 유지)
   찰쌓기: 모르타르 또는 콘크리트 충전
```

### 8-3. 공극률 적정성 확인

```
전석 메쌓기 적정 공극률: 30~40%
전석 찰쌓기 적정 공극률: 15~25%

공극률 과대 → 유효 단위중량 감소 → 자중 부족 우려
공극률 과소 → 배수 불량 (메쌓기 시)
```

## 산출물

1. **설계조건 확인서** (Step 1)
2. **유효 단위중량 산정서** — 공극률 반영 (Step 2)
3. **토압 산정서** — 경사면 Ka, Pa, 성분 분리 (Step 3)
4. **자중 산정서** — 사다리꼴 전석 + 기초 (Step 4)
5. **안정성 검토서** — 전도/활동/지지력 (Step 5~7)
6. **배수 및 뒷채움 검토서** (Step 8)
7. **결과 종합표** (Step 9)

---

## 체크리스트

- [ ] 모든 설계조건이 입력되었는가
- [ ] 공극률이 전석 형식에 맞게 설정되었는가 (메쌓기 30~40%, 찰쌓기 15~25%)
- [ ] 유효 단위중량이 공극률을 반영하여 산정되었는가
- [ ] 전면경사가 반영되었는가
- [ ] 배면 경사각 β가 토압계수에 반영되었는가
- [ ] 전도 안전율 ≥ 1.5 (상시)
- [ ] 활동 안전율 ≥ 1.5 (상시)
- [ ] 최대 지반반력 ≤ 허용지지력
- [ ] 편심 e ≤ B2/6
- [ ] 배수시설(유공관, 배수층)이 확인되었는가
- [ ] 뒷채움재 투수성 및 다짐 조건이 확인되었는가
- [ ] 수압 고려 여부가 판단되었는가 (배수 불량 시 추가)
- [ ] **모든 계산에 6단계 형식 적용되었는가**
- [ ] **중간 과정 생략이 없는가**
- [ ] **모든 수치에 단위가 표기되었는가**
