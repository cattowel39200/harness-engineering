# BOX 암거 구조검토

BOX형 암거의 상부슬래브, 벽체, 하부슬래브에 대해 연직토압, 수평토압, DB하중을 고려하여 휨/전단을 KDS 24 14 21에 따라 검토한다.

---

## 규칙

1. **결정론적 계산**: 동일 설계조건 → 동일 결과. 임의 가정 금지.
2. **전체 산식 출력**: 모든 계산은 6단계 형식을 반드시 따른다. 중간 과정을 생략하거나 압축하지 않는다.
3. **기준 조항 명시**: 모든 공식에 KDS 조항번호를 인용한다.
4. **단위 표기**: 모든 수치에 단위를 표기한다.
5. **안전율 판정**: 기준값과 비교하여 PASS/FAIL을 명시한다.
6. **가정값 표시**: 기준에 없는 값은 [가정] 표시 + 근거를 기재한다.

### 6단계 산식 출력 형식 (모든 계산에 필수 적용)

```
① 기준 근거: KDS XX XX XX 조항번호 — 조항 제목
② 공식: 기준에 명시된 공식 원문
③ 변수 정의: 공식의 모든 변수 (의미 + 단위)
④ 값 대입: 각 변수에 실제 값을 넣은 식
⑤ 계산 과정: 단계별 산술 (중간값 전부 기재, 생략 금지)
⑥ 결과 판정: 계산값 vs 허용값 → PASS/FAIL
```

---

## 계산 순서 (10 Step, 순서 변경 금지)

```
Step 1: 설계조건 확인
Step 2: 하중 산정 (연직토압 + 수평토압 + 활하중)
Step 3: DB하중 분배폭 산정
Step 4: 하중조합
Step 5: 상부슬래브 휨 검토
Step 6: 상부슬래브 전단 검토
Step 7: 벽체 휨 검토
Step 8: 하부슬래브 휨 검토
Step 9: 하부슬래브 전단 검토 + 기초반력 확인
Step 10: 결과 종합표
```

## Step 2: 하중 산정

### 2-1. 연직토압 (상부 하중)

```
① 기준 근거: KDS 24 14 21 — 연직토압

② 공식:
   W_e = γs × H_c

③ 변수 정의:
   W_e = 연직토압 (kN/㎡)
   γs  = 뒤채움 단위중량 (kN/㎥)
   H_c = 토피고 (m)

④ 값 대입:
   W_e = [γs] × [H_c] = [W_e값] kN/㎡
```

### 2-2. 수평토압

```
① 기준 근거: KDS 24 14 21 — 수평토압

② 공식:
   상단: p_h1 = K₀ × γs × H_c
   하단: p_h2 = K₀ × γs × (H_c + H_i + ts/1000 + tb/1000)

③ 정지토압계수:
   K₀ = 1 − sinφ = 1 − sin([φ]) = [K₀값]
```

### 2-3. 자중

```
상부슬래브 자중: w_ts = γc × ts/1000 = 24.0 × [ts/1000] = [값] kN/㎡
하부슬래브 자중: w_tb = γc × tb/1000 = 24.0 × [tb/1000] = [값] kN/㎡
```

## Step 3: DB하중 분배폭 산정

```
① 기준 근거: KDS 24 14 21 — 활하중 분배

② 분배폭 공식:
   토피고 H_c ≥ 0.6 m 인 경우:
   분배폭 b_e = 바퀴 접지폭 + 2 × H_c × tan(30°)

③ DB-24 후륜하중:
   P_wheel = 96 kN (1개 후륜)
   충격계수 i = 15 / (40 + L_span) (단, i ≤ 0.3)

④ 등분포 환산:
   w_L = P_wheel × (1 + i) / b_e (kN/㎡)
```

---

## Step 4: 하중조합

```
① 기준 근거: KDS 24 12 21 — 하중조합

② 극한하중조합:
   U = 1.2 × D + 1.6 × L + 1.2 × E_v + 1.6 × E_h

   상부슬래브 등분포:
     w_u = 1.2 × (w_ts + W_e) + 1.6 × w_L

   벽체 수평하중:
     p_u = 1.6 × p_h (사다리꼴 분포)
```

---

## Step 5: 상부슬래브 휨 검토

```
① 기준 근거: KDS 24 14 21 / KDS 41 12 00 4.2.3

② 설계 모멘트 (양단 고정 가정):
   단부: Mu_neg = w_u × l² / 12
   중앙: Mu_pos = w_u × l² / 24
   l = 내공폭 + 벽체두께 = W + tw/1000

③ 단면 검토:
   d = ts − 피복 − db/2
   a = (As × fy) / (0.85 × fck × b)
   φMn = 0.85 × As × fy × (d − a/2) / 10⁶

④ 판정:
   Mu ≤ φMn → PASS ✅ 또는 FAIL ❌
```

---

## Step 6: 상부슬래브 전단 검토

```
① 기준 근거: KDS 41 12 00 5.3.1

② 설계 전단력:
   Vu = w_u × l / 2

③ 콘크리트 전단강도:
   φVc = 0.75 × (1/6) × √fck × b × d / 1000

④ 판정:
   Vu ≤ φVc → PASS ✅ 또는 FAIL ❌
```

---

## Step 7: 벽체 휨 검토

```
① 기준 근거: KDS 24 14 21 / KDS 41 12 00

② 수평토압 하중:
   사다리꼴 분포 (상단 p_h1, 하단 p_h2)
   벽체 높이 h_w = H_i + ts/1000 + tb/1000

③ 설계 모멘트 (양단 고정):
   단부: 사다리꼴 분포에 대한 고정단 모멘트
   Mu = 계수 × p_u × h_w²

④ 단면 검토:
   d = tw − 피복 − db/2
   φMn ≥ Mu

⑤ 판정:
   Mu ≤ φMn → PASS ✅ 또는 FAIL ❌
```

---

## Step 8: 하부슬래브 휨 검토

```
① 기준 근거: KDS 24 14 21 / KDS 41 12 00

② 하부슬래브 하중:
   상향 기초반력 = (전체 연직하중) / (저판폭)
   하향 자중 = w_tb

③ 순하중에 의한 모멘트 산정 (양단 고정)
④ 단면 검토: φMn ≥ Mu

⑤ 판정:
   Mu ≤ φMn → PASS ✅ 또는 FAIL ❌
```

---

## Step 9: 하부슬래브 전단 + 기초반력 확인

### 9-1. 하부슬래브 전단 검토

```
Vu ≤ φVc (Step 6과 동일 형식)
```

### 9-2. 기초반력 확인

```
① 기준 근거: KDS 11 50 00 — 지반지지력

② 공식:
   σ = ΣV / A_base ≤ qa

③ 계산:
   ΣV = 상부슬래브 자중 + 토압 + 활하중 + 벽체 자중 + 하부슬래브 자중
   A_base = (W + 2×tw/1000) × 1.0 m

④ 판정:
   σ ≤ qa → PASS ✅ 또는 FAIL ❌
```

3. **하중조합표** (Step 4)

## 체크리스트

- [ ] 모든 설계조건이 입력되었는가 (미입력 항목 없음)
- [ ] 연직토압 = γs × H_c 가 정확히 산정되었는가
- [ ] 수평토압 = K₀ × γs × H 가 사다리꼴 분포로 산정되었는가
- [ ] DB하중 분배폭이 토피고 기반으로 산정되었는가
- [ ] 충격계수가 적용되었는가
- [ ] 상부슬래브 휨: Mu ≤ φMn (단부 + 중앙)
- [ ] 상부슬래브 전단: Vu ≤ φVc
- [ ] 벽체 휨: 수평토압에 의한 Mu ≤ φMn
- [ ] 하부슬래브 휨/전단 검토 완료
- [ ] 기초반력 ≤ 허용지지력
- [ ] **모든 계산에 6단계 형식 적용되었는가**
- [ ] **중간 과정 생략이 없는가**
- [ ] **모든 수치에 단위가 표기되었는가**
