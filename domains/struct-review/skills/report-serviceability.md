# 보고서 사용성편 작성

구조계산서 제9장(사용성 검토)을 작성한다. 균열폭과 처짐을 6단계 산식으로 수록한다.

---

## 규칙

1. 균열폭과 처짐 각각 6단계 산식을 전부 기재한다.
2. 부재별로 반복 기재한다 (슬래브, 보 각각).

---

## 9.1 균열폭 검토

```
① 기준: KDS 41 12 00 4.2.4
② 공식: w ≤ 허용값 (일반 0.3mm, 부식환경 0.2mm)
③ 변수: β, fs, dc, A 등
④ 대입: 숫자
⑤ 과정: 중간값 전부
⑥ 판정: w = __mm ≤ 0.3mm → PASS/FAIL
```

## 9.2 처짐 검토

```
즉시처짐:
① 기준: KDS 41 12 00 4.2.3
② 공식: Δi = 5wL⁴/(384EcIe)
③~⑤: 유효단면2차모멘트 Ie 산정 포함
⑥ 판정

장기처짐:
② 공식: Δlt = λΔ × Δi, λΔ = ξ/(1+50ρ')
④~⑥: 산식 + 판정

허용값: L/240(바닥), L/480(비구조벽), L/360(기타)
```

## 체크리스트
- [ ] 균열폭 6단계 산식
- [ ] 즉시처짐 6단계 산식 (Ie 산정 포함)
- [ ] 장기처짐 6단계 산식
- [ ] 허용값 비교 + 판정
