# 지원사업 매칭

검증된 아이디어와 현재 사업 상황을 기반으로 최적의 정부지원사업을 매칭하고, 동시 신청 전략 및 일정을 수립한다.

---

## 규칙

1. **자격요건 먼저 확인**: 매칭 전 업력, 대표자 요건, 기술분야 등 기본 자격을 먼저 검증한다
2. **중복지원 제한 확인**: 동일 부처/유사 사업 간 중복지원 제한 규정을 반드시 확인한다
3. **사업 시기 확인**: 현재 시점에서 접수 가능한 사업인지, 차년도 준비가 필요한지 판단한다
4. **현실적 선정 가능성**: 경쟁률과 자사 역량을 고려하여 현실적으로 선정 가능한 사업을 우선한다

---

## 매칭 의사결정 트리

### Step 1: 업력 확인

```
업력이 어디에 해당하는가?

├─ 미창업 (예비창업자)
│   → 예비창업패키지, 청년창업사관학교(39세 이하), 지자체 예비창업
│
├─ 3년 이내 (초기기업)
│   → 초기창업패키지, TIPS(기술기반), 청년사관학교(39세 이하)
│
├─ 3~7년 (성장기업)
│   → 창업도약패키지, 도약패키지(스케일업), 중소기업 R&D
│
└─ 7년 초과 (일반 중소기업)
    → 중소기업 R&D, IITP R&D, 지자체 지원사업
```

### Step 2: 기술기반 여부

```
기술기반 창업인가?

├─ YES (기술기반)
│   → TIPS 우선 검토, R&D 사업 병행, 기술성 배점 높은 사업 유리
│
└─ NO (비기술/서비스)
    → 창업패키지 중심, 시장성/사업화 배점 높은 사업 집중
```

### Step 3: 투자유치 가능성

```
엔젤/VC 투자유치가 가능한가?

├─ YES → TIPS 최우선 (운영사 접촉 필수)
└─ NO  → 창업패키지, R&D 사업 집중
```

### Step 4: 필요 지원규모

```
필요한 지원 금액은?

├─ 5,000만원 이하 → 지자체, 소상공인 사업
├─ 5,000만~1억원 → 창업패키지
├─ 1~3억원 → 도약패키지, 중소기업 R&D
└─ 3억원 이상 → TIPS, IITP R&D, 대형 R&D
```

---

## 중복지원 제한 규칙

| 구분 | 중복 불가 | 동시 가능 |
|------|-----------|-----------|
| 창업패키지 간 | 예비/초기/도약 중 택1 | 불가 (택1만 가능) |
| 패키지 + TIPS | 동시 수혜 불가 | 순차 가능 (종료 후) |
| 패키지 + R&D | 내용 중복 시 불가 | 내용 상이하면 가능 |
| 패키지 + 지자체 | 대부분 동시 가능 | 지자체 규정 확인 |
| R&D 사업 간 | 동일 과제 불가 | 과제 내용 상이 시 가능 |

**핵심**: 같은 사업화 내용으로 중복 수혜는 불가하나, 사업화(패키지)와 기술개발(R&D)은 내용이 다르면 병행 가능하다.

---

## 동시 신청 전략

### 우선순위 설정

```
1순위: 선정 시 사업 수행에 가장 적합한 사업 (규모, 내용 매칭)
2순위: 선정 가능성이 높은 사업 (경쟁률 낮거나 가점 보유)
3순위: 안전망 역할 (소규모지만 확실한 사업)
```

### 전략 조합 예시

| 상황 | 1순위 | 2순위 | 3순위 |
|------|-------|-------|-------|
| 예비창업+기술 | 예비창업패키지 | 청년사관학교 | 지자체 |
| 초기+기술 | TIPS | 초기창업패키지 | 지자체 R&D |
| 초기+비기술 | 초기창업패키지 | 소상공인창업 | 지자체 |
| 3~7년+기술 | 중소기업R&D | 창업도약패키지 | 지자체 |

---

## 신청 타이밍 전략

| 시기 | 전략 |
|------|------|
| 11~1월 | 차년도 사업 공고 모니터링, 사업계획서 초안 작성 시작 |
| 1~2월 | 공고 확정 즉시 세부요건 분석, 계획서 본격 작성 |
| 2~3월 | 접수 마감 2주 전 완성 목표, 최종 검토 및 제출 |
| 4~5월 | 발표평가 준비, 불합격 시 하반기/차년도 전략 수정 |
| 6~10월 | 하반기 추가공고 대응, TIPS 상시 접수 활용 |
| 11~12월 | 올해 성과 정리, 차년도 전략 재수립 |

---

## 산출물

1. **추천 사업 3순위**: 1순위/2순위/3순위 사업명 + 선정 이유
2. **신청 일정표**: 사업별 접수일~선정발표일 타임라인
3. **준비 체크리스트**: 사업별 필수 서류 및 준비사항

---

## 체크리스트

- [ ] 업력, 대표자 나이, 기술분야 등 기본 자격을 확인했는가
- [ ] 의사결정 트리를 따라 후보 사업을 도출했는가
- [ ] 중복지원 제한 규정을 확인하고 동시 신청 가능 여부를 판단했는가
- [ ] 1순위/2순위/3순위 사업을 선정하고 근거를 기록했는가
- [ ] 각 사업의 접수 일정을 확인하고 캘린더에 반영했는가
- [ ] 사업계획서 작성 일정을 역산하여 준비 시작일을 정했는가
- [ ] 전년도 경쟁률과 선정률을 파악하여 현실성을 검증했는가
- [ ] 가점 항목을 확인하고 해당 여부를 체크했는가
