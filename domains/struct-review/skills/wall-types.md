# 옹벽 형식 분류 (Wall Type Classification)

옹벽의 형식을 정확히 분류하고, 하중 방향에 따른 거동을 판별한다. 형식 분류가 틀리면 전도 기준점, 안전율 산정이 모두 틀리므로 **구조검토 전에 반드시 정확히 판별**해야 한다.

---

## 규칙

1. **형식 판별은 저판 위치로 결정**: 벽체에 작용하는 주 하중(토압/수압) 방향과 저판 위치의 관계로 판별한다.
2. **양방향 하중 구조물은 CASE별로 거동이 다름**: 홍수방지벽처럼 상시/홍수시 하중이 반대면 각각의 거동을 별도 판별한다.
3. **추정 금지**: 반드시 도면 또는 계산서에서 확인한다.

---

## 옹벽 형식 정의

### L형 옹벽

```
      하중 →→→

      ┌──┐
      │벽│
      │체│
      │  │
  ┌───┴──┘
  저판
  (앞꿈치)

  정의: 저판이 하중 반대쪽에 있음
  = 벽체 앞쪽(하중이 오는 쪽의 반대)으로 저판이 돌출
  = 벽체가 저판 위에서 한쪽으로 치우쳐 있고, 저판이 하중 반대편으로 나감
```

**L형 특징:**
- 저판이 벽체 전면(하중 반대쪽)으로 돌출
- 뒤채움 토사의 무게가 저판 위에 직접 실리지 않음
- 벽체가 캔틸레버로 토압을 직접 받음
- 전도 기준점 = 저판 앞꿈치 끝단

### 역L형 옹벽

```
      하중 →→→

  ┌──┐
  │벽│  뒤채움
  │체│  (토사)
  │  │
  └──┴─────┐
           저판
           (뒤꿈치)

  정의: 저판이 하중 받는 쪽에 있음
  = 벽체 뒤쪽(하중이 오는 쪽)으로 저판이 돌출
  = 뒤채움 토사의 무게가 저판 위에 실려서 안정에 기여
```

**역L형 특징:**
- 저판이 벽체 배면(하중 쪽)으로 돌출
- 뒤채움 토사의 자중이 저판 위에 실려 전도 저항에 유리
- 경제적 (같은 높이에서 저판 폭이 작아도 됨)
- 전도 기준점 = 저판 앞꿈치 끝단 (하중 반대쪽)

### 역T형 (T형) 옹벽

```
      하중 →→→

      ┌──┐
      │벽│  뒤채움
      │체│
      │  │
  ┌───┴──┴─────┐
  앞꿈치  뒤꿈치
      저판

  정의: 저판이 양쪽으로 돌출
  = 앞꿈치 + 뒤꿈치 모두 있음
```

### 중력식 옹벽

```
      하중 →→→

  ╲          ╱
   ╲        ╱
    ╲      ╱
     ╲    ╱
      ╲  ╱
       ╲╱

  정의: 자중으로 안정 확보
  = 무근콘크리트 또는 석축
  = 철근 없음
```

## 형식 판별 절차

### Step 1: 기본 형상 확인

```
질문 1: 벽체에 대해 저판이 어느 쪽으로 돌출하는가?

  하중 방향을 기준으로:
  ┌───────────────────────────────────────┐
  │ 저판이 하중 반대쪽으로 돌출 → L형      │
  │ 저판이 하중 쪽으로 돌출     → 역L형    │
  │ 저판이 양쪽으로 돌출        → 역T형    │
  │ 저판 없이 자중으로 안정     → 중력식   │
  └───────────────────────────────────────┘
```

### Step 2: 하중 방향 확인

```
질문 2: 주 하중은 어느 방향에서 오는가?

  ┌───────────────────────────────────────┐
  │ 배면 토압만 작용        → 일반 옹벽    │
  │ 수압만 작용             → 홍수방지벽   │
  │ 토압 + 수압 양방향 작용 → 양방향 옹벽  │
  └───────────────────────────────────────┘
```

### Step 3: 양방향 하중 시 CASE별 거동

```
양방향 옹벽 (홍수방지벽)의 경우:

  하천측         제내지측
  (수압 →)      (토압 ←)
      ┌──┐
      │벽│  뒤채움
      │체│
      │  │
      └──┴────┐
           저판(뒤꿈치)

  CASE 1 (상시 - 토압):
    하중 = 토압 ← (제내지→하천)
    저판이 토압 쪽(제내지)에 있음
    → 역L형 거동

  CASE 2 (홍수시 - 수압):
    하중 = 수압 → (하천→제내지)
    저판이 수압 반대쪽(제내지)에 있음
    → L형 거동

  결론: 상시 역L형 / 홍수시 L형
```

## 홍수방지벽 분류 상세

### 저판 위치에 따른 분류

```
Type A: 저판이 제내지측 (뒤꿈치 = 제내지)

  하천          제내지
  ┌──┐
  │  │ 뒤채움
  │  │
  └──┴────┐
       저판(뒤꿈치)

  상시(토압←): 역L형  ← 기본 형식으로 표기
  홍수시(수압→): L형


Type B: 저판이 하천측 (뒤꿈치 = 하천)

  하천          제내지
      ┌──┐
      │  │ 뒤채움
      │  │
  ┌───┴──┘
  저판(앞꿈치)

  상시(토압←): L형  ← 기본 형식으로 표기
  홍수시(수압→): 역L형
```

### 전도 기준점

```
전도 기준점 = 하중 방향으로 밀렸을 때 회전하는 점
           = 저판 끝단 중 하중 방향 쪽 끝

예: 수압이 → 방향이면
    전도 기준점 = 우측(제내지측) 저판 끝
```

## 흔한 오류와 방지

| 오류 | 원인 | 방지법 |
|------|------|--------|
| L/역L 혼동 | 하중 방향 미확인 | **하중→저판 관계로 판별** |
| 전도 기준점 오류 | 형식 혼동 | 하중 방향 쪽 저판 끝 |
| 홍수방지벽 단방향 검토 | 양방향 하중 미인식 | **상시+홍수시 2 CASE 필수** |
| 도면 좌우 반대 해석 | 단면도 방향 미확인 | **평면도와 대조** |
| 수위 적용 오류 | HWL/저판고 혼동 | **표고 관계 먼저 정리** |

---

## 산출물

1. **옹벽 형식 판별서**: 형식, 하중방향, CASE별 거동
2. **단면 스케치**: ASCII 또는 치수 포함 스케치
3. **전도 기준점**: CASE별 기준점 위치

---

## 체크리스트

- [ ] 옹벽 형식이 도면/계산서에서 확인되었는가 (추정 아님)
- [ ] L형/역L형 판별이 **하중→저판 관계**로 수행되었는가
- [ ] 양방향 하중(수압+토압)이면 2 CASE 모두 판별했는가
- [ ] 전도 기준점이 CASE별로 정확한가
- [ ] 단면 스케치에 하중 방향, 저판 위치가 명시되었는가
- [ ] 도면의 좌/우 방향이 실제 방향과 일치하는지 확인했는가
