# Seismic Design Review

내진설계 적정성을 검토한다. 내진등급, 내진설계범주(SDC), 층간변위, P-Delta 효과, 내진상세 및 접합부 전단강도를 확인하여 지진하중에 대한 구조 안전성을 판정한다.

---

## 규칙

- 설계기준은 KDS 41 17 00 (건축물 내진설계기준)을 따른다
- 모든 검토항목에 기준 조항번호를 명시한다
- 층간변위비, 안정성 계수 등 수치에 반드시 단위와 판정(PASS/FAIL)을 포함한다
- SDC에 따른 내진상세 등급을 반드시 확인하고 적용한다

## 1. 내진등급 결정 (KDS 41 17 00 4.2)

| 내진등급 | 중요도 | 적용 대상 |
|----------|--------|-----------|
| 특등급 | 1.5 | 원자력, 방재시설 등 |
| 1등급 | 1.2 | 병원, 학교, 소방서, 공연장(1,000인 이상) |
| 2등급 | 1.0 | 일반 건축물 |

- 건축물의 용도 및 규모에 따라 내진등급을 결정한다
- 중요도계수(IE)는 설계응답스펙트럼 산정 시 반영한다

## 2. 내진설계범주(SDC) 판정 (KDS 41 17 00 6.2)

### 설계스펙트럼가속도

| 파라미터 | 산정식 | 비고 |
|----------|--------|------|
| SDS | 2/3 × Fa × Ss | 단주기 설계스펙트럼가속도 |
| SD1 | 2/3 × Fv × S1 | 1초주기 설계스펙트럼가속도 |

### SDC 판정표

| SDS 범위 | SD1 범위 | SDC |
|----------|----------|-----|
| SDS < 0.167 | SD1 < 0.067 | A |
| 0.167 <= SDS < 0.33 | 0.067 <= SD1 < 0.133 | B |
| 0.33 <= SDS < 0.50 | 0.133 <= SD1 < 0.20 | C |
| SDS >= 0.50 | SD1 >= 0.20 | D |
| S1 >= 0.75 | - | E (중요도 1,2등급), F (특등급) |

- SDC는 SDS 기준과 SD1 기준 중 불리한 값(높은 등급)을 적용한다

## 3. 층간변위 검토 (KDS 41 17 00 6.5.4)

### 허용 층간변위비 (Δ/h)

| 구조형식 | 허용 층간변위비 | 비고 |
|----------|----------------|------|
| RC 골조 (비구조벽 분리) | 0.015 | 일반적 |
| RC 골조 (조적벽 부착) | 0.010 | 비구조요소 손상 방지 |
| RC 골조 (특등급) | 0.010 | 중요 시설 |

### 검토 절차

1. 탄성 층간변위 δxe 산정 (해석결과)
2. 비탄성 층간변위 산정: Δx = Cd × δxe / IE
3. 층간변위비 산정: Δx / hsx
4. 허용값과 비교 판정

| 층 | hsx (mm) | δxe (mm) | Cd | IE | Δx (mm) | Δx/hsx | 허용값 | 판정 |
|----|----------|----------|----|----|---------|--------|--------|------|
| | | | | | | | | PASS/FAIL |

## 4. P-Delta 효과 (KDS 41 17 00 6.5.5)

### 안정성 계수

$$\theta = \frac{P_u \times \Delta}{V_u \times h_{sx}}$$

| 파라미터 | 설명 | 단위 |
|----------|------|------|
| Pu | 해당 층 이상 총 수직하중 (1.0D + 0.5L) | kN |
| Δ | 층간변위 | mm |
| Vu | 해당 층 설계 층전단력 | kN |
| hsx | 층고 | mm |

### 판정 기준

| 조건 | 판정 | 조치 |
|------|------|------|
| theta <= 0.10 | PASS | P-Delta 효과 무시 가능 |
| 0.10 < theta <= theta_max | 조건부 PASS | P-Delta 효과 반영 필요 |
| theta > theta_max | FAIL | 구조 불안정, 부재 보강 필요 |

- theta_max = 0.5 / (beta × Cd) <= 0.25

## 5. 내진상세 (KDS 41 17 00 7장)

### SDC별 요구 내진상세

| SDC | 요구 골조 등급 | 비고 |
|-----|---------------|------|
| A, B | 보통모멘트골조 (OMF) | 일반 상세 |
| C | 중간모멘트골조 (IMF) | 중간 상세 |
| D, E, F | 특수모멘트골조 (SMF) | 특수 상세 |

### 특수모멘트골조 (SDC D 이상) 주요 요구사항

- 보: 양단 소성힌지 구간 횡보강근 간격 d/4 이하
- 기둥: 강기둥-약보 조건 ΣMnc >= 1.2 × ΣMnb
- 기둥 횡보강근: 소성힌지 구간 간격 min(b/4, 6db, 150mm)
- 전이층 없는 연속 골조 구성

## 6. 보-기둥 접합부 검토 (KDS 41 17 00 7.5)

### 접합부 전단강도

| 접합부 유형 | 공칭전단강도 Vn | 비고 |
|------------|----------------|------|
| 4면 구속 | 1.7 × sqrt(fck) × Aj | 보가 4면 접합 |
| 3면 구속 | 1.25 × sqrt(fck) × Aj | 보가 3면 접합 |
| 기타 | 1.0 × sqrt(fck) × Aj | 2면 이하 |

- Aj = bj × hc (유효접합부 면적)
- bj = min(bc, bw + hc, bw + 2x) (유효폭)
- 접합부 작용전단력: Vu = T1 + T2 - Vcol

### 접합부 검토표

| 접합부 | 유형 | Vu (kN) | phi×Vn (kN) | Vu/phi×Vn | 판정 |
|--------|------|---------|-------------|-----------|------|
| | | | | | PASS/FAIL |

## 산출물

- 내진등급 및 SDC 판정 결과
- 각 층별 층간변위비 검토표
- P-Delta 안정성 검토표
- 내진상세 적정성 확인
- 보-기둥 접합부 전단강도 검토표
- 종합 내진설계 판정 (PASS/FAIL)

## 체크리스트

- [ ] 내진등급 및 중요도계수 결정
- [ ] SDS, SD1 산정 및 SDC 판정
- [ ] 전 층 층간변위비 허용값 이내 확인
- [ ] P-Delta 안정성 계수 theta <= 0.10 확인
- [ ] SDC에 따른 내진상세 등급 적용 확인
- [ ] SDC D 이상 시 특수모멘트골조 상세 확인
- [ ] 보-기둥 접합부 전단강도 PASS 확인
- [ ] 모든 수치에 단위 및 기준 조항번호 명기
