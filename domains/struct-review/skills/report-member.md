# 보고서 부재검토편 작성

구조계산서 제4~7장(부재 검토)을 작성한다. 슬래브, 보, 기둥, 기초(또는 옹벽) 각각에 대해 **제원표 + 전체 산식**을 수록한다.

---

## 규칙

1. **부재마다 제원표를 먼저 쓴다**: 치수, 배근, 재료, 유효깊이를 표로 정리.
2. **모든 검토에 6단계 산식**: 기준→공식→변수→대입→과정→판정.
3. **부재가 여러 개여도 각각 산식을 쓴다**: "위와 동일" 금지.
4. **판정은 Mu/φMn, Vu/φVc 비율로 표시**: ≤ 1.0이면 PASS.

---

## 각 부재의 필수 구성

### 부재 제원표 (매 부재 앞에 필수)

```
┌──────────────┬────────┬──────┐
│ 항목          │ 값      │ 단위 │
├──────────────┼────────┼──────┤
│ 부재 크기     │        │ mm   │
│ 유효깊이 d    │        │ mm   │
│ 주철근        │ D__@__ │      │
│ 철근량 As     │        │ mm²  │
│ fck          │        │ MPa  │
│ fy           │        │ MPa  │
└──────────────┴────────┴──────┘
```

### 휨강도 검토 (6단계)

```
① 기준: KDS 41 12 00 4.2.3
② 공식: Mu ≤ φMn, Mn = As × fy × (d − a/2), a = As×fy/(0.85×fck×b)
③ 변수: (모든 변수 의미+단위)
④ 대입: (숫자 넣은 식)
⑤ 과정: Mu 산정 → a 산정 → Mn 산정 → φMn 산정 (중간값 전부)
⑥ 판정: Mu/φMn = ___ ≤ 1.0 → PASS/FAIL
```

### 전단강도 검토 (6단계)

```
① 기준: KDS 41 12 00 5.3.1
② 공식: Vu ≤ φVc, Vc = (1/6)√fck × b × d
③~⑥: (동일 형식)
```

### 부재별 추가 검토

| 부재 | 추가 검토 항목 |
|------|--------------|
| 슬래브 | 최소철근비, 최대간격 |
| 보 | 전단보강근, 처짐, 균열 |
| 기둥 | P-M 상호작용, 세장비 |
| 기초 | 지지력, 전도, 활동, 뚫림전단, 침하 |
| 옹벽 | 토압, 전도, 활동, 지지력, 벽체, 저판 |

## 체크리스트
- [ ] 각 부재별 제원표 작성
- [ ] 휨강도 6단계 산식 (Mu→a→Mn→φMn→판정)
- [ ] 전단강도 6단계 산식 (Vu→Vc→φVc→판정)
- [ ] 부재별 추가 검토 포함
- [ ] 모든 수치에 단위 표기
- [ ] 생략 표현 없음
