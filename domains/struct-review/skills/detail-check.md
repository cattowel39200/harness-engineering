# Reinforcement Detail Check

철근콘크리트 부재의 배근 상세를 검토한다. 최소/최대 철근비, 철근 간격, 정착길이, 겹침이음길이, 기둥 띠철근 간격 등이 설계기준을 만족하는지 확인한다.

---

## 규칙

- 설계기준은 KDS 41 12 00 8장 (철근 상세)을 따른다
- 모든 검토항목에 기준 조항번호를 명시한다
- 수치 산출 시 반드시 단위를 표기하고 판정(PASS/FAIL)을 포함한다
- fck, fy 값은 설계조건 블록에서 제공된 값을 사용한다

## 1. 최소철근비 (KDS 41 12 00 8.6)

### 슬래브 최소철근비

| 철근 종류 | 최소 철근비 | 비고 |
|-----------|-----------|------|
| SD400 (fy = 400 MPa) | 0.0018 | 수축온도 철근 |
| SD500 (fy = 500 MPa) | 0.0014 | |

- As,min = rho_min x b x h
- 수축온도 철근은 양방향 모두 배치

### 보 최소철근비

$$\rho_{min} = \max\left(\frac{0.25\sqrt{f_{ck}}}{f_y},\ \frac{1.4}{f_y}\right)$$

| fck (MPa) | fy (MPa) | 0.25*sqrt(fck)/fy | 1.4/fy | rho_min |
|-----------|----------|-------------------|--------|---------|
| 24 | 400 | 0.00306 | 0.00350 | 0.00350 |
| 27 | 400 | 0.00325 | 0.00350 | 0.00350 |
| 30 | 400 | 0.00342 | 0.00350 | 0.00350 |
| 24 | 500 | 0.00245 | 0.00280 | 0.00280 |

- As,min = rho_min x b x d

### 기둥 최소철근비

- rho_min = 0.01 (Ag 기준)
- rho_max = 0.08 (Ag 기준)

## 2. 최대철근비 (KDS 41 12 00 8.6)

### 보 최대철근비

$$\rho_{max} = 0.85 \times \beta_1 \times \frac{f_{ck}}{f_y} \times \frac{600}{600 + f_y} \times 0.75$$

| fck (MPa) | beta1 | fy (MPa) | rho_max |
|-----------|-------|----------|---------|
| 24 | 0.85 | 400 | 0.0217 |
| 27 | 0.85 | 400 | 0.0244 |
| 30 | 0.85 | 400 | 0.0271 |
| 24 | 0.85 | 500 | 0.0157 |

- beta1 = 0.85 (fck <= 28), 0.85 - 0.007(fck-28) (fck > 28), 최소 0.65
- 실제 철근비가 rho_max를 초과하면 FAIL

## 3. 철근 간격 (KDS 41 12 00 8.7)

### 최소 간격

| 조건 | 최소 간격 | 비고 |
|------|----------|------|
| 수평 순간격 | max(25mm, db) | db: 철근 직경 |
| 상하 순간격 | max(25mm, db) | 2단 이상 배근 시 |
| 묶음철근 | 묶음 등가직경 기준 | dbs = db x sqrt(n) |

### 최대 간격

| 부재 | 최대 간격 | 비고 |
|------|----------|------|
| 슬래브 (주근) | min(3h, 450mm) | h: 슬래브 두께 |
| 슬래브 (배력근) | min(5h, 450mm) | |
| 보 (스터럽) | min(d/2, 600mm) | 일반 구간 |
| 보 (스터럽, Vs > 0.33sqrt(fck)bwd) | min(d/4, 300mm) | 전단보강 과다 구간 |

### 철근 간격 검토표

| 부재 | 철근 | db (mm) | 배근간격 (mm) | 최소 (mm) | 최대 (mm) | 판정 |
|------|------|---------|-------------|----------|----------|------|
| | | | | | | PASS/FAIL |

## 4. 정착길이 (KDS 41 12 00 8.2)

### 인장 정착길이

$$l_d = \frac{f_y \times d_b}{4 \times 1.4 \times \sqrt{f_{ck}}} \times \alpha \times \beta \times \gamma \times \lambda$$

### 보정계수

| 계수 | 조건 | 값 |
|------|------|------|
| alpha (철근 위치) | 상부근 (하부에 300mm 이상 콘크리트) | 1.3 |
| alpha | 기타 | 1.0 |
| beta (에폭시) | 에폭시 코팅 | 1.5 (피복 < 3db 또는 간격 < 6db), 1.2 (기타) |
| beta | 비코팅 | 1.0 |
| gamma (철근 크기) | D22 이하 | 0.8 |
| gamma | D25 이상 | 1.0 |
| lambda (경량콘크리트) | 보통콘크리트 | 1.0 |

- alpha x beta <= 1.7
- 최소 정착길이: 300mm 이상

### 정착길이 검토표

| 부재 | 위치 | db (mm) | 보정계수 | ld,req (mm) | ld,제공 (mm) | 판정 |
|------|------|---------|----------|------------|-------------|------|
| | | | | | | PASS/FAIL |

## 5. 겹침이음길이 (KDS 41 12 00 8.3)

### 이음등급

| 이음등급 | 적용 조건 | 이음길이 |
|----------|----------|----------|
| Class A | 이음부 As/As,req >= 2 이고, 이음비율 <= 50% | 1.0 x ld |
| Class B | 기타 모든 경우 | 1.3 x ld |

- 최소 이음길이: 300mm 이상
- 이음부 위치를 최대응력 구간에서 가급적 피한다

### 겹침이음 검토표

| 부재 | 위치 | 이음등급 | ld (mm) | 이음길이 (mm) | 제공 (mm) | 판정 |
|------|------|---------|---------|-------------|----------|------|
| | | | | | | PASS/FAIL |

## 6. 기둥 띠철근 간격 (KDS 41 12 00 8.5)

### 일반 기둥 띠철근 간격

$$s_{max} = \min(16d_b,\ 48d_t,\ b_{min})$$

| 파라미터 | 설명 |
|----------|------|
| db | 주근 직경 |
| dt | 띠철근 직경 |
| b_min | 기둥 최소 단면치수 |

### 내진 기둥 (SDC D 이상) 소성힌지 구간

$$s_{max} = \min\left(\frac{b}{4},\ 6d_b,\ 150mm\right)$$

- 소성힌지 구간 길이: max(h, b, L/6, 450mm) (기둥 양단)

### 기둥 띠철근 검토표

| 기둥 | db (mm) | dt (mm) | b_min (mm) | s_max (mm) | s_제공 (mm) | 판정 |
|------|---------|---------|-----------|-----------|------------|------|
| | | | | | | PASS/FAIL |

## 산출물

- 최소/최대 철근비 검토표 (부재별)
- 철근 간격 검토표
- 정착길이 검토표
- 겹침이음길이 검토표
- 기둥 띠철근 간격 검토표
- 배근 상세 종합 판정 (PASS/FAIL)

## 체크리스트

- [ ] 슬래브 최소철근비 (수축온도 철근) 확인
- [ ] 보 최소철근비 산정 및 확인
- [ ] 보 최대철근비 초과 여부 확인
- [ ] 기둥 철근비 0.01 ~ 0.08 범위 확인
- [ ] 철근 최소 순간격 확인 (25mm, db)
- [ ] 슬래브 최대 간격 min(3h, 450mm) 확인
- [ ] 정착길이 보정계수 적용 및 최소 300mm 확인
- [ ] 겹침이음등급(A/B) 판정 및 이음길이 확인
- [ ] 기둥 띠철근 간격 min(16db, 48dt, b_min) 확인
- [ ] 내진 상세 적용 시 소성힌지 구간 간격 별도 확인
- [ ] 모든 수치에 단위 및 기준 조항번호 명기
