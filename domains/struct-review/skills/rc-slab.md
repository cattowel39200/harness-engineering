# RC 슬래브 검토

KDS 41 12 00 기준에 따라 철근콘크리트 슬래브의 휨강도, 전단강도, 최소철근비, 철근간격을 검토하여 구조적 적합성을 확인한다.

---

## 규칙

1. **기준 준수**: KDS 41 12 00 (콘크리트구조 설계기준)을 적용한다
2. **모든 검토항목 필수**: 최소두께, 휨, 전단, 최소철근비, 철근간격 전부 검토한다
3. **판정 명시**: 각 항목에 대해 O.K. / N.G. 판정을 반드시 표기한다
4. **단위 통일**: kN, mm, MPa 단위계를 사용한다
5. **결과표 작성**: 최종 결과를 항목별 판정표로 정리한다

---

## 1. 최소두께 검토

KDS 41 12 00 Table 4.1에 의한 처짐 계산을 하지 않는 경우의 최소두께:

| 지지조건 | 최소두께 |
|----------|----------|
| 단순지지 | L / 20 |
| 일단연속 | L / 24 |
| 양단연속 | L / 28 |
| 캔틸레버 | L / 10 |

```
h_min = L / (해당 계수)
판정: h ≥ h_min → O.K.
```

- L: 슬래브 순경간 (mm)
- h: 슬래브 두께 (mm)

---

## 2. 휨강도 검토

설계휨모멘트에 대한 단면 강도를 확인한다.

```
a = As × fy / (0.85 × fck × b)
Mn = As × fy × (d - a/2)
φMn = 0.85 × Mn

판정: Mu ≤ φMn → O.K.
```

- Mu: 계수하중에 의한 설계휨모멘트 (kN·m)
- φ: 강도감소계수 = 0.85 (휨)
- As: 인장철근 단면적 (mm²)
- fy: 철근 항복강도 (MPa)
- fck: 콘크리트 설계기준압축강도 (MPa)
- b: 슬래브 폭 (= 1000mm, 단위폭)
- d: 유효깊이 (mm)
- a: 등가 직사각형 응력블록 깊이 (mm)

---

## 3. 전단강도 검토

슬래브는 일반적으로 전단철근을 배치하지 않으므로 콘크리트 전단강도로 검토한다.

```
Vc = (1/6) × √fck × b × d
φVc = 0.75 × Vc

판정: Vu ≤ φVc → O.K.
```

- Vu: 계수전단력 (kN)
- φ: 강도감소계수 = 0.75 (전단)
- Vc: 콘크리트 전단강도 (kN)

---

## 4. 최소철근비 검토

```
ρ_min = max(0.25 × √fck / fy, 1.4 / fy)
As_min = ρ_min × b × d

판정: As ≥ As_min → O.K.
```

- ρ_min: 최소철근비
- 수축·온도 철근비와 별도로 검토한다

---

## 5. 최대철근간격 검토

```
s_max = min(3h, 450mm)

판정: s ≤ s_max → O.K.
```

- s: 주철근 간격 (mm)
- h: 슬래브 두께 (mm)
