# 옹벽 구조검토

## 적용 대상
- 중력식 옹벽 / 캔틸레버 옹벽
- 높이 2m 이상

## 실행 순서
1. 설계조건 (옹벽 제원, 토질정수, 지반)
2. 하중 산정 (자중 + 토압 + 지진시 토압)
3. 하중조합
4. 옹벽 검토 (전도, 활동, 지지력, 벽체, 저판)
5. 검토서 생성

## 적용 기준
- KDS 11 80 05 (옹벽 설계기준)
