# Crack and Deflection Review

사용성 한계상태를 검토한다. 균열폭과 처짐(즉시처짐, 장기처짐)을 산정하여 허용값 이내임을 확인한다.

---

## 규칙

- 설계기준은 KDS 41 12 00 4.2 (사용성 검토)를 따른다
- 모든 검토항목에 기준 조항번호를 명시한다
- 수치 산출 시 반드시 단위를 표기하고 판정(PASS/FAIL)을 포함한다
- 처짐 산정 시 유효단면2차모멘트(Ie)를 반드시 사용한다
- 장기처짐 계수는 하중 지속기간에 따라 적용한다

## 1. 균열폭 검토 (KDS 41 12 00 4.2.4)

### 허용 균열폭

| 노출 조건 | 허용 균열폭 w (mm) | 비고 |
|-----------|-------------------|------|
| 일반 환경 (건조) | 0.3 | 보통 노출 |
| 부식 환경 (습윤/탄산화) | 0.2 | 부식 방지 |
| 심한 부식 (염화물/해안) | 0.1 | 내구성 확보 |

### 균열폭 산정

$$w = 1.1 \times k_b \times \beta \times \sqrt{d_c \times s}$$

| 파라미터 | 설명 | 비고 |
|----------|------|------|
| kb | 표면 피복 계수 | 보: 1.2, 슬래브: 1.0 |
| beta | 변형률 구배 계수 | = (h - kd) / (d - kd) |
| dc | 인장연 최외측 철근 중심~인장연 표면 거리 | mm |
| s | 인장철근 간격 | mm |

### 균열폭 검토표

| 부재 | 위치 | dc (mm) | s (mm) | w (mm) | 허용값 (mm) | 판정 |
|------|------|---------|--------|--------|------------|------|
| | | | | | | PASS/FAIL |

## 2. 즉시처짐 (KDS 41 12 00 4.2.3)

### 산정식

$$\Delta_i = \frac{5 \times w \times L^4}{384 \times E_c \times I_e}$$

### 유효단면2차모멘트 Ie

$$I_e = \left(\frac{M_{cr}}{M_a}\right)^3 \times I_g + \left[1 - \left(\frac{M_{cr}}{M_a}\right)^3\right] \times I_{cr}$$

| 파라미터 | 설명 | 산정식 |
|----------|------|--------|
| Mcr | 균열모멘트 | fr × Ig / yt |
| Ma | 사용하중 최대모멘트 | 해석결과 |
| Ig | 전단면 단면2차모멘트 | bh^3/12 (직사각형) |
| Icr | 균열단면 단면2차모멘트 | 변환단면법 |
| fr | 콘크리트 파괴계수 | 0.63 × sqrt(fck) MPa |
| Ec | 콘크리트 탄성계수 | 8,500 × (fck+4)^(1/3) MPa |

- Ma <= Mcr 인 경우: Ie = Ig (균열 미발생)
- Ma > Mcr 인 경우: Ie 산정 적용

## 3. 장기처짐 (KDS 41 12 00 4.2.3)

### 장기처짐 산정

$$\Delta_{lt} = \lambda_\Delta \times \Delta_i$$

### 장기처짐 계수

$$\lambda_\Delta = \frac{\xi}{1 + 50\rho'}$$

| 하중 지속기간 | xi | 비고 |
|--------------|------|------|
| 5년 이상 | 2.0 | 일반 적용 |
| 12개월 | 1.4 | |
| 6개월 | 1.2 | |
| 3개월 | 1.0 | |

| 파라미터 | 설명 | 비고 |
|----------|------|------|
| rho' | 압축철근비 (As'/bd) | 압축철근 없으면 0 |
| xi | 시간의존 계수 | 위 표 참조 |

- 총 처짐 = 즉시처짐(활하중) + 장기처짐(고정하중 + 지속활하중)
