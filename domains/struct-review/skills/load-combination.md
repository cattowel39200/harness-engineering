# Load Combination

산정된 각 하중을 조합하여 극한강도 설계 및 사용성 검토를 위한 하중조합을 구성한다. KDS 41 12 00 기준에 따라 하중조합을 작성하고 지배 하중조합을 결정한다.

---

## 규칙

- KDS 41 12 00 (콘크리트구조 설계기준) 5.2절 하중조합을 따른다
- 강도설계법(USD)의 하중계수를 적용한다
- 모든 하중조합을 검토하여 지배조합을 결정한다
- 풍하중과 지진하중은 동시에 작용하지 않는 것으로 본다

## 하중 기호

| 기호 | 하중 | 비고 |
|------|------|------|
| D | 고정하중 (Dead Load) | |
| L | 활하중 (Live Load) | |
| L_r | 지붕 활하중 (Roof Live Load) | |
| W | 풍하중 (Wind Load) | |
| E | 지진하중 (Seismic Load) | |
| S | 적설하중 (Snow Load) | 해당 시 |
| R | 강우하중 (Rain Load) | 해당 시 |
| T | 온도하중 (Temperature) | 해당 시 |

## 극한강도 하중조합 (KDS 41 12 00 5.2)

### 기본 조합

| 조합 | 하중조합식 | 적용 |
|------|----------|------|
| U1 | 1.4D | 고정하중 단독 |
| U2 | 1.2D + 1.6L + 0.5L_r | 활하중 지배 |
| U3 | 1.2D + 1.0L + 1.6L_r | 지붕 활하중 지배 |
| U4 | 1.2D + 1.0L + 1.0W | 풍하중 조합 |
| U5 | 1.2D + 1.0L + 1.0E | 지진하중 조합 |
| U6 | 0.9D + 1.0W | 풍하중 전도/부상 |
| U7 | 0.9D + 1.0E | 지진하중 전도/부상 |

### 주요 조합 상세

**U2: 1.2D + 1.6L** (가장 일반적인 지배조합)
- 연직하중 지배 부재: 슬래브, 보, 기둥 축력
- 대부분의 RC 부재 설계에서 지배

**U4: 1.2D + 1.0L + 1.0W** (풍하중 포함)
- 횡력 저항 부재: 기둥, 전단벽, 골조
- 풍방향 모멘트가 큰 부재

**U5: 1.2D + 1.0L + 1.0E** (지진하중 포함)
- 내진 부재: 기둥, 전단벽, 특수모멘트골조
- 지진 시 횡변위 및 전단력 검토

**U6: 0.9D + 1.0W** (전도 검토)
- 고정하중 감소 시 불리한 경우
- 인발력, 전도 모멘트 검토

**U7: 0.9D + 1.0E** (지진 시 전도)
- 지진하중에 의한 전도 검토
- 기초 인발력 검토

## 사용성 하중조합

### 서비스 조합

| 조합 | 하중조합식 | 적용 |
|------|----------|------|
| S1 | D + L | 처짐 검토 (전체 하중) |
| S2 | D + 0.7L | 장기 처짐 검토 |
| S3 | D + 0.5L + W | 풍하중 시 사용성 |
| S4 | D + 0.5L + 0.7E | 지진 시 사용성 |

### 처짐 검토 기준

| 부재 | 처짐 제한 | 하중 조합 |
|------|---------|----------|
| 비구조 부재 손상 우려 없음 | L/240 | S1: D + L |
| 비구조 부재 손상 우려 있음 | L/480 | S2: 추가 처짐 |
| 캔틸레버 | L/180 | S1: D + L |
| 바닥 진동 | L/360 | D + 0.3L |

## 지배 하중조합 결정

지배 하중조합은 부재의 종류와 설계 상태에 따라 달라진다.

| 부재 | 일반적 지배조합 | 검토 필요 조합 |
|------|-------------|-------------|
| 슬래브 | U2 (1.2D+1.6L) | - |
| 보 (중앙) | U2 (1.2D+1.6L) | U4, U5 (연속보) |
| 보 (단부) | U4 또는 U5 | 횡력 골조 |
| 기둥 (내부) | U2 (1.2D+1.6L) | 축력 지배 |
| 기둥 (외부) | U5 (1.2D+1.0L+1.0E) | 모멘트 지배 |
| 전단벽 | U5 (1.2D+1.0L+1.0E) | U7 (전도) |
| 기초 | U5 또는 U7 | 인발력 검토 |

## 산출물

- 극한강도 하중조합표 (U1~U7)
- 사용성 하중조합표 (S1~S4)
- 부재별 지배 하중조합 및 설계하중
- 지배조합 결정 근거
