# 보강토옹벽 외적안정 검토

보강토옹벽(MSEW: Mechanically Stabilized Earth Wall)의 보강토체를 하나의 중력식 블록으로 간주하여 전도, 활동, 지지력, 전체안정(원호활동)을 검토한다.

---

## 규칙

1. **결정론적 계산**: 동일 설계조건 → 동일 결과. 임의 가정 금지.
2. **전체 산식 출력**: 모든 계산은 6단계 형식을 반드시 따른다. 중간 과정을 생략하거나 압축하지 않는다.
3. **기준 조항 명시**: 모든 공식에 KDS 조항번호를 인용한다.
4. **단위 표기**: 모든 수치에 단위를 표기한다.
5. **안전율 판정**: 기준값과 비교하여 PASS/FAIL을 명시한다.
6. **가정값 표시**: 기준에 없는 값은 [가정] 표시 + 근거를 기재한다.
7. **블록 가정**: 보강토체 전체를 하나의 중력식 블록(폭 B, 높이 H)으로 간주한다.

### 6단계 산식 출력 형식 (모든 계산에 필수 적용)

```
① 기준 근거: KDS XX XX XX 조항번호 — 조항 제목
② 공식: 기준에 명시된 공식 원문
③ 변수 정의: 공식의 모든 변수 (의미 + 단위)
④ 값 대입: 각 변수에 실제 값을 넣은 식
⑤ 계산 과정: 단계별 산술 (중간값 전부 기재, 생략 금지)
⑥ 결과 판정: 계산값 vs 허용값 → PASS/FAIL
```

---

## 계산 순서 (8 Step, 순서 변경 금지)

```
Step 1: 설계조건 확인
Step 2: 토압 산정
Step 3: 보강토체 자중 산정
Step 4: 전도 안전율 검토
Step 5: 활동 안전율 검토
Step 6: 지지력 검토
Step 7: 전체안정 (원호활동) 검토
Step 8: 결과 종합표
```

---

## Step 1: 설계조건 확인

```
[필수 입력 — 하나라도 빠지면 진행 금지]

보강토체:
  벽체 높이 H        : _____ m
  보강재 길이 L       : _____ m (= 보강토체 폭 B)
  전면판 형식         : 콘크리트 패널 / 블록식 / 와이어메쉬
  보강재 형식         : 지오그리드 / 지오텍스타일 / 스틸스트립

보강토 단위중량 γ_r  : _____ kN/㎥ (보강토체 내 다짐토)

배면토(보강토체 뒤):
  단위중량 γs        : _____ kN/㎥
  내부마찰각 φ       : _____ °
  점착력 c           : _____ kPa

기초지반:
  허용지지력 qa      : _____ kN/㎡
  마찰계수 μ         : _____
  내부마찰각 φ_f     : _____ °

상재하중 q           : _____ kN/㎡ (있는 경우)

[고정 상수 — 변경 금지]
콘크리트 단위중량 γc  : 24.0 kN/㎥
```

---

## Step 2: 토압 산정

보강토체 배면에 작용하는 토압 (보강토체 뒤의 배면토에 의한 토압).

```
① 기준 근거: KDS 11 80 05 3.3.1 — Rankine 주동토압

② 공식:
   Ka = tan²(45° − φ/2)
   Pa = (1/2) × γs × H² × Ka
   상재하중 있을 때: Pa_q = q × H × Ka

③ 토압 합력:
   Pa_total = Pa + Pa_q
   Pah = Pa_total (수평 뒤채움)

④ 작용점:
   Pa에 의한 작용점: H/3
   Pa_q에 의한 작용점: H/2
   합산 작용점 = (Pa × H/3 + Pa_q × H/2) / Pa_total
```

---

## Step 3: 보강토체 자중 산정

보강토체를 하나의 블록(폭 B = L, 높이 H)으로 간주.

```
① 기준 근거: KDS 11 80 10 — 보강토옹벽

② 보강토체 무게:
   W_r = B × H × γ_r
       = [L] × [H] × [γ_r]
       = [W_r값] kN/m

③ 전면판 자중:
   W_f = 전면판 두께 × H × γc (콘크리트 패널 시)
   ※ 전면판 자중은 전체 대비 미소 → 무시 가능 [가정]

④ 상재하중에 의한 연직력:
   V_q = q × B = [q] × [L] = [V_q값] kN/m

⑤ 합계:
   ΣV = W_r + V_q = [W_r] + [V_q] = [ΣV값] kN/m

⑥ 저항 모멘트 (기준점: 전면 하단):
   ΣM_R = W_r × B/2 + V_q × B/2
        = [W_r] × [B/2] + [V_q] × [B/2]
        = [ΣM_R값] kN·m/m
```

---

## Step 4: 전도 안전율 검토

```
① 기준 근거: KDS 11 80 10 — 전도에 대한 안정

② 공식:
   FS_ot = ΣM_R / ΣM_O
   기준: FS_ot ≥ 2.0 (상시), FS_ot ≥ 1.5 (지진시)

③ 전도 모멘트:
   ΣM_O = Pa × H/3 + Pa_q × H/2

④ 계산:
   FS_ot = [ΣM_R] / [ΣM_O] = [FS_ot값]

⑥ 판정:
   FS_ot = [값] ≥ 2.0 (상시)
   → PASS ✅ 또는 FAIL ❌
```

---

## Step 5: 활동 안전율 검토

```
① 기준 근거: KDS 11 80 10 — 활동에 대한 안정

② 공식:
   FS_sl = (μ × ΣV) / ΣH
   기준: FS_sl ≥ 1.5 (상시), FS_sl ≥ 1.1 (지진시)

③ 저항력:
   μ × ΣV = tan(φ_f) × ΣV (기초지반 마찰)
   ※ μ = tan(min(φ_r, φ_f)) 또는 주어진 값

④ 활동력:
   ΣH = Pah

⑤ 계산:
   FS_sl = [저항력] / [Pah] = [FS_sl값]

⑥ 판정:
   FS_sl = [값] ≥ 1.5 (상시)
   → PASS ✅ 또는 FAIL ❌
```

---

## Step 6: 지지력 검토

```
① 기준 근거: KDS 11 80 10 — 지지력에 대한 안정

② 편심거리:
   e = B/2 − (ΣM_R − ΣM_O) / ΣV

③ 편심 확인:
   e ≤ B/6 (전단면 압축)

④ 유효 기초폭:
   B' = B − 2e (Meyerhof 유효폭)

⑤ 최대 지반반력:
   σ_max = ΣV / B' ≤ qa
   (또는: σ_max = (ΣV/B) × (1 + 6e/B) ≤ qa)

⑥ 판정:
   σ_max = [값] ≤ qa = [qa] kN/㎡
   → PASS ✅ 또는 FAIL ❌
```

---

## Step 7: 전체안정 (원호활동) 검토

```
① 기준 근거: KDS 11 80 10 — 전체안정

② 방법: Bishop 간편법 또는 절편법
   FS_global = Σ(c'×b + W×tanφ') / Σ(W×sinα)

③ 검토 범위:
   보강토체 + 배면토 + 기초지반을 포함하는 원호 활동면

④ 기준:
   FS_global ≥ 1.3 (상시)
   FS_global ≥ 1.1 (지진시)

⑤ 비고:
   전체안정 검토는 사면안정 프로그램(SLOPE/W 등) 사용이 일반적
   수계산 시 최소 3개 이상의 원호 활동면에 대해 검토

⑥ 판정:
   FS_global = [값] ≥ 1.3 (상시)
   → PASS ✅ 또는 FAIL ❌
```

## 산출물

1. **설계조건 확인서** (Step 1)
2. **토압 산정서** — Ka, Pa, 상재하중 영향 포함 (Step 2)
3. **보강토체 자중 산정서** — 블록 무게, 모멘트 (Step 3)
4. **안정성 검토서** — 전도/활동/지지력 (Step 4~6)
5. **전체안정 검토서** — 원호활동 (Step 7)
6. **결과 종합표** (Step 8)

---

## 체크리스트

- [ ] 모든 설계조건이 입력되었는가
- [ ] 보강토체를 블록(B×H)으로 정확히 모델링하였는가
- [ ] 보강토체 무게 = B × H × γ_r 가 정확한가
- [ ] 배면토 토압이 보강토체 배면에 작용하는 것으로 산정되었는가
- [ ] 전도 안전율 ≥ 2.0 (상시), ≥ 1.5 (지진시)
- [ ] 활동 안전율 ≥ 1.5 (상시), ≥ 1.1 (지진시)
- [ ] 최대 지반반력 ≤ 허용지지력
- [ ] 편심 e ≤ B/6
- [ ] 전체안정(원호활동) FS ≥ 1.3 (상시)
- [ ] 상재하중이 적용되었는가 (있는 경우)
- [ ] **모든 계산에 6단계 형식 적용되었는가**
- [ ] **중간 과정 생략이 없는가**
- [ ] **모든 수치에 단위가 표기되었는가**
