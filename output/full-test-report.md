## 4. 발견된 문제점 (BUG + ISSUE)

### 🔴 BUG (기능 오류)

| # | 문제 | 심각도 | 설명 |
|---|------|--------|------|
| B1 | **다운로드 구조계산서가 요약본** | 높음 | 12장 상세 양식이 아닌 ~40줄 요약만 출력. `generateReport` 함수가 옛날 코드 |
| B2 | **로그인 form_input 미연동** | 중간 | 외부 도구(Chrome MCP)로 입력 시 React state와 동기화 안 됨. 일반 사용자 직접 타이핑은 정상 |

### 🟡 ISSUE (미구현/부분구현)

| # | 문제 | 우선순위 | 설명 |
|---|------|---------|------|
| I1 | **의뢰→기술사 대기열 미연동** | 높음 | asj가 기술사 날인 의뢰해도 pss 대기열에 안 나타남 (localStorage 간 격리) |
| I2 | **관리자 버튼 미작동** | 중간 | 기술사 승인/거절 클릭 시 상태 실제로 안 바뀜 (UI만) |
| I3 | **QR 검증 고정 데이터** | 중간 | /verify/[code]가 항상 같은 목업 데이터 표시 |
| I4 | **스킬 마크다운 미렌더링** | 낮음 | 스킬 브라우저에서 마크다운이 pre태그로 출력 (##, ** 등 그대로 보임) |
| I5 | **헤더 사용자 깜빡임** | 낮음 | 페이지 이동 시 헤더 사용자 정보가 잠깐 사라졌다 나타남 (hydration) |
| I6 | **프로젝트 상세 다운로드도 요약본** | 높음 | B1과 동일 — 프로젝트 상세의 다운로드 버튼도 요약본 |
