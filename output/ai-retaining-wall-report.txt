## 제3장 토압 및 자중 산정

### 3.1 주동토압 산정

#### 3.1.1 주동토압계수

① 기준 근거: KDS 11 80 05 3.3.1 — Rankine 주동토압

② 공식:
   Ka = tan²(45° − φ/2)

③ 변수 정의:
   Ka = Rankine 주동토압계수 (무차원)
   φ  = 뒤채움 흙의 내부마찰각 (°) = 30°

④ 값 대입:
   Ka = tan²(45° − 30°/2)

⑤ 계산 과정:
   45° − φ/2 = 45° − 30°/2 = 45° − 15° = 30°
   tan(30°) = 0.577
   Ka = (0.577)² = 0.577 × 0.577 = 0.333

⑥ 결과:
   Ka = 0.333

#### 3.1.2 주동토압 합력

① 기준 근거: KDS 11 80 05 3.3.2 — 주동토압 합력

② 공식:
   Pa = (1/2) × γs × H² × Ka

③ 변수 정의:
   Pa = 주동토압 합력 (kN/m)
   γs = 뒤채움 흙의 단위중량 (kN/㎥) = 18 kN/㎥
   H  = 옹벽 높이 (m) = 5.0 m
   Ka = 주동토압계수 (무차원) = 0.333

④ 값 대입:
   Pa = (1/2) × 18 × 5.0² × 0.333

⑤ 계산 과정:
   H² = 5.0 × 5.0 = 25.0 m²
   (1/2) × γs = 0.5 × 18 = 9.0 kN/㎥
   9.0 × H² = 9.0 × 25.0 = 225.0 kN/m
   Pa = 225.0 × 0.333 = 74.93 kN/m

⑥ 결과:
   Pa = 74.93 kN/m
   수평성분 Pah = Pa × cos(0°) = 74.93 kN/m (수평 뒤채움)
   작용점 y_a = H / 3 = 5.0 / 3 = 1.67 m (저판 하면에서)

### 3.2 자중 산정

모멘트 기준점: 저판 앞꿈치 끝 하면 (전도 검토 기준점)

#### 3.2.1 벽체 자중

① 기준 근거: KDS 11 80 05 4.2 — 자중에 의한 안정

② 벽체 형상 및 자중 산정:
   
   벽체 높이 h_wall = H − D/1000 = 5.0 − 0.6 = 4.4 m
   벽체 단면: 사다리꼴 (상단 300mm, 하단 500mm)
   
   벽체 단면적:
   A_wall = (t1/1000 + t2/1000) / 2 × h_wall
          = (300/1000 + 500/1000) / 2 × 4.4
          = (0.3 + 0.5) / 2 × 4.4
          = 0.8 / 2 × 4.4
          = 0.4 × 4.4
          = 1.76 ㎡
   
   W_wall = A_wall × γc × 1.0 (단위폭)
          = 1.76 × 24.0
          = 42.24 kN/m

③ 벽체 도심 위치 (앞꿈치 끝에서):
   사다리꼴 도심까지 거리 = (t1 + 2×t2) / (3×(t1 + t2)) × h_wall (수평방향)
                          = (300 + 2×500) / (3×(300 + 500)) × (t2/1000)
                          = (300 + 1000) / (3×800) × 0.5
                          = 1300 / 2400 × 0.5
                          = 0.542 × 0.5
                          = 0.271 m (벽체 전면에서)
   
   x_wall = Lt + t2/1000 − 0.271
          = 0.8 + 0.5 − 0.271
          = 1.029 m

#### 3.2.2 저판 자중

① 저판 자중:
   W_base = B × (D/1000) × γc
          = 3.5 × 0.6 × 24.0
          = 50.4 kN/m

② 저판 도심 위치:
   x_base = B / 2 = 3.5 / 2 = 1.75 m

#### 3.2.3 뒤채움 토사 자중

① 뒤채움 토사 자중:
   뒤채움 높이 h_soil = h_wall = 4.4 m
   W_soil = Lh × h_soil × γs
          = 2.2 × 4.4 × 18
          = 174.24 kN/m

② 토사 도심 위치:
   x_soil = Lt + (t2/1000) + Lh/2
          = 0.8 + 0.5 + 2.2/2
          = 0.8 + 0.5 + 1.1
          = 2.4 m

#### 3.2.4 자중 합계

① 연직하중 합계:
   ΣV = W_wall + W_base + W_soil
      = 42.24 + 50.4 + 174.24
      = 266.88 kN/m

② 저항 모멘트 합계:
   ΣM_R = W_wall × x_wall + W_base × x_base + W_soil × x_soil
        = 42.24 × 1.029 + 50.4 × 1.75 + 174.24 × 2.4
        = 43.46 + 88.20 + 418.18
        = 549.84 kN·m/m

## 제4장 외적 안정성 검토

### 4.1 전도에 대한 안정성 검토

① 기준 근거: KDS 11 80 05 4.3.2 — 전도에 대한 안정

② 공식:
   FS_ot = ΣM_R / ΣM_O ≥ 1.5 (상시)

③ 변수 정의:
   FS_ot = 전도 안전율 (무차원)
   ΣM_R  = 저항 모멘트의 합 (kN·m/m) = 549.84 kN·m/m (Step 3에서 산정)
   ΣM_O  = 전도 모멘트의 합 (kN·m/m) — 토압에 의한 모멘트

④ 전도 모멘트 산정:
   ΣM_O = Pah × y_a
        = 74.93 × 1.67
        = 125.13 kN·m/m

⑤ 전도 안전율 계산:
   FS_ot = ΣM_R / ΣM_O
         = 549.84 / 125.13
         = 4.39

⑥ 판정:
   FS_ot = 4.39 ≥ 1.5 (기준값)
   → PASS ✅

### 4.2 활동에 대한 안정성 검토

① 기준 근거: KDS 11 80 05 4.3.3 — 활동에 대한 안정

② 공식:
   FS_sl = (ΣV × μ + Pp) / ΣH ≥ 1.5 (상시)

③ 변수 정의:
   FS_sl = 활동 안전율 (무차원)
   ΣV    = 연직하중의 합 (kN/m) = 266.88 kN/m
   μ     = 기초와 지반 사이의 마찰계수 = 0.5
   Pp    = 수동토압 (kN/m) = 0 (보수적으로 무시)
   ΣH    = 수평하중의 합 (kN/m) = Pah = 74.93 kN/m

④ 값 대입:
   FS_sl = (266.88 × 0.5 + 0) / 74.93

⑤ 계산 과정:
   ΣV × μ = 266.88 × 0.5 = 133.44 kN/m
   ΣV × μ + Pp = 133.44 + 0 = 133.44 kN/m
   FS_sl = 133.44 / 74.93 = 1.78

⑥ 판정:
   FS_sl = 1.78 ≥ 1.5 (기준값)
   → PASS ✅

### 4.3 지지력에 대한 안정성 검토

#### 4.3.1 편심 계산

① 기준 근거: KDS 11 80 05 4.3.4 — 지지력에 대한 안정

② 편심 계산:
   편심 e = |M_net| / ΣV

   M_net = ΣM_R − ΣM_O
         = 549.84 − 125.13
         = 424.71 kN·m/m

   e = 424.71 / 266.88 = 1.59 m

③ 기준점에서 합력 작용점:
   합력 작용점 = B/2 − e = 3.5/2 − 1.59 = 1.75 − 1.59 = 0.16 m (중심에서)

④ 유효폭 계산:
   중심에서 저판 끝까지 거리 = B/2 = 1.75 m
   편심이 중핵 내부에 있는지 확인: e ≤ B/6 = 3.5/6 = 0.583 m
   
   e = 1.59 m > B/6 = 0.583 m → 중핵을 벗어남
   
   유효폭 B' = 3 × (B/2 − e) = 3 × (1.75 − 1.59) = 3 × 0.16 = 0.48 m

#### 4.3.2 최대 지반 반력

① 기준 근거: KDS 11 80 05 4.3.4 — 지지력 검토

② 공식 (편심이 중핵을 벗어나는 경우):
   q_max = 2 × ΣV / B'

③ 변수 정의:
   q_max = 최대 지반 반력 (kN/㎡)
   ΣV    = 연직하중의 합 (kN/m) = 266.88 kN/m
   B'    = 유효폭 (m) = 0.48 m

④ 값 대입:
   q_max = 2 × 266.88 / 0.48

⑤ 계산 과정:
   2 × ΣV = 2 × 266.88 = 533.76 kN/m
   q_max = 533.76 / 0.48 = 1,112.0 kN/㎡

⑥ 판정:
   q_max = 1,112.0 kN/㎡ > qa = 250 kN/㎡ (허용지지력)
   → FAIL ❌
