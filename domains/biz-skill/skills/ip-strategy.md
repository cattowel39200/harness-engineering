# 지식재산권

특허, 상표, 디자인 등 지식재산권 확보 전략을 수립하고 정부사업 가점 및 기술 경쟁력 확보에 활용한다.

---

## 규칙

1. **사업 시작 전 출원**: 공개 전에 특허 출원을 완료해야 신규성이 유지된다
2. **핵심기술 우선**: 모든 기술이 아닌 경쟁우위의 핵심만 특허 출원한다
3. **비용 대비 효과**: 스타트업은 출원→등록 비용과 유지비를 고려해 전략적으로 선택한다
4. **IP + 사업 연계**: 특허 자체가 아닌 사업화 관점에서 IP 포트폴리오를 구성한다
5. **정부사업 가점 활용**: IP 보유 현황이 심사 가점에 반영되는 사업이 많다

---

## 특허 출원 전략

### 특허 유형별 전략

| 유형 | 대상 | 비용(출원) | 난이도 | 활용도 |
|------|------|-----------|--------|--------|
| 발명특허 | 핵심 기술/알고리즘 | 80~150만원 | 높음 | 기술 보호, 가점 |
| BM특허 | 비즈니스 방법 | 60~120만원 | 중간 | 서비스 보호 |
| 디자인특허 | UI/UX, 제품 외관 | 30~50만원 | 낮음 | 디자인 보호 |
| 실용신안 | 물품의 형상/구조 | 30~50만원 | 낮음 | 빠른 등록 |

**출원 우선순위**: 1순위 핵심기술 특허(필수) → 2순위 BM특허(차별점 명확 시) → 3순위 디자인특허(빠른 등록/가점) → 4순위 해외 PCT(글로벌 진출 시)

## 선행기술 조사

### 조사 도구

| 도구 | URL | 범위 | 비용 |
|------|-----|------|------|
| KIPRIS | kipris.or.kr | 한국 특허 전체 | 무료 |
| Google Patents | patents.google.com | 전세계 | 무료 |
| Espacenet | worldwide.espacenet.com | 유럽/전세계 | 무료 |
| WIPO | patentscope.wipo.int | PCT 국제출원 | 무료 |

**조사 절차**: 키워드 검색(한/영) → IPC 분류 검색 → 경쟁사 출원 현황 → 유사 특허 청구항 분석 → 차별점 정리

**핵심**: 선행기술과의 차이점이 곧 청구항의 핵심이다. 차별점이 불명확하면 등록 가능성이 낮다.

## 특허명세서 핵심 구성

```
[1] 발명의 명칭
    - 기술 내용을 구체적으로 반영 (예: "딥러닝 기반 실시간 이상 탐지 시스템")

[2] 요약서
    - 기술의 목적, 구성, 효과를 200자 내외로 요약

[3] 청구항 (Claims) ★ 가장 중요
    - 독립항: 발명의 핵심 구성요소 (넓게)
    - 종속항: 세부 특징 (좁게, 독립항 보호)
    - 방법 청구항 + 시스템 청구항 병행 권장

[4] 상세한 설명
    - 기술 분야, 배경기술, 해결 과제
    - 과제의 해결 수단 (구체적 구현 방법)
    - 실시예 (도면 참조 설명)
    - 발명의 효과 (정량적 개선 수치)

[5] 도면
    - 시스템 구성도, 플로우차트, UI 화면
    - 7~15장이 일반적
```

## 출원 타이밍

| 시점 | 장점 | 단점 | 권장 |
|------|------|------|------|
| 사업 시작 전 | 신규성 확보, 가점 활용 | 기술 미완성 | 핵심 아이디어 확정 시 |
| 사업 수행 중 | 기술 구체화 후 출원 | 공개 위험 | 발표/논문 전 반드시 |
| 사업 종료 후 | 완성된 기술 기반 | 가점 불가, 공개 후 | 비추천 |

**주의**: 논문 발표, 전시회 출품, 제품 출시 전에 반드시 출원해야 한다. 공개 후 6개월(국내) 또는 12개월(미국) 이내 출원해야 신규성 인정.

## 비용 가이드

| 항목 | 비용 (VAT 별도) | 비고 |
|------|----------------|------|
| 특허 출원 (대리인) | 80~150만원 | 명세서 작성 포함 |
| 특허 출원 (직접) | 4.6만원 | 관납료만 (비추천) |
| 심사청구 | 13~46만원 | 청구항 수에 따라 |
| 우선심사 신청 | 20만원 | 심사기간 2~4개월 단축 |
| 등록료 (1~3년) | 4.5만원/년 | 이후 점증 |
| 상표 출원 | 21~30만원 | 1류 기준 |
| 디자인 출원 | 30~50만원 | 1건 기준 |
| PCT 국제출원 | 300~500만원 | 국가 진입 별도 |

**스타트업 감면**: 중소기업 70% 감면, 개인 발명자도 유사 감면 가능. 정부사업 시제품 제작비에 특허 비용 편성 가능.

## 상표 출원

```
출원 대상: 서비스명(문자), 로고(도형), 결합상표
주요 분류: 9류(SW/앱), 35류(경영/쇼핑몰), 42류(IT/SaaS)
절차: 출원 → 심사(6~10개월) → 등록 → 10년(갱신 가능)
비용: 관납료 6.2만원 + 대리인 15~25만원
```

## 체크리스트

- [ ] 핵심기술에 대한 선행기술 조사를 완료했는가
- [ ] 출원할 특허 유형(발명/BM/디자인)을 결정했는가
- [ ] 청구항의 핵심 차별점이 선행기술과 구분되는가
- [ ] 출원 타이밍이 공개(발표, 출시) 이전인가
- [ ] 특허 비용이 사업비에 적절히 편성되었는가
- [ ] 서비스명/브랜드 상표 출원 계획이 있는가
- [ ] 스타트업 감면 혜택을 확인했는가
- [ ] 해외 출원 필요 여부를 검토했는가
