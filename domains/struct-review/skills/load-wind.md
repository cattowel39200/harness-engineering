# Wind Load

구조물에 작용하는 풍하중(Wind Load)을 산정한다. KDS 41 10 15 기준에 따라 기본풍속, 풍속압, 풍압계수를 산정하여 설계풍하중을 결정한다.

---

## 규칙

- KDS 41 10 15 (건축물 하중) 풍하중 규정을 따른다
- 기본풍속은 건설지역에 해당하는 값을 적용한다
- 높이 60m 이하 일반 건축물은 정적풍하중 해석을 적용할 수 있다
- 높이 60m 초과 또는 풍방향 진동이 중요한 구조물은 동적해석을 수행한다

## 설계속도압 산정

### 풍속압 공식

```
q_z = 0.5 x rho x V_z^2
```

- q_z : 높이 z에서의 속도압 (N/m2)
- rho : 공기밀도 = 1.225 kg/m3
- V_z : 높이 z에서의 설계풍속 (m/s)

### 설계풍속

```
V_z = V0 x K_zr x K_zt x I_w
```

- V0 : 기본풍속 (m/s)
- K_zr : 풍속고도분포계수
- K_zt : 지형계수 (평탄지 = 1.0)
- I_w : 중요도계수

### 중요도계수 I_w

| 중요도 | I_w | 적용 |
|--------|-----|------|
| 특 | 1.10 | 위험물 저장, 필수시설 |
| 1 | 1.10 | 연면적 1,000m2 이상 집회 |
| 2 | 1.00 | 일반 건축물 |
| 3 | 0.85 | 농업시설 등 |

## 지표면 조도구분

| 지표면 조도 | 구분 | 환경 |
|------------|------|------|
| A | 대도시 중심부 | 10층 이상 건물 밀집 |
| B | 중소도시 | 3~5층 건물 산재 |
| C | 개활지 | 높이 1.5~10m 장애물 산재 |
| D | 해안/초원 | 장애물 거의 없음 |

### 풍속고도분포계수 K_zr

```
K_zr = (z / z_G)^(2 x alpha)     (z_b <= z <= z_G)
K_zr = (z_b / z_G)^(2 x alpha)   (z < z_b)
```

| 조도구분 | alpha | z_b (m) | z_G (m) |
|---------|-------|---------|---------|
| A | 0.33 | 20 | 500 |
| B | 0.22 | 15 | 400 |
| C | 0.15 | 10 | 300 |
| D | 0.10 | 5 | 250 |

## 풍압계수

### 외압계수 Cpe

| 면 | Cpe | 비고 |
|----|-----|------|
| 바람받이면 (풍상면) | +0.8 | |
| 바람뒷면 (풍하면) | -0.5 | |
| 측면 | -0.7 | |
| 지붕 (풍상) | -0.9~+0.2 | 경사에 따라 |
| 지붕 (풍하) | -0.5 | |

### 내압계수 Cpi

| 조건 | Cpi |
|------|-----|
| 밀폐 건축물 | +/-0.2 |
| 부분밀폐 | +/-0.55 |

## 설계풍하중 산정

### 풍압력

```
p = q_z x G x (Cpe - Cpi)
```

- p : 설계풍압 (N/m2)
- G : 가스트영향계수 (폐합형 강구조/RC: G = 0.85 일반적)
- Cpe : 외압계수
- Cpi : 내압계수

### 층별 풍하중 산정 Template

| 층 | 높이 z (m) | K_zr | V_z (m/s) | q_z (N/m2) | 수압면적 A (m2) | F (kN) |
|----|----------|------|---------|----------|-------------|--------|
| 1F | ___ | ___ | ___ | ___ | ___ | ___ |
| 2F | ___ | ___ | ___ | ___ | ___ | ___ |
| ... | ___ | ___ | ___ | ___ | ___ | ___ |
| RF | ___ | ___ | ___ | ___ | ___ | ___ |
| **합계** | | | | | | **___** |

- F = p x A (각 층 풍하중)
- A = 수압면적 = 폭 x 부담높이
