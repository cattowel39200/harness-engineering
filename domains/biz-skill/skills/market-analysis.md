# 시장분석

선정된 아이디어의 시장을 정량적으로 분석하여 TAM/SAM/SOM을 도출하고, 경쟁사를 체계적으로 비교하여 시장 진입 전략의 근거를 확보한다.

---

## 규칙

1. **TAM/SAM/SOM 필수**: 시장규모는 반드시 3단계로 분류하며, 각 수치에 산출 근거와 출처를 명시한다
2. **출처 명시**: 모든 시장 데이터에는 공신력 있는 출처를 기재한다 (날짜 포함)
3. **최소 3개 경쟁사**: 직접 경쟁사 최소 3개를 분석하고, 간접 대안도 파악한다
4. **이중 추정법**: Top-down과 Bottom-up 두 가지 방법으로 시장규모를 교차검증한다
5. **보수적 추정**: 정부사업 심사에서는 과장된 수치보다 근거 있는 보수적 추정이 신뢰를 얻는다

---

## 시장규모 추정법

### Top-down 방식 (산업보고서 기반)

전체 산업 시장에서 세분화하여 목표 시장을 도출한다.

```
전체 산업 규모 (글로벌/국내)
  → 세분 시장 (기술/제품군 기준)
    → 타겟 시장 (지역/고객군 기준)
      → 실제 획득 가능 시장
```

**예시**: 국내 SaaS 시장 5조원 → HR SaaS 8,000억 → 중소기업 HR SaaS 2,400억 → 초기 확보 가능 240억

### Bottom-up 방식 (고객수 x 객단가)

실제 타겟 고객으로부터 역산하여 시장규모를 추정한다.

```
타겟 고객 수 x 연간 객단가 x 구매 전환율 = 시장규모

예시: 국내 중소기업 50만개 x HR솔루션 필요 비율 30%
      = 15만개 x 연 300만원 x 도입률 20% = 900억원
```

**두 방식의 결과가 2배 이상 차이나면 가정을 재점검한다.**

## TAM/SAM/SOM 작성 템플릿

```
┌─────────────────────────────────────────────────┐
│ TAM (Total Addressable Market) 전체 시장         │
│ 금액:          억원                              │
│ 근거:                                            │
│ 출처:                               (발행일:   ) │
├─────────────────────────────────────────────────┤
│ SAM (Serviceable Available Market) 유효 시장      │
│ 금액:          억원                              │
│ 근거: TAM에서 [조건]으로 세분화                   │
│ 출처:                               (발행일:   ) │
├─────────────────────────────────────────────────┤
│ SOM (Serviceable Obtainable Market) 수확 시장     │
│ 금액:          억원                              │
│ 근거: SAM에서 초기 [N]년 내 확보 가능 비율 적용   │
│ 목표 점유율:       % (근거:                    ) │
│ 출처:                               (발행일:   ) │
└─────────────────────────────────────────────────┘
```

**SOM 산출 기준**: 유사 서비스의 초기 3년 시장점유율 참고, 일반적으로 SAM의 1-5%

## 시장 성장률 및 CAGR

### CAGR 계산

```
CAGR = (최종연도 시장규모 / 기준연도 시장규모)^(1/기간) - 1

예시: 2023년 1,000억 → 2027년 1,800억 (4년)
CAGR = (1800/1000)^(1/4) - 1 = 15.8%
```

### 성장률 해석 기준

| CAGR | 해석 | 정부사업 매력도 |
|------|------|----------------|
| 0% 미만 | 축소 시장 | 낮음 (구조전환 논리 필요) |
| 0-5% | 성숙 시장 | 보통 (효율화 논리 가능) |
| 5-15% | 성장 시장 | 높음 (신규 진입 적기) |
| 15% 이상 | 고성장 시장 | 매우 높음 (혁신 필요 논리) |

## 산출물

1. **시장분석 보고서**: TAM/SAM/SOM + 성장률 + 출처 종합
2. **경쟁사 분석표**: 최소 3개 경쟁사의 4P + SWOT + 기능비교
3. **시장 데이터 출처 목록**: 사용한 모든 데이터의 출처, 발행일, 페이지 기록

## 체크리스트

- [ ] TAM/SAM/SOM을 모두 산출하고 각각 출처를 명시했는가
- [ ] Top-down과 Bottom-up 두 가지 방식으로 교차 검증했는가
- [ ] 시장 성장률(CAGR)을 계산하고 출처를 밝혔는가
- [ ] 직접 경쟁사를 최소 3개 이상 분석했는가
- [ ] 간접 대안(대체재)도 파악했는가
- [ ] 4P 비교표와 기능 비교표를 작성했는가
- [ ] 자사 SWOT 분석을 수행했는가
- [ ] 모든 데이터 출처가 공신력 있고 발행일이 최근 2년 이내인가
- [ ] SOM이 지나치게 낙관적이지 않은가 (SAM의 1-5% 수준)
- [ ] 심사위원에게 "시장이 충분히 크고 성장하고 있다"를 설득할 수 있는가
