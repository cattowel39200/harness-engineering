# 옹벽 구조검토

캔틸레버 옹벽 또는 중력식 옹벽의 외적 안정성(전도, 활동, 지지력)과 부재 단면(벽체, 저판)을 KDS 11 80 05에 따라 검토한다.

---

## 규칙

1. **결정론적 계산**: 동일 설계조건 → 동일 결과. 임의 가정 금지.
2. **전체 산식 출력**: 모든 계산은 6단계 형식을 반드시 따른다. 중간 과정을 생략하거나 압축하지 않는다.
3. **기준 조항 명시**: 모든 공식에 KDS 조항번호를 인용한다.
4. **단위 표기**: 모든 수치에 단위를 표기한다.
5. **안전율 판정**: 기준값과 비교하여 PASS/FAIL을 명시한다.
6. **가정값 표시**: 기준에 없는 값은 [가정] 표시 + 근거를 기재한다.

### 6단계 산식 출력 형식 (모든 계산에 필수 적용)

```
① 기준 근거: KDS XX XX XX 조항번호 — 조항 제목
② 공식: 기준에 명시된 공식 원문
③ 변수 정의: 공식의 모든 변수 (의미 + 단위)
④ 값 대입: 각 변수에 실제 값을 넣은 식
⑤ 계산 과정: 단계별 산술 (중간값 전부 기재, 생략 금지)
⑥ 결과 판정: 계산값 vs 허용값 → PASS/FAIL
```

## 계산 순서 (9 Step, 순서 변경 금지)

```
Step 1: 설계조건 확인
Step 2: 토압 산정
Step 3: 자중 산정
Step 4: 전도 안전율 검토
Step 5: 활동 안전율 검토
Step 6: 지지력 검토
Step 7: 벽체 단면 검토 (휨 + 전단)
Step 8: 저판 단면 검토 (앞꿈치 + 뒤꿈치)
Step 9: 결과 종합표
```

## Step 2: 토압 산정

### 2-1. 주동토압계수 (Rankine)

```
① 기준 근거: KDS 11 80 05 3.3.1 — Rankine 주동토압

② 공식:
   Ka = tan²(45° − φ/2)

③ 변수 정의:
   Ka = Rankine 주동토압계수 (무차원)
   φ  = 뒤채움 흙의 내부마찰각 (°)

④ 값 대입:
   Ka = tan²(45° − [φ값]/2)

⑤ 계산 과정:
   45° − φ/2 = 45° − [φ값]/2 = [결과]°
   tan([결과]°) = [탄젠트값]
   Ka = [탄젠트값]² = [탄젠트값] × [탄젠트값] = [Ka값]

⑥ 결과:
   Ka = [Ka값]
```

### 2-2. 주동토압 합력

```
① 기준 근거: KDS 11 80 05 3.3.2 — 주동토압 합력

② 공식:
   Pa = (1/2) × γs × H_total² × Ka

③ 변수 정의:
   Pa      = 주동토압 합력 (kN/m)
   γs      = 뒤채움 흙의 단위중량 (kN/㎥)
   H_total = 저판 하면에서 벽체 상단까지 전체 높이 (m) = H
   Ka      = 주동토압계수 (무차원)

④ 값 대입:
   Pa = (1/2) × [γs] × [H]² × [Ka]

⑤ 계산 과정:
   H² = [H] × [H] = [H²값] m²
   (1/2) × γs = 0.5 × [γs] = [절반값] kN/㎥
   [절반값] × [H²값] = [중간값] kN/m
   Pa = [중간값] × [Ka] = [Pa값] kN/m

⑥ 결과:
   Pa = [Pa값] kN/m
   수평성분 Pah = Pa × cos(0°) = [Pa값] kN/m (수평 뒤채움)
   작용점 y_a = H_total / 3 = [H] / 3 = [y_a값] m (저판 하면에서)
```

## Step 3: 자중 산정

모멘트 기준점: 저판 앞꿈치 끝 하면 (전도 검토 기준점)

```
① 기준 근거: KDS 11 80 05 4.2 — 자중에 의한 안정

② 각 부분 자중 및 모멘트 팔길이 (개별 산정, 생략 금지)

[벽체 자중]
  형상: 사다리꼴 (상단 t1, 하단 t2, 높이 h_wall = H − D/1000)
  면적 A_wall = (t1/1000 + t2/1000) / 2 × h_wall
              = ([t1]/1000 + [t2]/1000) / 2 × [h_wall]
              = ([t1+t2]/2000) × [h_wall]
              = [A_wall값] ㎡
  W_wall = A_wall × γc × 1.0(단위폭)
         = [A_wall값] × 24.0
         = [W_wall값] kN/m
  도심 x_wall = Lt + 벽체 도심 위치 (앞꿈치 끝에서)
              = [계산] m

[저판 자중]
  W_base = B × (D/1000) × γc
         = [B] × [D/1000] × 24.0
         = [W_base값] kN/m
  x_base = B / 2 = [B] / 2 = [x_base값] m

[뒤채움 토사 자중]
  h_soil = h_wall = H − D/1000 = [h_soil값] m
  W_soil = Lh × h_soil × γs
         = [Lh] × [h_soil] × [γs]
         = [W_soil값] kN/m
  x_soil = Lt + (t2/1000) + Lh/2
         = [x_soil값] m

[합계]
  ΣV = W_wall + W_base + W_soil
     = [W_wall] + [W_base] + [W_soil]
     = [ΣV값] kN/m

  ΣM_R = W_wall × x_wall + W_base × x_base + W_soil × x_soil
       = [W_wall] × [x_wall] + [W_base] × [x_base] + [W_soil] × [x_soil]
       = [항1] + [항2] + [항3]
       = [ΣM_R값] kN·m/m
```

## Step 4: 전도 안전율 검토

```
① 기준 근거: KDS 11 80 05 4.3.2 — 전도에 대한 안정

② 공식:
   FS_ot = ΣM_R / ΣM_O

   기준: FS_ot ≥ 1.5 (상시), FS_ot ≥ 1.2 (지진시)

③ 변수 정의:
   FS_ot = 전도 안전율 (무차원)
   ΣM_R  = 저항 모멘트의 합 (kN·m/m) — Step 3에서 산정
   ΣM_O  = 전도 모멘트의 합 (kN·m/m) — 토압에 의한 모멘트

④ 전도 모멘트 산정:
   ΣM_O = Pah × y_a
        = [Pah] × [y_a]

⑤ 계산 과정:
   ΣM_O = [Pah] × [y_a] = [ΣM_O값] kN·m/m

   ΣM_R = [ΣM_R값] kN·m/m (Step 3 결과)

   FS_ot = [ΣM_R값] / [ΣM_O값]
         = [FS_ot값]

⑥ 판정:
   FS_ot = [FS_ot값] ≥ 1.5 (상시 기준)
   → PASS ✅ 또는 FAIL ❌
```

## Step 5: 활동 안전율 검토

```
① 기준 근거: KDS 11 80 05 4.3.3 — 활동에 대한 안정

② 공식:
   FS_sl = (μ × ΣV) / ΣH

   기준: FS_sl ≥ 1.5 (상시), FS_sl ≥ 1.2 (지진시)

③ 변수 정의:
   FS_sl = 활동 안전율 (무차원)
   μ     = 기초저면 마찰계수 (무차원)
   ΣV    = 연직력의 합 (kN/m) — Step 3의 ΣV
   ΣH    = 수평력의 합 (kN/m) = Pah

④ 값 대입:
   저항력 = μ × ΣV = [μ] × [ΣV] = [저항력] kN/m
   활동력 = ΣH = Pah = [Pah] kN/m

⑤ 계산 과정:
   FS_sl = [저항력] / [Pah]
         = [FS_sl값]

⑥ 판정:
   FS_sl = [FS_sl값] ≥ 1.5 (상시 기준)
   → PASS ✅ 또는 FAIL ❌
```

## Step 6: 지지력 검토

```
① 기준 근거: KDS 11 80 05 4.3.4 — 지지력에 대한 안정

② 편심거리 공식:
   e = B/2 − (ΣM_R − ΣM_O) / ΣV

③ 변수 정의:
   e = 합력의 편심거리 (m)
   B = 저판 전체 폭 (m)

④ 값 대입 및 계산:
   e = [B]/2 − ([ΣM_R] − [ΣM_O]) / [ΣV]
   B/2 = [B/2값] m
   ΣM_R − ΣM_O = [차이값] kN·m/m
   (ΣM_R − ΣM_O)/ΣV = [비율값] m
   e = [B/2값] − [비율값] = [e값] m

⑤ 편심 확인:
   e = [e값] m, B/6 = [B]/6 = [B/6값] m
   e ≤ B/6 → 전단면 압축 확인

---

② 최대 지반반력 공식:
   σ_max = (ΣV/B) × (1 + 6e/B)

④ 값 대입 및 계산:
   ΣV/B = [ΣV]/[B] = [기본응력] kN/㎡
   6e/B = 6 × [e] / [B] = [편심비]
   1 + 6e/B = [증가계수]
   σ_max = [기본응력] × [증가계수] = [σ_max값] kN/㎡

⑥ 판정:
   σ_max = [σ_max값] kN/㎡ ≤ qa = [qa] kN/㎡
   → PASS ✅ 또는 FAIL ❌
```

## Step 7: 벽체 단면 검토

### 7-1. 벽체 휨강도

```
① 기준 근거: KDS 41 12 00 4.2.3 — 휨부재의 설계휨강도

② 공식:
   Mu ≤ φ × Mn
   Mn = As × fy × (d − a/2)
   a  = (As × fy) / (0.85 × fck × b)

③ 변수 정의:
   Mu  = 계수 휨모멘트 (kN·m/m)
   φ   = 휨 강도감소계수 = 0.85
   Mn  = 공칭 휨강도 (kN·m/m)
   As  = 인장철근 단면적 (mm²/m)
   fy  = 철근 항복강도 (MPa)
   d   = 유효깊이 (mm)
   a   = 등가 응력블록 깊이 (mm)
   fck = 콘크리트 강도 (MPa)
   b   = 1000 mm (단위폭)

④ 계수 휨모멘트 산정:
   h_wall = H − D/1000 = [값] m
   Pa_wall = 0.5 × γs × h_wall² × Ka
           = 0.5 × [γs] × [h_wall]² × [Ka]
           = 0.5 × [γs] × [h_wall²] × [Ka]
           = [Pa_wall] kN/m

   Mu = 1.6 × Pa_wall × h_wall / 3
      = 1.6 × [Pa_wall] × [h_wall] / 3
      = 1.6 × [Pa_wall] × [h_wall/3]
      = [Mu값] kN·m/m

⑤ 단면강도 산정:
   배근: D[직경] @ [간격] → As = [As값] mm²/m
   d = t2 − 피복 − db/2 = [t2] − 40 − [db/2] = [d값] mm

   a = ([As] × [fy]) / (0.85 × [fck] × 1000)
     = [분자] / [분모]
     = [a값] mm

   Mn = [As] × [fy] × ([d] − [a]/2) / 10⁶
      = [As] × [fy] × [유효팔] / 10⁶
      = [Mn값] kN·m/m

   φMn = 0.85 × [Mn값] = [φMn값] kN·m/m

⑥ 판정:
   Mu/φMn = [Mu값] / [φMn값] = [비율]
   [Mu값] ≤ [φMn값]
   → PASS ✅ 또는 FAIL ❌
```

### 7-2. 벽체 전단강도

```
① 기준 근거: KDS 41 12 00 5.3.1 — 콘크리트 전단강도

② 공식:
   Vu ≤ φ × Vc
   Vc = (1/6) × √fck × b × d

③ 변수 정의:
   Vu  = 계수 전단력 (kN/m)
   φ   = 전단 강도감소계수 = 0.75
   Vc  = 콘크리트 전단강도 (kN/m)

④ 계산:
   Vu = 1.6 × Pa_wall = 1.6 × [Pa_wall] = [Vu값] kN/m

   √fck = √[fck] = [루트값]
   Vc = (1/6) × [루트값] × 1000 × [d] / 1000
      = [Vc값] kN/m
   φVc = 0.75 × [Vc값] = [φVc값] kN/m

⑥ 판정:
   Vu/φVc = [Vu값] / [φVc값] = [비율]
   → PASS ✅ 또는 FAIL ❌
```

## Step 8: 저판 단면 검토

Step 7과 동일한 6단계 형식을 적용한다.

### 8-1. 뒤꿈치 휨 검토
- 하중: 뒤채움 토사하중(하향) − 순지반반력(상향)
- 위험단면: 벽체 뒷면
- 캔틸레버 길이: Lh

### 8-2. 앞꿈치 휨 검토
- 하중: 지반반력(상향) − 저판자중(하향)
- 위험단면: 벽체 앞면
- 캔틸레버 길이: Lt

### 8-3. 저판 전단 검토
- 위험단면: 벽체면에서 d 떨어진 위치
- Vu ≤ φVc (Step 7-2와 동일 형식)

## 산출물

1. **설계조건 확인서** (Step 1)
2. **토압 산정서** — Ka, Pa, 작용점 (Step 2)
3. **자중 산정서** — 부분별 무게, 모멘트 (Step 3)
4. **안정성 검토서** — 전도/활동/지지력 (Step 4~6)
5. **벽체 단면 검토서** — 휨/전단 6단계 산식 (Step 7)
6. **저판 단면 검토서** — 앞꿈치/뒤꿈치 6단계 산식 (Step 8)
7. **결과 종합표** (Step 9)

## 체크리스트

- [ ] 모든 설계조건이 입력되었는가 (미입력 항목 없음)
- [ ] 토압계수 Ka가 Rankine 공식으로 산정되었는가
- [ ] 전도 안전율 ≥ 1.5 (상시), ≥ 1.2 (지진시)
- [ ] 활동 안전율 ≥ 1.5 (상시), ≥ 1.2 (지진시)
- [ ] 최대 지반반력 ≤ 허용지지력
- [ ] 편심 e ≤ B/6 (전단면 압축 확인)
- [ ] 벽체 휨: Mu ≤ φMn (6단계 산식 전체 기재)
