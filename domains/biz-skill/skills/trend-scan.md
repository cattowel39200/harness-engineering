# 트렌드 탐색

정부지원사업 아이디어 발굴의 첫 단계로, 정부 정책방향과 산업/기술/사회 트렌드를 체계적으로 탐색하여 사업 기회를 도출한다.

---

## 규칙

1. **정부 정책방향 최우선**: 정부가 예산을 집중 투입하는 분야에서 출발한다
2. **최근 2-3년 트렌드 중심**: 너무 오래된 트렌드는 이미 레드오션, 너무 새로운 트렌드는 정부사업 미반영
3. **출처 명시 필수**: 모든 트렌드 정보에 출처(URL, 발행일, 기관명)를 기록한다
4. **교차 검증**: 하나의 트렌드를 최소 2개 이상의 출처에서 확인한다
5. **정량 데이터 우선**: "성장하고 있다" 대신 "연평균 15% 성장" 등 수치 기반으로 기술한다

---

## 탐색 영역

### 1. 정부 정책 방향
- 한국판 뉴딜 2.0 (디지털 뉴딜, 그린 뉴딜, 휴먼 뉴딜)
- 디지털 전환 (DX) 정책: AI, 클라우드, 데이터, 메타버스
- 탄소중립 2050: 친환경 에너지, 순환경제, 탄소배출권
- 지역균형발전: 지역 특화산업, 로컬크리에이터
- 규제샌드박스 대상 분야: 신기술 실증 허용 영역
- 국가전략기술: 반도체, 이차전지, 바이오, 우주항공, 양자

### 2. 산업 트렌드
- AI/ML 응용: 생성형AI, 엣지AI, AI에이전트
- 바이오/헬스케어: 디지털치료제, 정밀의료, 원격의료
- 우주/항공: 소형위성, 도심항공(UAM), 우주데이터
- 로봇/자동화: 협동로봇, 서비스로봇, 스마트팩토리
- 모빌리티: 자율주행, 전기차, MaaS

### 3. 기술 트렌드
- 특허 동향: KIPRIS 키워드 검색으로 출원 증가 분야 확인
- 논문 동향: Google Scholar, NTIS R&D 과제 트렌드
- 표준화 동향: 국제표준(ISO, IEEE) 신규 제정 분야
- 오픈소스 트렌드: GitHub Trending, 주요 프레임워크 성장세

### 4. 사회문제/니즈
- 인구구조: 고령화, 저출생, 1인가구 증가
- 환경: 미세먼지, 폐기물, 수질오염
- 안전: 산업재해, 자연재해, 사이버보안
- 교육: 디지털격차, 평생교육, 직무전환
- 지역소멸: 인구감소지역, 빈집 문제

## 트렌드 보고서 작성 템플릿

각 트렌드 항목별로 아래 형식으로 정리한다:

```
### [트렌드명]
- 분야: (정책/산업/기술/사회)
- 핵심 키워드: (3-5개)
- 정부 관심도: (상/중/하) — 근거:
- 시장 규모/성장률: (국내/글로벌)
- 기회 요인: (왜 지금 이 트렌드인가)
- 관련 정부사업: (확인된 공고명 또는 예산 항목)
- 출처: (URL, 발행일)
```

---

## 산출물

1. **트렌드 보고서**: 탐색 영역별 주요 트렌드 정리 (최소 10개 트렌드)
2. **기회 후보 리스트**: 트렌드에서 도출한 사업 기회 5-10개
   - 각 기회에 대해 한 줄 설명 + 관련 트렌드 + 정부사업 연계 가능성 표기

---

## 체크리스트

- [ ] 정부 정책 문서를 최소 3개 이상 확인했는가
- [ ] 최근 2년 이내 자료를 기반으로 작성했는가
- [ ] 모든 트렌드에 출처(URL, 날짜)가 명시되어 있는가
- [ ] 시장 규모를 정량적으로 기술했는가
- [ ] 정부 관심도를 근거와 함께 평가했는가
- [ ] 기회 후보가 5개 이상 도출되었는가
- [ ] 각 기회 후보에 관련 정부사업이 연결되어 있는가
- [ ] 너무 포괄적이지 않고 구체적인 기회로 좁혀졌는가
