# 프리셋 정의 (Input Presets)

구조검토에 반복적으로 사용되는 재료물성, 토질정수, 지진계수 등의 표준값을 정의한다. 설계조건 입력 시 프리셋을 참조하여 일관된 값을 적용한다.

---

## 규칙

1. **프리셋 우선 적용**: 설계조건이 미입력된 경우, 아래 프리셋에서 적합한 값을 제안한다.
2. **프리셋 변경 금지**: 아래 정의된 값은 기준에 근거하므로 임의 변경하지 않는다.
3. **프리셋 외 값 사용 시**: [사용자 지정] 표시 + 근거를 기재한다.
4. **단위 통일**: SI 단위계 (kN, MPa, mm, m, kN/㎡, kN/㎥)

---

## 1. 콘크리트 프리셋

### 1-1. 설계기준강도별 물성

| fck (MPa) | Ec (MPa) | fr (MPa) | 용도 예시 |
|-----------|----------|----------|-----------|
| 18 | 24,870 | 2.51 | 무근콘크리트 (중력식 옹벽, 기초) |
| 21 | 26,325 | 2.71 | 무근콘크리트, 소규모 RC |
| 24 | 27,662 | 2.90 | 일반 RC (옹벽, 암거, 슬래브) |
| 27 | 28,905 | 3.07 | 일반 RC (보, 기둥) |
| 30 | 30,070 | 3.24 | 고강도 RC (기둥, 벽체) |

```
산정 근거:
  Ec = 8,500 × ∛fck (MPa) — KDS 41 12 00 4.1.3
  fr = 0.63 × √fck (MPa) — KDS 41 12 00 4.2.1 (파괴계수)
```

### 1-2. 콘크리트 고정값

```
단위중량 γc = 24.0 kN/㎥ (KDS 41 10 15)
포아송비 ν = 0.18
열팽창계수 = 1.0 × 10⁻⁵ /°C
```

## 3. 뒤채움토 프리셋

| 분류 | γ (kN/㎥) | φ (°) | c (kPa) | Ka | 비고 |
|------|-----------|-------|---------|-----|------|
| 일반사질토 | 18 | 30 | 0 | 0.333 | 일반적 뒤채움 |
| 양질사질토 | 19 | 35 | 0 | 0.271 | 보강토옹벽 뒤채움 |
| 다짐쇄석 | 20 | 35 | 0 | 0.271 | 배수층, 보강토체 |
| 풍화토 | 18 | 25 | 5 | 0.406 | 자연지반 풍화토 |

```
Ka = tan²(45° − φ/2) — Rankine 주동토압계수
정지토압계수 K₀ = 1 − sinφ
  φ=25°: K₀ = 0.577
  φ=30°: K₀ = 0.500
  φ=35°: K₀ = 0.426
```

### 3-1. 마찰계수 프리셋

| 기초저면 조건 | μ (마찰계수) |
|--------------|-------------|
| 콘크리트-사질토 | tanφ (일반: 0.577) |
| 콘크리트-점성토 | 0.3~0.4 |
| 콘크리트-암반 | 0.6~0.7 |

## 4. 지반 프리셋

| 분류 | qa (kN/㎡) | 비고 |
|------|-----------|------|
| 연약 | 100 | 연약점토, N < 4 |
| 보통 | 200 | 풍화토, 4 ≤ N < 15 |
| 양호 | 300 | 다짐토, 15 ≤ N < 30 |
| 암반 | 500 | 연암 이상, N ≥ 50 |

```
허용지지력 qa 기준:
  - 지반조사 보고서 값 우선 적용
  - 지반조사 미시행 시 상기 프리셋 적용 [가정] 표시 필수
  - 건축물 기초: KDS 11 50 00
  - 토목구조물: KDS 11 10 15
```

## 5. 지진 프리셋

### 5-1. 지진구역

| 구역 | 지진구역계수 S (g) | 해당 지역 |
|------|-------------------|-----------|
| 1구역 | 0.22 | 서울, 수도권, 중부 대부분 |
| 2구역 | 0.14 | 남해안 일부, 제주도 |

### 5-2. 지반종류별 지반계수

| 지반종류 | Fa (단주기) | Fv (장주기) | 비고 |
|---------|------------|------------|------|
| S1 (경암) | 0.9 | 0.9 | Vs ≥ 1,500 m/s |
| S2 (보통암) | 1.0 | 1.0 | 760 ≤ Vs < 1,500 |
| S3 (풍화암) | 1.2 | 1.5 | 360 ≤ Vs < 760 |
| S4 (사질토) | 1.4 | 1.7 | 180 ≤ Vs < 360 |
| S5 (연약토) | 1.7 | 2.0 | Vs < 180 |

### 5-3. 옹벽 내진설계

```
수평지진계수 kh:
  1구역: kh = S × Fa / 2 = 0.22 × Fa / 2
  2구역: kh = S × Fa / 2 = 0.14 × Fa / 2

수직지진계수 kv:
  kv = 0 (일반적) 또는 kv = kh/2

Mononobe-Okabe 동적토압계수:
  KAE = 주동토압계수 (지진시)
  적용 기준: KDS 11 80 05 3.4
```

## 7. 보강재 프리셋

### 7-1. 지오그리드

| 등급 | Tult (kN/m) | 용도 | 비고 |
|------|-------------|------|------|
| 40 | 40 | 저벽(H≤3m) | 경량 보강 |
| 60 | 60 | 일반(H≤5m) | 표준 |
| 80 | 80 | 중벽(H≤7m) | |
| 100 | 100 | 고벽(H≤9m) | |
| 150 | 150 | 고벽(H≤12m) | |
| 200 | 200 | 특수(H>12m) | 고강도 |

```
감소계수 표준값 (지오그리드):
  RF_CR = 1.6 ~ 2.5 (크리프)
  RF_ID = 1.1 ~ 1.5 (시공손상)
  RF_D  = 1.1 ~ 1.3 (내구성)
  총 감소계수 RF = RF_CR × RF_ID × RF_D = 약 1.9 ~ 4.9
```

| 등급 | Tult (kN/m) | 용도 | 비고 |
|------|-------------|------|------|

| 40 | 40 | 일반(H≤4m) | |

| 80 | 80 | 고벽(H≤8m) | |

## 8. 프리셋 사용 규칙

```
1. 설계조건에 구체적 값이 있으면 → 설계조건 값 우선 적용
2. 설계조건에 값이 없으면 → 프리셋에서 적합한 값 제안
3. 프리셋 적용 시 → 반드시 "[프리셋: OO]" 표시
4. 프리셋에 없는 값 → "[사용자 지정]" 표시 + 근거
5. 지반조사/시험 결과가 있으면 → 시험값 우선 (프리셋보다 우선)
```

---

## 산출물

1. **재료물성 정리표** — 콘크리트, 철근 물성
2. **토질정수 정리표** — 뒤채움, 지반 정수
3. **지진계수 정리표** — 구역, 지반종류별 계수
4. **보강재 물성 정리표** — 지오그리드/지오텍스타일

---

## 체크리스트

- [ ] 콘크리트 fck에 대응하는 Ec가 정확한가
- [ ] 철근 fy가 SD400 또는 SD500 중 하나인가
- [ ] 뒤채움토 물성(γ, φ, c)이 실제 조건에 부합하는가
- [ ] 허용지지력 qa가 지반조사 결과 또는 프리셋에 부합하는가
- [ ] 지진구역(1/2구역)이 정확한가
- [ ] 석재 단위중량이 석재 종류에 맞는가 (화강석 26, 자연석 25)
- [ ] 보강재 극한인장강도 Tult가 제품 사양에 맞는가
- [ ] 감소계수(RF_CR, RF_ID, RF_D)의 근거가 있는가
- [ ] 프리셋 적용 시 "[프리셋: OO]" 표시가 되어 있는가
- [ ] 지반조사/시험 결과가 있을 때 시험값이 프리셋보다 우선 적용되었는가
