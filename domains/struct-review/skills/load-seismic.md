# Seismic Load

구조물에 작용하는 지진하중(Seismic Load)을 산정한다. KDS 41 17 00 기준에 따라 내진설계 변수를 결정하고, 등가정적해석법으로 밑면전단력 및 층별 지진하중을 산출한다.

---

## 규칙

- KDS 41 17 00 (건축물 내진설계기준)을 따른다
- 내진등급에 따른 내진설계 수행 여부를 먼저 판단한다
- 등가정적해석법은 비정형성이 없고 높이 제한을 만족하는 건축물에 적용 가능하다
- 내진설계범주(SDC)에 따라 구조 시스템 제한 및 상세 요구를 확인한다

## 지진위험도 변수

### 지역계수 S

| 지진구역 | S | 적용 지역 |
|---------|---|----------|
| 1 | 0.22g | 서울, 부산, 대구, 인천, 광주, 대전 등 |
| 2 | 0.14g | 강원 북부, 제주 등 |

- 2400년 재현기간 기준 (최대고려지진)
- 설계지진: S x (2/3) 적용

### 지반종류

| 지반종류 | Vs30 (m/s) | 설명 |
|---------|-----------|------|
| S1 | > 760 | 암반 |
| S2 | 360~760 | 얕은 연암/매우 조밀한 토사 |
| S3 | 180~360 | 조밀한 토사/연암 |
| S4 | < 180 | 연약한 토사 |
| S5 | - | 부지고유 특성 평가 필요 |

- 지반조사 결과 없을 시 S3 적용 (보수적)

### 지반계수

| 지반종류 | Fa (단주기) | Fv (장주기) |
|---------|-----------|-----------|
| S1 | 0.9 | 0.9 |
| S2 | 1.0 | 1.0 |
| S3 | 1.2 | 1.5 |
| S4 | 1.6 | 2.2 |

- S = 0.22g 기준 (지진구역 1)

## 설계응답스펙트럼

### 설계스펙트럼 가속도

```
SDS = (2/3) x Fa x S
SD1 = (2/3) x Fv x S
```

- SDS : 단주기 설계스펙트럼 가속도
- SD1 : 1초주기 설계스펙트럼 가속도

### 서울 지역 (S=0.22g, S3 지반) 예시

```
SDS = (2/3) x 1.2 x 0.22 = 0.176g
SD1 = (2/3) x 1.5 x 0.22 = 0.220g
```

### 스펙트럼 주기

```
T0 = 0.2 x (SD1 / SDS)
TS = SD1 / SDS
```

## 중요도계수 Ie

| 내진등급 | Ie | 적용 |
|---------|----|----|
| 특 | 1.5 | 원자력, 위험물 대량 저장 |
| I | 1.2 | 병원, 소방서, 통신시설 |
| II | 1.0 | 일반 건축물 |
| III | 0.8 | 농업시설, 가설건축물 |

## 내진설계범주 (SDC)

### SDC 결정

| SDS / SD1 범위 | 내진등급 II | 내진등급 I |
|---------------|-----------|-----------|
| SDS < 0.167g | A | A |
| 0.167g <= SDS < 0.33g | B | C |
| 0.33g <= SDS < 0.50g | C | D |
| SDS >= 0.50g | D | D |

- 서울, S3 지반, 일반건물: SDS = 0.176g -> SDC = B

## 구조 시스템 (반응수정계수 R)

| 구조 시스템 | R | Omega_0 | Cd |
|-----------|---|---------|-----|
| 철근콘크리트 보통모멘트골조 | 3.0 | 3.0 | 2.5 |
| 철근콘크리트 중간모멘트골조 | 5.0 | 3.0 | 4.5 |
| 철근콘크리트 특수모멘트골조 | 8.0 | 3.0 | 5.5 |
| RC 보통전단벽 | 4.0 | 2.5 | 4.0 |
| RC 특수전단벽 | 5.0 | 2.5 | 5.0 |
| RC 이중골조 (특수+특수) | 7.0 | 2.5 | 5.5 |

- SDC B: 보통모멘트골조 이상
- SDC C: 중간모멘트골조 이상
- SDC D: 특수모멘트골조 또는 특수전단벽

## 등가정적해석법

### 밑면전단력 V

```
V = Cs x W
```

- W : 건물 유효중량 (고정하중 + 활하중의 일부)

### 지진응답계수 Cs

```
Cs = SDS / (R / Ie)                          (기본)
Cs <= SD1 / (T x (R / Ie))                   (T <= TL)
Cs >= 0.044 x SDS x Ie >= 0.01               (최소)
```

- T : 구조물 고유주기

### 고유주기 약산식

```
T = Ct x h_n^x
```

| 구조형식 | Ct | x |
|---------|-----|---|
| RC 모멘트골조 | 0.0466 | 0.9 |
| 철골 모멘트골조 | 0.0724 | 0.8 |
| RC 전단벽/기타 | 0.0488 | 0.75 |

- h_n : 건물 높이 (m)

### 층별 지진하중 배분

```
F_x = Cvx x V
Cvx = (w_x x h_x^k) / SUM(w_i x h_i^k)
```

- k = 1 (T <= 0.5s), k = 2 (T >= 2.5s), 선형보간 (0.5s < T < 2.5s)
- w_x : x층 중량
- h_x : x층 높이

### 층별 지진하중 산정 Template
