# 도면 판독 (Drawing Reader)

구조도면(DWG, PDF, 이미지)에서 구조검토에 필요한 정보를 정확하게 추출한다. 도면을 잘못 읽으면 이후 모든 계산이 틀리므로, **도면 판독은 구조검토의 첫 번째이자 가장 중요한 단계**이다.

---

## 규칙

1. **추정 금지**: 도면에서 읽을 수 없는 값은 "판독 불가"로 표기한다. 추측하지 않는다.
2. **표고 기준**: 도면의 표고(EL.)를 기준으로 높이를 산정한다. 치수선이 있으면 치수선을 우선한다.
3. **방향 확인 필수**: 하천측/제내지측, 성토측/절토측, 전면/배면을 반드시 확인한다.
4. **단면도 방향**: 단면도의 좌/우가 실제 어느 방향인지 도면 범례와 평면도를 대조한다.
5. **축척 확인**: 도면 축척을 확인하고, 치수선이 없는 부분은 축척으로 환산하지 않는다.

---

## 도면에서 읽어야 할 항목

### 1. 구조물 형식 판별

```
도면에서 확인할 것:
□ 구조물 명칭 (타이틀 블록, 범례)
□ 단면 형상 → 형식 판별
□ 하중 방향 표시 (화살표, 수위선, 토압 방향)
□ 지반선/지표면 위치
```

### 2. 치수 읽기

```
[옹벽의 경우]
□ 전체 높이 H (저판 하면 ~ 벽체 상단)
□ 벽체 높이 (저판 상면 ~ 벽체 상단)
□ 벽체 상단 두께 t1
□ 벽체 하단 두께 t2
□ 저판 전체 폭 B
□ 저판 두께 D
□ 앞꿈치 길이 (벽체 전면 ~ 저판 끝)
□ 뒤꿈치 길이 (벽체 배면 ~ 저판 끝)

[건축물의 경우]
□ 층수, 층고
□ 스팬 (기둥 간격)
□ 부재 단면 크기 (보, 기둥, 슬래브 두께)
```

### 3. 표고·수위 읽기

```
[하천 옹벽의 경우]
□ 옹벽 상단고 (EL. ___m)
□ 계획홍수위 HWL (EL. ___m)
□ 평수위 (EL. ___m)
□ 저판 상면 표고 (EL. ___m)
□ 저판 하면 표고 (EL. ___m)
□ 기존 지반고 (EL. ___m)
□ 계획 지반고 (EL. ___m)

높이 관계 산정:
  벽체 높이 = 옹벽 상단고 - 저판 상면고
  저판 두께 = 저판 상면고 - 저판 하면고
  작용 수심 = HWL - 저판 상면고
  여유고    = 옹벽 상단고 - HWL
```

### 4. 방향 판별 (핵심!)

```
┌──────────────────────────────────────────────────┐
│ 도면에서 방향을 판별하는 방법                      │
│                                                    │
│ 1. 평면도의 하천 흐름 방향 확인                    │
│ 2. 단면도에 "하천측", "제내지측" 표기 확인         │
│ 3. 수위선(HWL) 표시 위치 = 하천측                 │
│ 4. 뒤채움 해칭 위치 = 배면(토압측)                │
│ 5. 배수공 위치 = 하천측(배수 방향)                │
│                                                    │
│ 주의:                                              │
│ - 단면도 좌/우가 실제 방향과 반대일 수 있음        │
│ - 반드시 평면도와 대조하여 확인                    │
│ - 화살표 표기가 있으면 최우선                      │
└──────────────────────────────────────────────────┘
```

### 5. 배근 읽기

```
□ 주철근 (직경, 간격): D__ @ ___mm
□ 배력근 (직경, 간격): D__ @ ___mm
□ 피복두께
□ 배근 위치 (전면/배면, 상면/하면)
```

## DWG 파일 판독 방법

DWG는 바이너리 CAD 파일이므로 직접 텍스트로 읽을 수 없다.

```
판독 방법 (우선순위):
1. 사용자가 제공한 설계조건 텍스트 → 최우선
2. 동일 폴더의 구조계산서(.txt, .docx, .pdf) → 설계조건 추출
3. 프로젝트 폴더의 관련 문서 → 참조
4. 파일명에서 추출 가능한 정보 (예: "H=2.0" → 높이 2.0m)
5. 사용자에게 설계조건 질문

절대 하지 않을 것:
✗ DWG 파일을 추측으로 해석
✗ 일반적인 값으로 가정
✗ 확인되지 않은 치수를 사용
```

## 산출물

1. **도면 판독 결과표**: 구조물 형식, 치수, 표고, 방향, 배근
2. **미확인 항목 목록**: 도면에서 확인 불가한 항목
3. **추가 확인 요청**: 사용자에게 질문할 항목

---

## 체크리스트

- [ ] 구조물 형식이 정확히 판별되었는가
- [ ] 하중 방향(수압/토압)이 확인되었는가
- [ ] 모든 치수가 도면 또는 계산서에서 확인되었는가
- [ ] 표고 관계가 정리되었는가
- [ ] 추정값이 있으면 [가정]으로 표기되었는가
- [ ] 방향(하천측/제내지측)이 평면도와 대조되었는가
