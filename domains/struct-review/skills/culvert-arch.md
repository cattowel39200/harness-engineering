# ARCH 암거 / 파이프 구조검토

ARCH형 암거 또는 파이프(원형/타원형)에 대해 토압, 관두께, 파쇄하중, 좌굴, 이음부 강도를 검토한다.

---

## 규칙

1. **결정론적 계산**: 동일 설계조건 → 동일 결과. 임의 가정 금지.
2. **전체 산식 출력**: 모든 계산은 6단계 형식을 반드시 따른다. 중간 과정을 생략하거나 압축하지 않는다.
3. **기준 조항 명시**: 모든 공식에 KDS 조항번호를 인용한다.
4. **단위 표기**: 모든 수치에 단위를 표기한다.
5. **안전율 판정**: 기준값과 비교하여 PASS/FAIL을 명시한다.
6. **가정값 표시**: 기준에 없는 값은 [가정] 표시 + 근거를 기재한다.

### 6단계 산식 출력 형식 (모든 계산에 필수 적용)

```
① 기준 근거: KDS XX XX XX 조항번호 — 조항 제목
② 공식: 기준에 명시된 공식 원문
③ 변수 정의: 공식의 모든 변수 (의미 + 단위)
④ 값 대입: 각 변수에 실제 값을 넣은 식
⑤ 계산 과정: 단계별 산술 (중간값 전부 기재, 생략 금지)
⑥ 결과 판정: 계산값 vs 허용값 → PASS/FAIL
```

---

## 계산 순서 (8 Step, 순서 변경 금지)

```
Step 1: 설계조건 확인
Step 2: 토압 산정 (연직 + 수평)
Step 3: 활하중 분배 (DB하중)
Step 4: 파쇄하중 (Crushing Strength) 검토
Step 5: 좌굴 검토 (Buckling)
Step 6: 관두께 검토
Step 7: 이음부 강도 검토
Step 8: 결과 종합표
```

## Step 2: 토압 산정

### 2-1. 연직토압

```
① 기준 근거: KDS 24 14 21 — 매설관 연직토압

② 공식:
   돌출형: W_v = Cc × γs × H_c × D_o/1000
   함몰형: W_v = Cc × γs × H_c × B_d

③ 변수 정의:
   W_v  = 연직토압 (kN/m)
   Cc   = 매설계수 (무차원)
   γs   = 뒤채움 단위중량 (kN/㎥)
   H_c  = 토피고 (m)
   D_o  = 외경 (mm) → m 환산
   B_d  = 도랑폭 (m) — 함몰형

④ 계산 과정: (6단계 형식)

⑥ 결과:
   W_v = [W_v값] kN/m
```

### 2-2. 수평토압

```
수평토압 = K₀ × γs × (H_c + D_o/2000)
K₀ = 1 − sinφ
```

## Step 3: 활하중 분배

```
① 기준 근거: KDS 24 14 21 — 활하중 분배

② DB-24 후륜하중:
   P_wheel = 96 kN
   분배폭 = 바퀴 접지폭 + 2 × H_c × tan(30°)
   충격계수 i = 15 / (40 + L) (단, i ≤ 0.3)

③ 관 정상부 등분포 환산:
   w_L = P_wheel × (1 + i) / 분배폭 (kN/m)
```

---

## Step 4: 파쇄하중 검토

```
① 기준 근거: KS F 4403 / KDS 24 14 21 — 파쇄하중

② 공식:
   FS = 파쇄하중 / 설계하중 ≥ 안전율

③ 설계하중:
   W_total = W_v + w_L (kN/m)

④ 파쇄하중 (3등분점 하중 시험 기준):
   RC관: 제조사 시험값 또는 KS 기준표
   강관: 재료강도 기반 산정

⑤ 안전율:
   RC관: FS ≥ 1.5
   강관/HDPE: FS ≥ 2.0

⑥ 판정:
   FS = [FS값] ≥ [기준] → PASS ✅ 또는 FAIL ❌
```

---

## Step 5: 좌굴 검토

```
① 기준 근거: KDS 24 14 21 — 매설관 좌굴

② 공식 (연성관):
   q_cr = 32 × R_w × B' × E' × (EI / D³)

   연성관(강관, HDPE):
   좌굴 안전율 FS_b = q_cr / q_design ≥ 2.5

③ 변수 정의:
   q_cr   = 좌굴 임계압력 (kPa)
   R_w    = 물보정계수 (= 1.0, 지하수 없음)
   B'     = 탄성지지계수 (= 1/(1+4×e^(-0.065×H_c)))
   E'     = 뒤채움 반력계수 (kPa)
   EI     = 관의 휨강성 (kN·m²/m)
   D      = 관 직경 (m)

④ 강성관(RC관):
   좌굴 검토 생략 가능 (관두께/직경비에 따라)
   t/D > 1/10 이면 좌굴 지배하지 않음

⑥ 판정:
   FS_b = [값] ≥ 2.5 → PASS ✅ 또는 FAIL ❌
```

---

## Step 6: 관두께 검토

```
① 기준 근거: KDS 24 14 21 — 관두께

② 최소 관두께:
   RC관: t_min = D_i / 12 (경험식) 또는 KS 기준표
   강관: t_min = 산정 (외압 + 내압 조합)
   HDPE: t_min = 장기강도 기반

③ 판정:
   t_actual = [t] mm ≥ t_min = [t_min] mm
   → PASS ✅ 또는 FAIL ❌
```

---

## Step 7: 이음부 강도 검토

```
① 기준 근거: KDS 24 14 21 — 이음부

② 검토 항목:
   (a) 전단강도: 이음부 전단 ≥ 설계 전단력
   (b) 수밀성: 내수압 시험 합격 기준
   (c) 변위허용: 이음부 허용 변위 ≥ 설계 변위

③ 이음 형식:
   소켓 이음, 칼라 이음, 용접 이음, 플랜지 이음

④ 판정:
   이음부 강도 ≥ 관체 강도의 [기준비율]%
   → PASS ✅ 또는 FAIL ❌
```

## 체크리스트

- [ ] 모든 설계조건이 입력되었는가 (미입력 항목 없음)
- [ ] 매설 조건(돌출/함몰)이 확인되었는가
- [ ] 매설계수 Cc가 정확히 적용되었는가
- [ ] 연직토압이 정확히 산정되었는가
- [ ] DB하중 분배폭이 토피고 기반으로 산정되었는가
- [ ] 파쇄하중 안전율 ≥ 1.5 (RC관) 또는 ≥ 2.0 (강관/HDPE)
- [ ] 좌굴 검토 (연성관): FS ≥ 2.5
- [ ] 관두께 ≥ 최소 관두께
- [ ] 이음부 강도가 확인되었는가
- [ ] **모든 계산에 6단계 형식 적용되었는가**
- [ ] **중간 과정 생략이 없는가**
- [ ] **모든 수치에 단위가 표기되었는가**
