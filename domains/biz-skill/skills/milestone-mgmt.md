# 사업관리

정부사업 협약 체결부터 최종평가까지 분기별 보고, 중간점검, 정산, 변경 관리를 체계적으로 수행한다.

---

## 규칙

1. **일정 엄수**: 보고서 제출 기한을 반드시 지킨다. 지연 시 사업비 환수 위험이 있다
2. **증빙 즉시 정리**: 지출 발생 즉시 증빙을 정리한다. 나중에 모아서 하면 누락된다
3. **변경은 사전 승인**: 사업 내용, 기간, 비목 변경은 반드시 사전에 승인을 받는다
4. **성과 수시 기록**: 특허, 매출, 고용 등 성과 발생 시 즉시 기록하고 증빙을 확보한다
5. **소통 유지**: 담당 매니저와 정기적으로 소통하여 이슈를 조기에 해결한다

---

## 협약 체결 절차

```
Step 1: 선정 통보 → 시스템 로그인 확인
Step 2: 사업계획서 보완 (심사 지적사항 반영, 사업비 조정)
Step 3: 협약 서류 준비 (사업자등록증, 4대보험, 전용통장, 인감)
Step 4: 전자 협약 체결 (기간, 금액, 의무사항 확인)
Step 5: 사업비 수령 (전용 계좌 입금, 사업비 카드 발급)
```

**협약 시 필수 확인**: 사업기간, 사업비 총액(출연금+자부담), 비목별 한도, 보고 의무, 변경 승인 기준

---

## 분기별 보고서 작성 가이드

```
1. 사업 개요 (1p) - 과제명, 기관, 기간, 사업비
2. 추진 실적 (3~5p) - 계획 vs 실적, 마일스톤 달성률, 성과물
3. 사업비 집행 (1~2p) - 비목별 집행률, 주요 지출
4. 향후 계획 (1p) - 차분기 계획, 이슈 대응
5. 첨부 - 결과물, 회의록, 출장보고서
```

### 분기별 집행률 가이드

```
구분      Q1       Q2       Q3       Q4
목표      15~20%   35~45%   65~75%   95~100%
위험      10% 미만  25% 미만  50% 미만  80% 미만
조치      -        주의      시정요구   환수 위험
```

---

## 중간점검 대응

```
시기: 사업기간 50% 시점 (보통 6개월차)
방식: 서면 또는 현장 점검 (방문/화상)
평가단: 외부 전문가 2~3명 + 전담기관 매니저

준비물:
[1] 실적 보고서 - 계획 vs 실적 비교표, 달성률, 만회 계획
[2] 성과물 시연 - 시제품/MVP 데모, 기술 검증 결과
[3] 집행 현황 - 비목별 집행률 (50% 전후 적정)
[4] 변경 사항 - 변경 내용, 사유, 승인 여부
```

---

## 최종평가 보고서 구조

```
1. 총괄 요약 (2p) - 목표 및 달성 현황
2. 연구개발 성과 (5~10p) - 달성률, 기술 결과, 시제품 완성도
3. 사업화 성과 (3~5p) - 매출, 고용, 투자유치
4. 지식재산권 (2p) - 특허, 상표, 프로그램 등록
5. 사업비 정산 (3~5p) - 비목별 집행, 잔액 반납
6. 향후 계획 (1~2p) - 후속 사업화, 투자유치 계획
```

---

## 사업비 정산 절차

### 증빙 서류 종류

| 지출 유형 | 증빙 서류 |
|-----------|-----------|
| 인건비 | 급여이체 내역, 근로계약서, 4대보험 납부 |
| 재료/기자재 | 세금계산서, 거래명세서, 납품확인서 |
| 외주용역 | 계약서, 세금계산서, 결과보고서 |
| 여비 | 출장신청서, 출장보고서, 교통비 영수증 |
| 회의비 | 회의록, 카드전표, 참석자 명단 |

### 사업비 카드 사용 원칙

```
필수: 사업비 전용 카드로 결제 (현금 최소화)
불가: 유흥업소, 사행업소, 귀금속, 면세점
주의: 토/일/공휴일 사용 시 사유서 필요
한도: 건당 100만원 이상은 세금계산서 필수
정산: 월별 카드 사용 내역 정리 및 적격 증빙 확보
```

## 사업 변경 신청

### 변경 가능 항목

| 변경 유형 | 승인 기준 | 절차 |
|-----------|-----------|------|
| 사업 기간 연장 | 3개월 이내 가능 | 종료 1개월 전 신청 |
| 비목 간 전용 | 20~30% 이내 자체 | 초과 시 승인 필요 |
| 참여 인력 변경 | 핵심인력은 승인 | 변경 신청서 제출 |
| 사업 내용 변경 | 경미한 변경 자체 | 중대 변경은 승인 |
| 자부담 변경 | 감액 불가 원칙 | 증액은 가능 |

## 후속 지원사업 연계 전략

```
현재 사업           →  후속 사업
예비창업패키지      →  초기창업패키지, TIPS
초기창업패키지      →  도약패키지, TIPS, 수출바우처
TIPS              →  후속 TIPS, 민간투자, 정책자금
중기부 R&D         →  후속 R&D, 사업화 연계, 성과확산
지자체 창업         →  중앙부처 사업, TIPS, 투자유치
```

**핵심**: 현재 사업의 성과(매출, 특허, 고용)가 후속 사업 선정의 가점이 된다. 성과를 체계적으로 기록하고 증빙을 확보한다.

## 체크리스트

- [ ] 협약서 내용(기간, 금액, 의무사항)을 꼼꼼히 확인했는가
- [ ] 사업비 전용 계좌와 카드를 준비했는가
- [ ] 분기별 보고서 제출 일정을 캘린더에 등록했는가
- [ ] 증빙 서류를 지출 즉시 정리하고 있는가
- [ ] 비목별 집행률이 적정 범위(분기별 목표)에 있는가
- [ ] 중간점검 대비 자료(실적, 시연, 정산)를 준비했는가
- [ ] 변경 사항은 사전에 승인을 받았는가
- [ ] 성과(매출, 특허, 고용)를 수시로 기록하고 있는가
- [ ] 후속 지원사업 지원 일정을 파악하고 있는가
