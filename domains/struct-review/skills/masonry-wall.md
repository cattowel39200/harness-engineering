# 석축 쌓기 구조검토

석축(찰쌓기/메쌓기)의 토압(경사면 포함), 자중(사다리꼴), 전도, 활동, 지지력을 검토하고 기초 콘크리트 적합성을 확인한다.

---

## 규칙

1. **결정론적 계산**: 동일 설계조건 → 동일 결과. 임의 가정 금지.
2. **전체 산식 출력**: 모든 계산은 6단계 형식을 반드시 따른다. 중간 과정을 생략하거나 압축하지 않는다.
3. **기준 조항 명시**: 모든 공식에 KDS 조항번호를 인용한다.
4. **단위 표기**: 모든 수치에 단위를 표기한다.
5. **안전율 판정**: 기준값과 비교하여 PASS/FAIL을 명시한다.
6. **가정값 표시**: 기준에 없는 값은 [가정] 표시 + 근거를 기재한다.
7. **석재 단위중량**: 화강석 γ = 26 kN/㎥, 자연석 γ = 25 kN/㎥

### 6단계 산식 출력 형식 (모든 계산에 필수 적용)

```
① 기준 근거: KDS XX XX XX 조항번호 — 조항 제목
② 공식: 기준에 명시된 공식 원문
③ 변수 정의: 공식의 모든 변수 (의미 + 단위)
④ 값 대입: 각 변수에 실제 값을 넣은 식
⑤ 계산 과정: 단계별 산술 (중간값 전부 기재, 생략 금지)
⑥ 결과 판정: 계산값 vs 허용값 → PASS/FAIL
```

---

## 계산 순서 (8 Step, 순서 변경 금지)

```
Step 1: 설계조건 확인
Step 2: 토압 산정 (경사면 고려)
Step 3: 자중 산정 (사다리꼴 단면)
Step 4: 전도 안전율 검토
Step 5: 활동 안전율 검토
Step 6: 지지력 검토
Step 7: 기초 콘크리트 검토
Step 8: 결과 종합표
```

## Step 1: 설계조건 확인

```
[필수 입력 — 하나라도 빠지면 진행 금지]

석축 형식           : 찰쌓기 / 메쌓기
석축 높이 H         : _____ m
상단 폭 B1          : _____ m
하단 폭 B2          : _____ m
전면 경사           : 1 : n (예: 1:0.3, 1:0.5)
배면 경사           : 수직 / 1:n

석재 종류           : 화강석 / 자연석
석재 단위중량 γ_m   : _____ kN/㎥ (화강석 26, 자연석 25)

뒤채움 단위중량 γs  : _____ kN/㎥
뒤채움 내부마찰각 φ : _____ °
뒤채움 점착력 c     : _____ kPa
배면 경사각 β       : _____ ° (수평면 기준)

지반 허용지지력 qa  : _____ kN/㎡
기초 마찰계수 μ     : _____

기초 콘크리트:
  폭 B_f            : _____ m
  높이 H_f          : _____ m
  콘크리트 강도 fck  : _____ MPa

[고정 상수 — 변경 금지]
콘크리트 단위중량 γc : 24.0 kN/㎥
```

## Step 2: 토압 산정 (경사면 고려)

### 2-1. 주동토압계수 (경사면 Rankine)

```
① 기준 근거: KDS 11 80 05 3.3.1 — 경사면 Rankine 주동토압

② 공식 (배면 경사 β가 있는 경우):
   Ka = cosβ × [cosβ − √(cos²β − cos²φ)] / [cosβ + √(cos²β − cos²φ)]

   배면이 수평(β=0)인 경우:
   Ka = tan²(45° − φ/2)

③ 변수 정의:
   Ka = 주동토압계수 (무차원)
   φ  = 뒤채움 내부마찰각 (°)
   β  = 뒤채움 경사각 (°)

④~⑥: 6단계 형식으로 산정
```

### 2-2. 주동토압 합력

```
Pa = (1/2) × γs × H² × Ka (kN/m)
작용점 y_a = H / 3 (저면에서)
토압 수평성분 Pah = Pa × cosβ
토압 연직성분 Pav = Pa × sinβ
```

---

## Step 3: 자중 산정 (사다리꼴 단면)

모멘트 기준점: 저면 앞꿈치 끝 (전도 검토 기준점)

```
① 기준 근거: KDS 11 80 05 4.2

② 석축 단면: 사다리꼴 (상단 B1, 하단 B2, 높이 H)
   전면 경사 1:n → 전면 경사 수평거리 = n × H

[석축 자중]
  면적 A = (B1 + B2) / 2 × H
  W_m = A × γ_m × 1.0(단위폭)

[도심 위치] (앞꿈치 끝에서)
  사다리꼴 도심 계산 (전면경사 반영)

[기초 콘크리트 자중] (있는 경우)
  W_f = B_f × H_f × γc

[뒤채움 토사 자중] (배면 경사 시 삼각형 토사 무게)
  W_s = 배면 경사에 의한 토사 면적 × γs

[합계]
  ΣV = W_m + W_f + W_s + Pav
  ΣM_R = W_m × x_m + W_f × x_f + W_s × x_s + Pav × x_pav
```

---

## Step 4: 전도 안전율 검토

```
① 기준 근거: KDS 11 80 05 4.3.2 — 전도에 대한 안정

② 공식:
   FS_ot = ΣM_R / ΣM_O
   기준: FS_ot ≥ 1.5 (상시), FS_ot ≥ 1.2 (지진시)

③ 전도 모멘트:
   ΣM_O = Pah × y_a

④~⑥: 6단계 형식으로 판정
   FS_ot ≥ 1.5 → PASS ✅ 또는 FAIL ❌
```

---

## Step 5: 활동 안전율 검토

```
① 기준 근거: KDS 11 80 05 4.3.3 — 활동에 대한 안정

② 공식:
   FS_sl = (μ × ΣV + c × B) / ΣH
   기준: FS_sl ≥ 1.5 (상시), FS_sl ≥ 1.2 (지진시)

③ 변수:
   ΣH = Pah (수평력)
   c  = 점착력 (기초면, 보통 무시)

④~⑥: 6단계 형식으로 판정
   FS_sl ≥ 1.5 → PASS ✅ 또는 FAIL ❌
```

---

## Step 6: 지지력 검토

```
① 기준 근거: KDS 11 80 05 4.3.4 — 지지력에 대한 안정

② 편심거리:
   e = B2/2 − (ΣM_R − ΣM_O) / ΣV

③ 편심 확인:
   e ≤ B2/6 → 전단면 압축

④ 최대 지반반력:
   σ_max = (ΣV/B2) × (1 + 6e/B2)

⑥ 판정:
   σ_max ≤ qa → PASS ✅ 또는 FAIL ❌
```

---

## Step 7: 기초 콘크리트 검토

```
① 기준 근거: KDS 14 20 72 — 무근콘크리트

② 검토 항목:
   (a) 기초 콘크리트 압축응력:
       σ_c = ΣV / (B_f × 1000) ≤ 0.6 × fck

   (b) 기초 콘크리트 크기:
       B_f ≥ B2 + 2 × 여유폭 (최소 100 mm 양측)
       H_f ≥ 200 mm (최소)

③ 판정:
   압축응력 ≤ 0.6 × fck → PASS ✅ 또는 FAIL ❌
```

## 산출물

1. **설계조건 확인서** (Step 1)
2. **토압 산정서** — 경사면 Ka, Pa, 성분 분리 (Step 2)
3. **자중 산정서** — 사다리꼴 석축 + 기초 + 배면토사 (Step 3)
4. **안정성 검토서** — 전도/활동/지지력 (Step 4~6)
5. **기초 콘크리트 검토서** (Step 7)
6. **결과 종합표** (Step 8)

---

## 체크리스트

- [ ] 모든 설계조건이 입력되었는가
- [ ] 석재 단위중량이 정확한가 (화강석 26, 자연석 25 kN/㎥)
- [ ] 전면경사(1:0.3 또는 1:0.5)가 반영되었는가
- [ ] 배면 경사각 β가 토압계수에 반영되었는가
- [ ] 토압의 수평/연직 성분이 분리 산정되었는가
- [ ] 전도 안전율 ≥ 1.5 (상시)
- [ ] 활동 안전율 ≥ 1.5 (상시)
- [ ] 최대 지반반력 ≤ 허용지지력
- [ ] 편심 e ≤ B2/6
- [ ] 기초 콘크리트 압축응력 ≤ 0.6 × fck
- [ ] **모든 계산에 6단계 형식 적용되었는가**
- [ ] **중간 과정 생략이 없는가**
- [ ] **모든 수치에 단위가 표기되었는가**
