# 중기부 R&D 실행 가이드

## 사업 개요

중소벤처기업부 R&D 사업은 중소기업의 기술개발을 지원하는 프로그램으로, 기술성과 사업화 가능성을 중점 평가한다. 기술보증기금, 중소기업기술정보진흥원 등이 전담 운영한다.

- **지원 규모**: 과제당 1~5억원 (사업별 상이)
- **지원 기간**: 1~3년
- **대상**: 중소기업기본법상 중소기업
- **선정 방식**: 온라인 접수 → 서면평가 → 대면평가
- **자부담**: 총사업비의 20~40% (기업 규모별 차등)

## 핵심 체크포인트

### R&D 사업 특화 요구사항
- [ ] 기술 분류(IPC, 국가기술분류)가 공고의 지원 분야에 해당하는가
- [ ] 현재 TRL과 목표 TRL이 명확하고 R&D 범위가 적절한가
- [ ] 선행기술 대비 차별성이 특허 수준으로 증명 가능한가
- [ ] 연구책임자의 해당 기술 분야 경력이 충분한가

### 서면평가 핵심
- [ ] 기술 개발 목표가 정량적 지표(성능, 정확도 등)로 제시되는가
- [ ] WBS와 마일스톤이 구체적이고 검증 가능한가
- [ ] IP 확보 계획(특허 출원 건수, 시기)이 있는가
- [ ] 사업화 계획(매출, 기술이전)이 포함되었는가

### 대면평가 핵심
- [ ] 기술의 핵심 원리를 비전문가도 이해할 수 있게 설명하는가
- [ ] 기술 위험요인과 대응 방안을 준비했는가
- [ ] 유사 R&D 수행 실적이 있다면 성과를 정리했는가

### 주의사항
- R&D 사업은 기술성 배점이 40~50%로 가장 높다
- 선행기술 조사(KIPRIS)를 반드시 수행하고 결과를 첨부한다
- 자부담 비율이 높으므로 자금 계획을 미리 확보한다
- 연구노트 작성이 의무인 경우가 많다
- 성과 목표(특허, 논문, 시제품)가 협약에 포함되므로 현실적으로 설정한다
