# 보고서 종합판정편 작성

구조계산서 제11장(종합 판정)을 작성한다. 모든 검토 결과를 하나의 표로 종합하고 최종 판정을 내린다.

---

## 규칙

1. **전 항목 종합표**: 부재별 검토항목, 계산값, 허용값, 비율, 기준 조항, 판정을 한 표에 정리한다.
2. **FAIL 항목 보완안**: FAIL이 있으면 보완 방안을 구체적으로 명시한다.
3. **최종 판정**: 전 항목 PASS면 "구조안전 확인", FAIL 있으면 "보완 후 재검토 필요".

---

## 11.1 종합 판정표

```
┌────────┬──────────┬────────┬────────┬────────┬──────────┬──────┐
│ 부재    │ 검토항목  │ 계산값  │ 허용값  │ 비율    │ 기준 조항 │ 판정 │
├────────┼──────────┼────────┼────────┼────────┼──────────┼──────┤
│ (각 부재별 행)                                                  │
├────────┼──────────┼────────┼────────┼────────┼──────────┼──────┤
│ 전체    │          │        │        │        │          │      │
└────────┴──────────┴────────┴────────┴────────┴──────────┴──────┘
```

## 11.2 최종 판정

```
□ 전 항목 PASS → 구조안전 확인
□ FAIL 존재    → 보완 후 재검토 필요
```

## 11.3 보완 필요 항목 (FAIL 시)

```
┌─────┬────────┬──────────┬────────┬────────┬─────────────┐
│ No. │ 부재    │ 항목      │ 현재값  │ 기준값  │ 보완 방안    │
└─────┴────────┴──────────┴────────┴────────┴─────────────┘
```

## 체크리스트
- [ ] 전 검토항목이 종합표에 포함
- [ ] 계산값/허용값/비율/판정 빠짐없이 기재
- [ ] FAIL 항목 보완 방안 기재
- [ ] 최종 판정 명시
